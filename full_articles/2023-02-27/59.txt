项目简介
----

Aura Network 是一家位于越南河内的生态系统提供商，旨在加速全球 NFT 的采用。他们的愿景是为构建具有实际效用的 NFT 项目创建一站式目的地，并逐步帮助将现有的 web2 业务转变为 web3。

随着 NFT 的发展，更多关于 NFT 的底层设施需求变多，Aura Network 主要针对 NFT 进行开发，通过解决可扩展性问题来提高效率，由 Tendermint BFT 共识提供支持，支持 DeFi、GameFi 和 SocialFi，可以在 Aura Network 上构建任何 NFT dApp，实现 NFT 跨链操作。

![ ](https://img.foresightnews.pro/202302/0678e501b07e665c4e7a9254efcc5538.png?x-oss-process=style/scale70)

在创始人兼首席执行官 Giang Tran 的领导下，Aura Network 提供可扩展、敏捷且轻松的第 1 层区块链以及全面的生态系统，以加速全球 NFT 的采用。迈向 Mainnet 发布，该公司的开发团队目前正在致力于构建几款新产品，发布这些产品的时间表将很快在其 2023 年路线图中公布。

### 生态

![](https://img.foresightnews.pro/202302/77fbcc8f24189c0dd5cc0248a36c20d6.png?x-oss-process=style/scale70)

*官网公布的生态列表，绿色项目为目前已在进行开发，灰色为未来计划*

  


Aura 生态系统可分为 4 层

1.基础设施：包含与底层区块链平台相关的所有资源，这包括节点客户端 Aurad 和所有资源，例如创世块、配置文件和服务于 Aura Network 区块链的已部署服务。

2.币种：Aura 生态有 2 种代币：BNB 智能链上的 Aura Coin 和 Aura Token。Aura 还支持创建 CW20 代币，类似于使用 CosmWasm 模块的其他区块链。

3.dApps：在使用 DeFi 和 NFT 时提供丰富用户体验的应用程序。

* AuraScan：区块链浏览器，具有针对 Aura 持有者量身定制的治理、质押、NFT、通知等功能。
* Pyxis Safe：受 Gnosis ,使用多重签名管理资产安全的解决方案。
* 钱包：Aura 官方钱包为 Coin98 钱包。其他宇宙支持的钱包，如 Keplr，Cosmostation 也支持。
* Twilight Hub：Aura 社区与 NFT 和元宇宙互动的一站式目的地。
* Aura DEX：基于 Aura 的 CW-20 代币的去中心化交易所。
* Bridge : Aura 桥，用于在基于 EVM 的区块链和 Aurachain 之间交换资产。
* Launchpad : Aura 为构建在 Aura 链上的 NFT 项目提供技术和资金、业务推广等方面的密集支持。
4.API：基于 Aura 的应用程序需要为公共服务提供标准的开放 API。

### Aura Network 主网 The Xstaxy 即将启动

Aura Network 主网 The Xstaxy 将于 3 月 20 日 21 时（3 月 20 日 13：00 UTC）正式启动，经过了超 15 个月的持续开发以及稳定的测试和产品优化。随着主网的激活，Aura Network 将自己定位为第一个获得许可的智能合约 layer-1 链，推动在 Cosmos 生态系统中大规模采用 NFT。

![](https://img.foresightnews.pro/202302/0e3e2b598f636673bde22b9237261b71.png?x-oss-process=style/scale70)

主网开发的一些重要关键更新包括：

* 生态系统扩展：除了目前的产品，包括 AuraScan 和 Pyxis 安全，几个与可持续的 DeFi 和 NFT 市场有关的 dApp 目前正在建设和不断测试，作为 Aura 生态系统的重要组成部分。这些产品将很快以 alpha 版本引入社区，沿着为早期采用者提供体验活动。在产品改进方面向 Aura 技术团队提供适当反馈的人员将获得奖励。
* Token 迁移到主网：作为主网启动中最重要的阶段之一，BEP-20 Aura 令牌和主网上的本地令牌之间的迁移流现在已经完全优化，以实现安全和轻松的过程。Aura 团队一直在积极与 CEX 和 DEX 协调新的本地令牌支持。此外，Pyxis 安全正在通过一系列的安全审计，由一流的安全研究人员和专家提供支持。
* 招募额外验证程序：在主网准备的最后一步，Aura Network 正在招募额外的 18 个创世块验证者。全球开发者和验证者可以通过 Aura 基金会授权计划参与 Mainnet 的发布。这项活动的目的是通过适当分配授权资金，促进网络的普遍增长和权力下放。
* 伙伴关系和社区发展：自从 Hashed 和 Coin98 发起了 400 万美元的首轮融资以来，Aura Network 已经与 Web3 和区块链领域的几家合作伙伴达成了协议。这将有助于收集急需的动力，以推动其主网推出，在产品开发，运营和营销战略。
Aura Network 现在正式进入下一阶段，宣布备受期待的 Aura Network 主网 The Xstaxy 即将推出。这一重大成就代表了其在 Web 3 领域的革命性足迹，以及成为基于 Cosmos Layer1，专注于 NFT、Web 3.0 游戏和可持续 DeFi 的领导者之一的雄心。
