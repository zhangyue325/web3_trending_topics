**01**  
**什么是 Base，为什么 Coinbase 决定开发自己的 L2**Coinbase 是第一家推出自己的 L2 的美国上市公司。Base 将在未来几个月推出，它具有以下特点：

1. 开源；

2. 基于 Optimism；

3. EVM 兼容；

4. ETH 交易结算。Base 没有自己的原生代币，交易将在 ETH 上结算，这与其他 L2 不同，并不是靠代币的空投，来吸引最初的一波用户。**02**  
**为什么现在发布 Base**Coinbase 之前的一系列操作，尤其是 NFT 市场（最终因交易量不足而关闭）的失败让人怀疑。但是，对于最新发布的 Base，其影响力可能要更大，意义也更加深远。  
![](https://img.foresightnews.pro/202302/e374d9111fcc257469d63fb3f839d782.png?x-oss-process=style/scale70)

Brian Armstrong（Coinbase CEO）已经在许多采访中表示，一旦技术上准备就绪，他们就将推出自己的公链，对于 L2 的开发是他们一直所期待的事情。  
从战略上讲，交易所通常希望用户在其内部进行交易，以收取手续费。而 Base 的发布，则具有以下优势：1. 使收入来源多样化，不再单纯依靠交易所手续费；

2. 在自家的 L2 上留住用户，防止用户流转到其它二层，如 OP、Polygon、Arbitrum 等；

3. 基于模块化和可组合性的优势，拥有一个在链上发布未来 Coinbase 产品的地方；

4. 与其他基于 OP 的公链互连。虽然跨链不是一件难事，但请不要忘记大多数用户不会，尤其是那些使用 Coinbase 的用户。Base 能够为这些用户使用 dApps 提供最简单的入口，特别是在 DeFi 中。此外，缺乏代币和对 ETH 的依赖也显示了 Coinbase 的良好意图，这意味着他们将专注于建立一个蓬勃发展的生态系统来吸引用户。**03**  
**Base 与 Coinbase 的关联性**用户可以从 Base 中受益，因为他们能够对其资产进行自我监管，同时仍处于 Coinbase 生态系统中。目前，Coinbase 拥有超过 1.1 亿用户和超过 80b 美元的资产管理规模，最终可能会加入 DeFi。**04**  
**为什么选择 Optimism**

Coinbase 和 Optimism 团队在 EIP-4844 上进行合作，根据 Bankless 播客的说法，他们之间的谈话就是从那时开始的。那个时候，Coinbase 已经计划了几年来开发自己的区块链。通过建立在 OP 技术堆栈上，Base 会将一定比例的交易费用用于 OP 治理，而 Coinbase 将是 Base 上唯一的（许可的）排序器，类似于其他 L2，如 Arbitrum, OP，Base 计划最终转移到一个无需许可的排序器。作为一家受公共监管的公司，启动 L2 比启动 L1 具有更少的法律影响。Coinbase 推出 L2 是一个在战略上不同的选择，尤其是与推出 L1 的竞争对手（尤其是币安）相比。这些 L1 没有什么机构和主流用户在使用，其关键的问题还是过于中心化（例如 BNB 链的所有验证器都是由 Binance 控制）。![](https://img.foresightnews.pro/202302/9648a3163cfa772d9d694f6fae286d73.png?x-oss-process=style/scale70)

而 Coinbase 虽然控制排序器，但交易是在以太坊上结算的，这能够确保一致性和不可篡改。根据 Coinbase 团队的说法，选择 L2 意味着：「建造桥梁，而不是岛屿」。**05**  
**Base 生态系统**我们期待 Base 推出一个建设完善的生态系统，不少的 dApps 正试图通过加入 Base 以占领市场份额。在接下来的几周内，将会有一些协议进行治理投票，决定是否要在 Base 上启动。**06**  
**小结**1. Base 用户需要 KYC，在一些较小的国家，使用起来可能会有一些限制，同时加密市场整体的恶意也是比较大的，但是总的来说，敦本务实的 Base 对行业的发展仍然是一个积极因素，它有可能吸引数百万的用户使用 DeFi；2. Base 的推出是交易所领域的首创，如果最后证明是成功的，那么其他人可能会复制该操作；3. 这对其它 L2 意味着什么？它们也会像 OP 一样被机构所选择？还是说未来只有 OP 会成为专属合作对象？4. Base 的发布是非常积极的消息，可以在牛市中大幅推动加密市场的上涨，尽管目前不是牛市，但其仍然为下一个牛市的到来，建立了新的用户群。//
