Base 的推出是 Coinbase 在链上去中心化的又一重要实践。除了对 Coinbase 自身以及更加广泛的加密生态的利好之外，Base 也让我们重新将目光聚焦在了基于 OP Stack 之上的 “OP 宇宙”。本文将对 Base 的推出进行更进一步的介绍，并从 “OP” 宇宙的视角为大家解读 L2 发展的新叙事。

![](https://img.foresightnews.pro/202302/867e7228d32db04582422db2850a6301.jpeg?x-oss-process=style/scale70)2023 年 2 月 23 日，Coinbase 宣布推出以太坊 L2 网络 Base 的测试网版本，躬身入局 L2 赛道。据 Coinbase 高级工程师总监 Jesse Pollack 更详细的介绍，Base 将会以以太坊的 Goerli 测试网为模型，建立在开源的 OP Stack 之上，并且将作为第二核心开发主体，与 Optimism 共同合作共建 OP Stack。Base 还将整合现有的 Coinbase 开发工具与消费产品，推出 Base Ventures 为 Base 生态中的早期项目提供资金支持。Base 将不会发行新的网络代币，仍将使用 ETH 作为原生 Gas 代币，并且会将 Base 的部分交易收入返还给 Optimism Collective 以支持 OP Stack 的发展。同时，Optimism 也发布公告称计划将原本孤立的 L2 整合成一个单一的、可互操作的、可组合的系统超级链（Superchain）。Base 发布的消息一经推出，短时间内在市场上一石激起千层浪。本文将对 Base 的推出进行更进一步的介绍，并从 “OP” 宇宙的视角为大家解读 L2 发展的新叙事。1

  
**打造更加开放自由的世界经济**

根据 Coinbase 官方表述，Base 的推出或许已经是一个蕴含了数年的“秘密”。自 Coinbase 建立之初起，团队就计划创建一个开放的金融系统，使全球经济更加自由。从协议（比特币、以太坊）到交易所，再到为 Dapps 创建大众市场的接口，Coinbase 越来越着眼于第三个阶段的建设，也同时开始逐步实践品牌的链上拓展。数年的时间里，Coinbase 团队也曾设想过类似币安推出 BSC 一样，推出一个自我品牌的 L1，或是像众多解决方案各自不同，彼此孤立的 L2 一样推出一个 Rollup 解决方案。但最终因为对“他们应该如何构建 Dapps”以及“他们应该在哪里部署 Dapps”的思考和对目标与使命的追求而放弃了发链的想法。直到 Coinbase 团队逐渐认知到“未来将会有许多 L2，成为不同生态系统活动的枢纽，并且 L2 之间的互操作性会逐渐增强，直到它们形成一个网络或超级链，共同拓展以太坊”这一点。并确认了团队可以通过自己构建一个实现 Coinbase 使命并促进更加广泛的加密经济活动的主体时，他们才决意推出 Base。据官方表述，Base 推出的原因为：1.Base 将加速 Coinbase 在链上产品的开发。2.Base 将允许 Coinbase 的客户进入加密经济。3.Base 将增加 Coinbase 对核心加密基础设施（例如 EIP 4844）的投资。Base 的推出也将带来更多的潜在利好：  
1.Coinbase 中的 Coin 作为股票代码承接传统金融世界的用户，另一半的 Base 将作为加密世界的新兴 L2，承载下一代亿级用户。Base 的发布将推动链上应用的开发，迎接更多的用户绕开监管，进入加密生态。2.从 CEX、Wallet 到 Rollup 生态，Base 的发布也将助力 Coinbase 产品矩阵的更加完善，为用户提供一整套完备周全的链上链下产品服务。同时，Base 将为开发人员提供更加完备的开发工具，让更多的开发人员能够像部署新的 ERC 20 代币一样简单地部署去中心化应用以及 L2 网络，丰富加密生态。3.Coinbase 也不希望 Base 成为孤岛，而是寄希望于 Base 成为一个深度链接到以太坊及其 L2、比特币甚至其他 L1 网络的桥梁。Base 的推出也能够进一步推动 OP Stack 的发展以及超级链进程。4.基于 OP Stack 的 Base 的推出也是模块化区块链的又一进程，同时也将推动模块化区块链技术堆栈的发展，重启市场对于模块化区块链以及“OP Stack”的关注。5.推动加密经济实现更深层次的去中心化。2

**Stage 2  Rollup 的**

**去中心化宏伟蓝图**

2023 去中心化路线图：1.为 OP Stack 启动至少一个故障证明器，利用一组验证器来识别和解决故障，从而保护 Base 和 Optimism 主网。2.预计推出一个安全委员会以实现更加完全的去中心化，摆脱单一实体的控制。3.启动 Superchain 的初始版本以实现对 Base、Optimism 主网和其他 OP Stack Rollup 的交易排序功能，推动更加开放的、低准入门槛的去中心化。4.预计将在 2024 年，推动 Base 和 Optimism 主网达到 Stage 2 Rollups，进一步提高 Base 和 Optimism 的去中心化程度以及安全性。从 Base 的 2023 去中心化路线图中，不难看出 Base 的一大使命即是推动更加开放、无需许可、低准入门槛和更加广泛的去中心化。从 Coinbase 的角度来看，Coinbase 也不仅仅满足于成为一个中心化交易所，而是要进一步成为去中心化开发链上世界的主要活动网关之一，为 Base 生态、超级链生态的加密经济活动提供更加广泛的底层支持。底层 Rollups 的去中心化也能够在一定程度上激发应用层的活跃，减少中心化的作恶可能，保护应用层的健康发展。3

**从 Base 看 “OP 宇宙”的无限潜力**

除了 Base 之外，OP Stack 也是我们不容忽视的一环 。OP Stack 简言之即是 Optimism 的模块化战略。现有的 L2 解决方案层出不穷，虽然各式各样的 L2 都共享以太坊的安全性，但每条链独立的执行、结算和数据可用性上的策略差异使得 L2s 成为了孤岛式的生态。而 OP Stack 则致力于打造一个开源的标准化的代码库，实现 L2s 之间的耦合。OP Stack 将区块链功能分为了共识层、执行层和结算层，将不同的功能解耦，再以 API 的形式组合层级功能，为开发人员提供了一套标准化的、极致简便的开发工具。开发者可以基于 OP Stack 的代码库以 API 的形式实现模块化区块链的部署，并且可以自由修改任意模块。这将为开发人员提供极大的便利性以满足不同应用程序的需要。OP Stack 的另一极具吸引的设想即是共享消息传递格式的开发。当多个基于 OP Stacks 开发的链能够共享一个 Sequencer Set 的时候，链与链之间将获得原子跨链组合性。这意味着链与链之间的互操作就像单一链本身的原子操作一样，我们在 2022 年看到的频发的跨链桥资金安全事故或许将不再发生，链与链间互操作性的增强也将激发不同链上生态的活跃。如此一般基于 OP Stack 开发的多链生态也是 OP 与 Base 共同追求的超级链的设想，孤岛式的 L2 生态将被打破，L2 与 L2 之间，L2 与应用之间，应用与应用之间会获得更加有机的结合。或许在日后，我们关注的不再仅仅是 Base 或是 OP，而是 “OP 超级宇宙”，一个更加宏大的可组合的概念，一个更加迷人的区块链花园。![](https://img.foresightnews.pro/202302/e58a3aaa54f6d12357b008fc0fcc04f3.jpeg?x-oss-process=style/scale70)
