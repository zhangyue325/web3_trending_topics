**翻译：**Flow 技术大使 Jing  


  


越来越多的创业者、Web3 建设者和品牌方正在选择 Flow 来与多元化、全球性的社区互动，因为 Flow 专为跨领域协作、社区创作和深度个性化而设计。它是一个开源平台，积极支持创新和包容性社区体验，为你已经熟知的品牌提供服务，包括：**Shopify**、**Ticketmaster**、**Youtube**、**Genies**、**LaLiga**、**Mattel**、**Bud Light**、**Ballerz**、**American Airlines**、**NFL**、**NBA**、**UFC** 等等。

  


这些品牌有什么共同之处？他们都想为全球社区 - 包括你！- 提供前所未有的体验，并且都正在通过 Web3 来实现这一目标。他们希望你和你的所有朋友能够一起与所喜爱的运动员、艺术家、团队、游戏等进行无缝的体验。

  


为了能够实现这一无缝体验目标，开发人员们正在利用最具可扩展性和全球性的软件执行平台来做到这一点！**凭借其面向资源的编程语言 Cadence，在 Flow 上快速又直观地构建智能合约**。

  


Cadence 开发与面向对象编程非常相似，同时也具有专用的语言元素，Cadence 可以安全地实现所有权和访问机制。有理想的开发者、创业公司和知名品牌都希望在未来能够做到覆盖数十亿用户。Flow 的架构正是为能承载这样的用户体量而设计，同时在 web3 中拥有无与伦比的性能和硬件效率。

  


对于产品开发团队来说，在全球社区开发中保持一致的体验非常具有挑战性。当然，我们希望为所有粉丝和客户群体构建出满意体验，但要在不同的监管区域确保安全和一致性却非常困难。

  


在 Flow 上的 web3 应用可以利用良好标准和服务来使解决这些问题变得简单，但对于最有影响力的开发者而言，这些只是基础要求。

  


除了为全球客户提供服务和顶级标准外，开发者选择 Flow 还因为 Flow 虚拟机（FVM）可以提供个性化、自由和灵活性。你可以把 FVM 看作一个全球操作系统，支持原子操作，并且即使在出现故障的情况下也能保持高度的一致性。这些安全的基础设施允许社区由参与其中的个人定义，每个人都可以自由地在 Flow 的虚拟体验世界中移动自己和他们的财产。

  


作为开发人员，您可以添加产品和服务，利用个性化吸引用户，例如，Genies 提供了一个 SDK 来构建其虚拟形象的配件。这为一些早期社区提供了一个非常不错的机会，以有趣的方式扩展其产品。SDK 还为开发人员提供了所需的工具，以利用其虚拟形象生态系统并使用 Genies 组合自己的游戏或体验。我们将此内在的能力称为可组合性，即扩展在 FVM 上运行的产品的能力。

  


当今 Web 栈优先考虑个人自主权、社区参与和无缝全球可用性 - **Flow 通过 Flow 账户、Flow 虚拟机和其支持组合性的分散式可扩展架构来实现这些**。下面我们将深入探讨为什么您需要以不同的方式思考社区战略以及如何在 Flow 上实现它：

* 为什么选择 Flow？通过 Flow 账户提供真正的所有权。
* 为什么选择 Flow？在 FVM 上设计 Web 栈（即将推出）。
* 为什么选择 Flow？通过组合性解锁下一个创新时代的平台（即将推出）。

对于 Flow 分布式、容错的操作系统，了解 Flow 的架构是如何在保持组合性和分散化的同时扩展到数十亿用户的（即将推出）。

  


如果您想要创造一些有趣，一些以前从未见过的东西。如果您想要与分布在世界各地的朋友分享您所建造的东西。如果您已准备好迎接一个前所未有的创新时代，一个以社区、开源驱动，并能建立自己品牌的时代。

**什么是 Flow 福洛链？**  


  


Flow 福洛链是一个快速，去中心化，且对开发者友好的区块链，旨在为新一代游戏、娱乐应用程序提供动力的数字资产的基础。Flow 是唯一一个由始至终为消费者提供出色体验的 Layer-1 区块链团队。其团队创造的 dApp 包括：**CryptoKitties**、**Dapper Wallets**、**NBA Top shot**。

  


CrytoKitties 于 2017 年推出时便快速成为加密市场最受欢迎的 dApp，因其成功而导致以太坊堵塞。在 Flow 上运营的 NBA Top shot 也已成为增长最快的 dApp，在公开发布后的 6 个月创造了 7 亿美金销量。正因为 Flow 公链的可扩展性和消费者友好的体验，让这一切成为可能。目前有 1000 多个项目正在 Flow 链上筹备中，我们期待看到一个伟大的生态系统蓬勃发展。

  


**关于 Dapper Labs**

Dapper Labs 是一家位于加拿大的全球顶尖区块链服务商，在 2017 年年底通过 CryptoKitties 收藏游戏成功进入⽤户视野，并且因为加密猫的爆⽕导致以太坊拥堵，从而推出 Flow 公链以及全新的开发语言—— Cadence，旨在吸引更多的开发者在 Flow 上开发应⽤。
