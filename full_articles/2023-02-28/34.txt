**撰文：Liam，** **A&T Capital Analyst、DeFi farmer**

**导语：**2023 年 2 月 23 日，Coinbase 宣布推出 Base 测试网。Base 是基于 OP Stack 打造的以太坊二层网络 (L2)，旨在提供安全、低成本、且对开发人员友好的方式来构建链上去中心化应用程序。Coinbase 与 Base 的目标是吸引下一波 10 亿级用户进入加密经济。

TL; DR
------

* Base 是基于乐观假设 - 欺诈证明原理的以太坊二层网络，能够在几乎不降低网络共识安全性的前提下，为用户提供更低成本的 Dapp 交互方式。
* 作为二层网络，Base 本身并不需要发行 Token 也能正常运行。其潜在商业模式包括赚取 Gas 差价和提取 MEV 等。
* 现阶段，Base 并没有采用太多领先行业的“黑科技”，只是在 Optimism Rollup 技术栈相对成熟时直接采用了现成的技术方案，复制了一款市场中已经存在的产品。
* 不过，Base 的推出为互联网大厂切入 Web3.0 基础设施方向提供了新思路。二层网络方案中，通过“租用”底层公链的共识，即使不发行 Token 也能保障网络状态安全。受制于监管政策的不完善，上市公司在推出底层公链时面临许多合规问题，而二层网络方案恰能规避部分限制。
* 以太坊扩容能力的提升已经接近“拐点”，此时切入 Rollup 或许恰是良机。
  


一、Base 是什么？
-----------

产品上，Base 是一个通用型的以太坊二层网络，用户体验与已上线的 Optimism 和 Arbitrum 基本无异。开发者能够在 Base 上部署智能合约、搭建 Dapp，用户能够使用以太坊账户与这些 Dapp 交互。

技术上，Base 使用了 Optimism 团队开发 OP Stack，基于乐观假设 - 欺诈证明原理建立了一个 Optimism Rollup。

本质上，Base 由部署在 ETH 主网的智能合约和链下中心化服务器组成。

* 服务器提供兼容 EVM 的执行层，通过中心化手段高效地处理交易，并将计算出的状态变更结果上传至 ETH 主网的智能合约进行共识，获得最终性。
* ETH 主网的智能合约负责仲裁状态变更结果。在挑战期内，如果有人能够提供证据证明被上传的变更结果是有误的，那么错误的结果将得到纠正，举报人将获得奖励，而上传错误结果的一方则会受到惩罚。挑战期结束后，如果无人举证成功，合约将会“乐观地”裁定被上传的结果是正确的，在主网上获得最终性。

二、Base 能够用来做什么？
---------------

* 对开发者：Base 兼容 EVM，开发者在 Base 上部署智能合约的体验与主网近似，现有合约迁移成本也相对低。
* 对用户：Base 兼容以太坊原生账户，用户只需要在钱包中切换网络，即可与 Base 上的 Dapp 交互，交互体验与主网无异，Gas 成本则大大降低。
三、Base 的商业模式？
-------------

与许多其他的 Rollup 不同，Base 不会发行自己的 Token，不靠卖 Token 盈利。

目前，Base 尚未公布其商业模式。不过，作为二层网络，有两条潜在的商业化路径：

* Gas 差价。用户支付给 Base 的 Gas 可能高于 Base 在 ETH 主网共识最终性时所消耗的 Gas，其间的差额可以是 Base 的利润。
* 提取 MEV。用户提交给 Base 的交易按何种顺序执行完全由 Coinbase 决定，理论上可以赚取 MEV。

四、Base 能为 Coinbase 带来什么？
------------------------

* 把用户留在 Coinbase 生态内，盘活 Coinbase 的其他产品，如 Coinbase Wallet，NFT Marketplace 等。
* 战略性布局，避免错过行业下一轮牛市的主线。
* 为投资 Base 上的优质 Dapp 埋下伏笔。
上一轮牛市中，制约以太坊发展的重要原因是主网性能无法满足用户需求。随着二层方案提升以太坊扩容能力的可行性得到初步验证，即使作为一家头部中心化交易所，Coinbase 也不得不担心被更去中心化的 Dapp 版本取代。

所以，对交易所而言，发布公链、Rollup 是战略布局。Binance、OKX、Crypto.com 和 Huobi 等都有染指。Coinbase 也承认在 2018 年和 2020 年都有过发公链的想法，但没能实现。相较于真正的 Layer 1 公链，Rollup 不发 Token 也能运行，解决了不少合规的顾虑；而 Optimism 团队将发行 Rollup 的技术解决方案开源，也为 Coinbase 省去了不少工程上的问题。

五、Base 模式对互联网大厂的借鉴意义？
---------------------

即使没有 Token，也能切入公链级的基础设施赛道。在二层网络方案中，通过“租用”底层公链的共识，即使不发行 Token 也能保障网络安全。Base 模式为希望切入公链赛道但又不愿发行 Token 的上市公司提供了新思路。

以太坊扩容能力的提升已经接近「拐点」，此时切入 Rollup 或许恰是良机。Optimism 和 Arbitrum 主网上线的半年已经初步验证了二层网络方案的可行性。目前，在 Rollup 上发送一笔交易的费用不到主网上的十分之一。未来，随着 EIP-4844 等其他优化方案的落地，发送交易的费用可能只需要主网的千分之一。届时，在 Rollup 上搭建承载十亿级用户的 Dapp 将成为现实。
