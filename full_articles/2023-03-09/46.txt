2 月 23 日，Coinbase 宣布推出基于 OP Stack 的以太坊 Layer 2 网络 Base，以承载未来 10 亿级别的用户进入加密生态。目前 Base 已开放测试网并预计在两个月后发布主网。此前消息一出，整个市场都在讨论 Base 的未来。

3 月 5 日，在以太坊丹佛大会 ETH Denver 上，Coinbase 协议负责人 Jesse Pollak 表示，过去十年 Coinbase 一直以交易为中心，产品创新可能不是首要的，但现在该公司旨在改变这种状况，推出 Base 其实是对「创建一个平台」的投资，让开发人员更好地构建有用产品。 

今日 Beosin 热点研究系列，就和我们一起解读 Coinbase Layer2 网络 Base 与 OP Stack。

### **为何 Coinbase 要推出 Layer2 网络 Base？**

Coinbase 的最终目标是创建一个开放式的金融系统。Coinbase 早在 2016 就有一个 Coinbase 秘密计划。该计划讲述了 Coinbase 将在加密生态发展的 4 个阶段所要完成的任务：

![](https://img.foresightnews.pro/202303/cc115b2e7296eda78880ee79545a4b90.png?x-oss-process=style/scale70)

*表格源自 https://www.coinbase.com/blog/the-coinbase-secret-master-plan*

第一阶段 Coinbase 团队与比特币、以太坊的开源社区合作，提供资金支持，帮助构建底层协议。

第二阶段 Coinbase 团队创建 Coinbase 与 Coinbase Pro，为散户、机构和专业交易者提供安全便捷的加密生态服务。

第三阶段 Coinbase 参与构建 / 投资去中心化应用，为用户提供更加方便的应用层来使用加密货币。在此期间，Coinbase 开发了 Coinbase Wallet 来帮助用户安全快捷地存储加密资产和访问各种去中心化应用；Coinbase Ventures 也在 Web3 的各领域投资扶持有前景的项目：在 DeFi 里投资了 Uniswap、Synthetix、Rocket Pool 等项目，涵盖了交易、衍生品、合成资产、保险、ETH2.0 等方面；在 NFT 领域投资了 Yuga Labs、OpenSea、Rarible 等项目，涵盖音乐、奢饰品、粉丝类的 NFT 和 NFT 铸造、发行、交易的平台；在基础设施方面投资了 Aptos、Near、Polygon、Mina 等公链，Arweave、Kyve、Subspace 等存储设施，Alchemy、OpenZeppelin、Moralis 等 Web3 开发工具。

第四阶段 Coinbase 将打造一个全球开放的金融系统，重塑现在的金融体系，提高全球的经济自由度。Coinbase 将在这个金融系统中构建链上的身份和信用评分系统，为全球的普通用户和专业机构提供不同类别的贷款、储蓄、指数基金、股票投资以及跨境汇款等金融服务。此外，Coinbase 还期待该系统可以改进线下和网上购物的支付系统。

而打造开放式的金融系统需要公共区块链及其生态。根据 Coinbase 的资深技术总监 Jesse Pollak 的描述，Coinbase 在过去几年也有发布公链的想法，但最后都放弃了。

目前去中心化应用已经逐渐繁荣，Coinbase 可以确定整个加密生态已经进入他们的第三阶段并向第四阶段前进。

而 Base 正是 Coinbase 第三阶段和第四阶段之间的关键部分。如果 Base 成功推出，那么有助于 Coinbase 自己的去中心化应用的开发和部署，在未来结合 Coinbase Wallet 和智能合约钱包的账户抽象技术，可以让 Coinbase 的客户非常方便地进入 Base 承载的金融服务协议。

### **为什么 Coinbase 要选择 Layer2 和 OP Stack？**

1. 构建 Layer1 的成本和难度高于 Layer2。Coinbase 如果选择构建自己的 Layer1，那么它需要发行对应的公链代币，这会让它在未来持续承受 SEC、NYDFS 等机构的监管压力。此外，开发、运行和维护公链的成本要比运行 Layer 2 Rollup 的成本要高。

2. Layer2 更为「正统」。Layer2 是以太坊的重要扩容手段，已经承载了以太坊上的资金和流量。如果 Coinbase 推出公链而非 Layer2，那么这条新公链与以太坊是竞争关系。正如 2021 年下半年的公链战争，Coinbase 需要耗费大量资源才有可能吸引用户到它的新公链，加上 Layer2、BNB Chain 以及其它公链的竞争，Coinbase 的新链很大概率会成为一座孤岛。而 Coinbase 选择 Layer2，Base 无需发币能避免监管，又加入了以太坊 Layer2 的阵营，能像 Arbitrum 和 Optimism 一样承接以太坊的流量，再加上 Coinbase 的引流，Base 未来成功的概率大大提高。

3. OP Stack 是一套完整可行的构建模块化 Layer2 的方案且 OP Superchain 的叙事宏大。据 Jesse Pollak 透露，他们几乎与每一个致力于扩容以太坊和开发 Layer2 的团队沟通过。他们意识到未来将会有许多 Layer2 网络，那么 Layer2 之间的战争也不可避免。如果 Layer2 网络之间还是像公链那般用跨链桥链接起来，这就和当前的糟糕情况一样：不同的网络之间依然割裂，跨链桥的风险与成本摩擦依然存在。而就在 2022 年的下半年，Optimism 的团队正在将他们原先的开发工作拆解成多个层面的模块（Data Availability Layer，Derivation Layer，Execution Layer 和 Settlement Layer）并开发了 OP Stack。OP Stack 通过模块化解耦区块链各功能，并以 API 来组合各模块。只要某个功能模块 API 符合 OP Stack 的标准，那么就可以作为一个模块加入到 OP Chain 的搭建中。这使得其他开发者能快速地构建一个 Layer2 网络。而通过 OP Stack 构建的 OP Chain 不仅可以复用已有的 Optimism 的开发成果，还可以继续改造或者创新为 OP Stack 提供不同层面的不同模块，丰富 OP Stack 的生态。在此基础上， Optimism 的团队提出了 Superchain 的设想，期待在未来实现 Layer2 的「大一统」。

![](https://img.foresightnews.pro/202303/b3b084f502977fb18ef4aca5d027480f.png?x-oss-process=style/scale70)

*图源自：https://stack.optimism.io/docs/understand/explainer/#superchain-overview*

目前 OP Stack 推出了第一版：Bedrock。如图所示，目前 Bedrock 的 Consensus Layer（由 Data Availability Layer 和 Derivation Layer 组成）采用的是 Rollup，执行层为 EVM，而结算层为 Governance。这些层里的模块都可以被替换成其它模块。Lattice 团队就通过 Bedrock 将 Minecraft 搬到了链上，制作了 OPCraft。还有一位开发者将执行层的 EVM 换成了游戏模拟器，制作了一个游戏模拟器 Rollup。目前 Optimism 在结算层面并未采用 Fraud Proof 或是 ZK Proof，这也是它现在饱受争议的一点。但随着 OP Stack 开发的进行，在未来 Optimism 以及其它 OP Chain 能够在结算层方便地选择加入 Fraud Proof 或者 ZK Proof。

![](https://img.foresightnews.pro/202303/b66903e80db4ec1e4a5d074bf342de64.png?x-oss-process=style/scale70)

*图源自：https://optimism.mirror.xyz/fLk5UGjZDiXFuvQh6R\_HscMQuuY9ABYNF7PI76-qJYs*

除了 Coinbase 与 OP Labs 之前有共同研究 EIP4844 外，OP Stack 的初步成型和 Superchain 的宏大叙事大概率是 Coinbase 加入 OP Labs 并参与 OP Stack 的研究和开发的最大助力。未来 Base 的部分手续费也会流入 OP Collective，继续推动 OP Stack 的发展。也许在未来几年，我们将看到更多 OP Chain 的诞生和 OP Stack 的繁荣，Layer2 将迎来新的格局。

### **Base 的意义**

1. 帮助 Coinbase 为下一阶段去中心化应用的大规模采用做好准备。Base 将承载未来 Coinbase（2022 年 Q3 Coinbase 有近 1.1 亿用户数和超过 800 万的月交易用户）、Layer2 以及 Layer1 的流量。目前，已有近 60 个项目成为 Base 的生态合作伙伴，其中包括 Chainlink、Nansen、Dune Analytics、The Graph 等基础设施和工具，也有原先构建再以太坊的知名项目，如 Sushiswap、Balancer、PoolTogether。

![](https://img.foresightnews.pro/202303/c6b7330008f975e0155e7e6e2791a043.png?x-oss-process=style/scale70)

2. 帮助 Coinbase 增加收入来源。从下图可以看到，目前 Layer2 龙头在熊市一年的净利润约为 4-5 百万美元，而 Coinbase 在 2022 年净亏损超 20 亿美元，目前还面临因政策监管禁止提供 ETH2.0 质押服务的困境。Coinbase 除了交易收入，还在努力拓展其它收入：托管费用、USDC 以及其它加密资产的利息收入、Coinbase NFT 市场。在未来 Base 的手续费将成为 Coinbase 又一份收入来源。

![](https://img.foresightnews.pro/202303/fab62bcf36f987a8d499cbea8a7402b0.png?x-oss-process=style/scale70)

3. 进一步确定以太坊和 Layer2 的地位。Coinbase 明确表示没有发币计划，使用以太坊作为网络汽油费。此外 Coinbase 加入了 OP Stack 的开发，并宣布继续投资和建设 Layer2 基础设施和相关工具。

4. 挑战 Cosmos 应用链、Avalanche 应用链叙事。Base 基于 OP Stack 开发。目前 OP Stack 下已有 Optimism 和 Base 两条备受关注的区块链，未来随着 Layer2 生态的繁荣很可能会出现更多 OP Chain 并组合成 Superchain。在未来，OP Stack 的 Superchain 对 Cosmos 和 Avalanche 的应用链叙事会是一个重大打击。
