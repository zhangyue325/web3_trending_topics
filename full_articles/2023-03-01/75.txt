「NFTfi」，即 DeFi 解决方案与 NFT 用例的结合，是目前加密经济中增长最快的领域。

这一领域已经有非常多的项目可供探索，要想知道从哪里开始可能会让人眼花缭乱。Bankless 介绍了五个鲜为人知的项目，对 NFTfi 新人来说，既直接又充满吸引力。

目前，DeFi 总锁仓值（TVL）约为 490 亿美元，而 NFT 市值超过 150 亿美元。 

这意味着 DeFi 和 NFT 生态系统是当今加密经济中最大的领域。然而，这些领域正日益互相渗透，产生了「NFTfi」类别 -- 即在 DeFi 和 NFT 的交集点提供服务的项目。

当然，NFT 是独特的数字资产，代表对特定内容的所有权，而 DeFi 则是关于创造去中心化和开放的金融产品。当你把这两者结合起来时，你就会得到一个新世界，为 NFT 持有人提供一个绝佳的金融机会。 

这些机会有一系列不同的形式，从抵押 NFT 贷款到利用流动性协议上的闲置 NFT 进行收益农耕。今天，一些比较知名的 NFTfi 类别和项目正在创造这类机会，包括：

* NFT 自动做市商（AMM） - NFTX, NFT20, sudoswap
* 借贷协议 - Arcade, BendDAO, NFTfi
* 衍生品协议 - Hook, nftperp, Solv
* 定价协议 - Abacus, Upshot
但是，假设你是 NFTfi 新人，并发现这一切都非常有趣，但不确定从哪里开始入手。在这种情况下，你会想从使用起来相对简单，而且不需要持续跟踪一堆信息的项目开始。

我在 Metaversal 的研究过程中密切关注 NFTfi 的情况，所以我最近一直在策划一个相对简单的合作项目清单。今天我有一些很酷的东西要与你分享。

当然，把这些当作早期的实验，因为大多数都是很新项目。如果你想尝试其中一种，那么投入的金钱永远不要超过你能承受的损失范围。本着 "稳健第一 "的原则，这里有 5 个全新、未被大众熟知的项目，你可以考虑将它作为首次进入 NFTfi 的起点。

1.Caviar LPing
--------------

![](https://hx24-prod.mars-block.com/image/crawler/2023/03/01/1677654065528844.jpg)

  


Caviar 是一个用于交易 NFT 的 NFT AMM 协议。它支持 ERC-20 碎片化 NFT，提供了一个独特的「可取性分类器」系统，根据可取性的分数将 NFT 集合组织成子池。因此，每个 Caviar 主池有多个独立的价格曲线，以说明整个 NFT 集合的流动性。通过子池系统，该协议旨在增加较稀有的 NFT 的流动性，并激励流动性提供者（LP）的交易活动。

Caviar 的 LP 从每笔 NFT 交易中收取 1% 的费用。随着越来越多的人通过一个特定的资金池进行交易，该资金池的 LP 相应地可以赚取更多的收益。那么，这里的策略是用 ETH 和 NFT 在活跃的 Caviar 池中做 LP，从长期的交易活动中获利。例如，你可以在下图中看到，根据过去 30 天的交易活动，Caviar 上的 Cool Cats LP 目前正在赚取 58% 的年利率。

![](https://hx24-prod.mars-block.com/image/crawler/2023/03/01/1677654065512799.jpg)

2.Papr
------

![](https://hx24-prod.mars-block.com/image/crawler/2023/03/01/1677654065527758.jpg)

Papr 由 Backed 创立并受 Squeeth 启发，是一种新型的 NFT 借贷协议，被设为灵活、有弹性且无锁定期。这种新方法的关键在于代币 papr（“Perpetual APR”），借款人在将 NFT 作为抵押品发送到 papr 控制器智能合约时可以铸造这些代币。随后借款人可以根据需要在 Uniswap 等 DEX 上出售这些代币，这种市场活动在 papr 交易价格和协议利率之间创造了一个持续的反馈循环。 

![](https://hx24-prod.mars-block.com/image/crawler/2023/03/01/1677654065518747.jpg)

papr 代币为持有者和 LP 提供了一种低维护成本、「买入并持有」的方式，可以同时接触到多个 NFT 收藏品和贷款，因为它以其基于反馈的价格增值系统的形式提供了潜在的利益。例如，第一个这样的代币，$paprMEME，提供了触及 10 个 NFT 项目的贷款机会，包括 Cool Cats、Forgotten Runes Wizard's Cult 和 Mfers。

3. insrt finance ShardVaults
----------------------------

![](https://hx24-prod.mars-block.com/image/crawler/2023/03/01/1677654065518867.jpg)

insrt finance 是一种智能收益策略 NFT 协议，其第一类产品被称为 ShardVaults。这些金库被描述为「部分 NFT 现货敞口与被动收益耕作相结合」，通过 Shard 提供低价的蓝筹 NFT 所有权，Shard 是代表 ShardVault 的基础 NFT 的部分所有权的 NFT，例如 Bored Ape。随后，基础 NFT 本身在 NFTfi 中运行，为该 NFT 的组成 Shard 持有人产生收益。例如，insrt 的第一个 ShardVaults（为 CryptoPunk #9620 和 BAYC #1369）现在都为其 Shard 持有人产生了超过 15% 的年利率。

ShardVaults #1、#2 和#3 已经被完全铸造出来了，所以在更多的金库被推出之前，获得这些基于 NFT 的初始收益率的唯一途径是在像 OpenSea 这样的二级市场上获得 Shard。如果你这样做了，那么你将自动开始从相关的 NFT 的部署中获得被动收益。

4.MetaStreet Capital Vaults
---------------------------

![](https://hx24-prod.mars-block.com/image/crawler/2023/03/01/1677654065523444.jpg)

MetaStreet 是一个去中心化的元宇宙利率协议。其最初的旗舰产品是 Capital Vaults，支持人们存入 ETH、DAI 以及其他加密货币，从基于 NFT 的金库策略中获得收益。具体来说，Capital Vaults 使用这些存入的资金来收购通过 nftfi.com 等借贷项目促成的基于 NFT 的 P2P 本票，这些本票通过利息支付和清算提供收益。在这个意义上，你可以把 MetaStreet 的 Capital Vaults 看作是 NFT 借贷方流动性的聚合器。

![](https://hx24-prod.mars-block.com/image/crawler/2023/03/01/1677654065530226.jpg)

在这里，我们的想法是将资金存入 MetaStreet 的一个 Capital Vaults 以获得被动收益。由于储户可以选择在高级（高风险）和初级（低风险）金库头寸之间分配多少资金，你赚取多少年利率取决于你如何评估风险。例如，通过 General-WETH 金库的 30%/70% 划分，目前将产生约 20% 的年利率。

5.Hook Earn
-----------

![](https://hx24-prod.mars-block.com/image/crawler/2023/03/01/1677654065524039.jpg)

Hook 是一个去中心化的 NFT 衍生品协议，专门研究欧式看涨期权，它允许人们在特定的时间内对 NFT 进行叫价，而不必为该 NFT 支付全部的价格。要深入了解这里的基本细微差别，请查看我去年写的关于 Hook 的指南。简单来说，Hook 让 NFT 持有人赚取 NFT 的溢价，或以最佳价格出售，同时为交易者提供机会，从投机性多头或空头中赚钱。

Hook 的最新产品是 Hook Earn，它由该协议下的看涨期权驱动。这项新服务允许 NFT 所有者从他们闲置的 NFT 上获得即时收益，只要他们同意在未来某个特定的日期以特定的价格列出上述 NFT。如果在到期日没有购买，你可以保留 NFT，并可以将其重新存入盈利系统。如果一个 NFT 的售价高于其上市价（始终高于集合的地板价），那么参与的交易员就可以赚取差价。因此，"交易商正在为这些未来的潜在收益向 NFT 持有人预付费用，"Hook 解释道。还要注意的是，Hook Earn 是一个全新的项目，所以在流动性积累的同时，仍然有一个候补名单。你可以在项目的网站上注册，抢先体验。
