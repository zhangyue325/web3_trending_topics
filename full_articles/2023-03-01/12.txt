基于比特币网络的 NFT 协议 Ordinals 让比特币第一次有了金融用例之外的可能性，虽然在比特币原教主义者眼中 NFT 违背了「点对点电子现金」这一「初心场景」，但毫无疑问，在比特币上发行自己的 NFT 已成为很多创作者们想要尝试一回的愿望，寻求新叙事的藏家们更不会放弃这一投资机会，[Dune Analytics](https://dune.com/dataalways/ordinals) 数据显示，Ordinals 协议自发布以来累计铸造 NFT 已突破 20 万枚。

*![](https://img.foresightnews.pro/202303/5-1677639912201.webp?x-oss-process=style/scale70)NFT 领域最强 IP 缔造者 Yuga Labs 也将进军比特币区块链，将推出比特币 NFT 系列 TwelveFold*

 

硬币的另一面是，比特币生态中缺乏相应的 NFT 基础设施，无论是对于想要尝试在比特币网络中发行 NFT 的创作者，还是想要购买和交易的投资者，比特币网络至少是在短期无法满足它们的需求。

由 Loot 核心骨干 [Tim Shel](https://twitter.com/TimshelXYZ) 和 Zora 工程师 [Iain](https://twitter.com/isiain) 发起的实验性 NFT 项目 Chainspace 的诞生为我们提供了另一种解法，那就是在比特币网络中存储 NFT，并通过在以太坊链上创建对应的镜像 NFT 进行初始发行，这样持有者无需拥有比特币钱包就可以获得一枚比特币 NFT，此外以太坊上的镜像 NFT 兼容 EVM，因此可解决比特币 NFT 因缺乏基础设施无法被交易和流转的问题。

通过这种方式，不仅可以降低铸造门槛实现顺畅的项目初始启动，同时，还可充分利用以太坊生态中的 NFT 二级市场、做市平台等丰富的基础设施，为 NFT 项目的流动性和价格发现创建基础条件。而 Chainspace 项目的 800 个应用本体将永远存储在比特币网络中，即使没有以太坊也可以访问，并将永远存在。

Chainspace 初始发行的 780 枚 NFT 在一分钟内售罄，而发行 10 天以来，交易量已突破 600ETH，对于一个带点实验性质的比特币 NFT 项目来说，Chainspace 可以说是成功了，Chainspace 计划将这种双链结构的合约代码开源出来，为更多创作者所用，这意味着无需专门开发团队，普通的创作者也可以轻松地通过简单 fork 在 NFT 上发行自己的 NFT 项目。而这才是这个项目背后更远大的目的——为比特币全新生态引入更多新用户。

作为一个实验性项目，Chainspace 的探索具有重要意义。

它是首个双链项目，跨越以太坊和比特币，用 Tim Shel 的话说是一个 200% 的纯链上项目。 

叙事之外，Chainspace 实际上是为了解决实际问题，在目前比特币生态基础设施普遍缺乏而 NFT 用例需求旺盛的情况下的一个折衷方案，即将核心要件永久存储在 BTC 网络中，独立运行（即使没有 ETH 也可以访问），同时充分利用可用的以太坊生态资源，通过以太坊上的镜像 NFT 改善用户体验和解决后续的流动性问题。

一旦 Chainspace 开源这种双链结构的合约代码，比特币生态中将有望生长出更多的 NFT 项目和应用，甚至会遍地开花。随着比特币 NFT 的生态以及基础设施正在不断的增加和完善，后续从以太坊向比特币转移 NFT 自动化工具作为一项紧迫需求，或将会吸引更多开发者创建连接双链的「中间件」工具。

以下将对 Chainspace 进行简要的项目介绍，并通过专访创始人 Tim Shel 了解它的使用场景，未来计划和叙事潜力。

![](https://img.foresightnews.pro/202303/5-1677640139400.webp?x-oss-process=style/scale70) 

Chainspace 是什么？
---------------

Chainspace 是一个基于比特币网络协议 Ordinals 的 NFT 项目，包括 800 个存储在比特币网络中的视频渲染应用，你可以用它来自拍，使用预设的、代表不同加密社区美学风格的滤镜，生成自拍版的 gm 表情符号，进行身份表达和社区互动。

不过，说它是比特币网络的 NFT 项目也不完全准确，事实上，Chainspace 同时存在于以太坊网络上，具体来说，800 个 Chainspace 入口（Portal）铭刻在比特币网络中，起始于 Inscription 10,004，然后通过 tokenURI 从 BTC 映射至 ETH 中。不涉及 AWS, IPFS 和 Arweave，用开发者 Tim Shel 的话说，这是一个 200% 的纯链上项目，它跨越以太坊和比特币网络，并且可在以太坊和比特币网络中转移。

之所以选择在以太坊上创建镜像 NFT，完全是基于藏家体验的考量。2 月 17 日，Chainspace 在以太坊上进行了初始发行，780 枚 Chainspace 在不足一分钟的时间内售罄，剩余 20 枚称作是超入口（X-Portals）的 NFT，将在未来通过拍卖（18 枚将被用于拍卖）和赠送（2 枚将作为礼物赠送给 Loot 发起人 Dom）进行分配。

作为首个双链结构的 NFT，Chainspace 特殊之处在于充分利用了目前阶段的可用资源和基础设施，在初始发行和交易增长上取得了不错的成效。它首先从以太坊上启动，并通过以太坊上丰富的 NFT 基础设施获得了早期增长，后续将通过[手动操作](https://twitter.com/TimshelXYZ/status/1627357246341656576)和自动化工具引导藏家将 NFT 托管至比特币钱包中。Tim Shel 称，此举是一种策略上的考量，目的是为了探索比特币网络早期 NFT 基础设施不够完善情况下，如何最大限度地调用可用资源。

专为加密原住民打造的数字链空间
---------------

如果看完前面的简介你还没有完全理解这个项目到底要做什么，那不妨打开官网手动操作一番，你立马会明白这是一个 for crypto community, by crypto community 的加密原生应用。

打开[官网](https://chainspace.app/)，选择 (tune to）进入任意一个加密社区频道，即可进入相应的社区入口（Portal）。

![](https://img.foresightnews.pro/202303/5-1677640195245.webp?x-oss-process=style/scale70)

*Chainspace 中来自各个加密社区的滤镜调频一览*

 

本质上，每一个入口都是一个基于网页端的 html 应用，它会启动你的手机 / 电脑摄像头，你可以用它进行自拍，或者拍摄你和背景物们的合影，然后用它对拍摄的照片进行像素化渲染。

每一个入口对应一种特定的加密社区风格滤镜，通常由字符、emoji 符号等构成，属于是 old-school 风格的 ASCII 艺术风格。选择其中一个社区，相当于选定了一个社区风格的渲染滤镜，你可用这款特定的滤镜渲染自拍图片，用它做你想做的任何事，比如以这种方式向你所属的社区发送 gm 打招呼。

![](https://img.foresightnews.pro/202303/5-1677640215785.webp?x-oss-process=style/scale70)

*来自 toadz 社区的 GM*

如果你是一名加密用户，肯定不会不明白 gm 的意思。gm 全称早上好，属于加密社区通用的 meme 符号，也是一种独属于加密社区的打招呼方式，当你向加密社区说 gm (fam)，类似于程序员输出 hello wolrd。

Chainspace 可以说是真人表情符号版本的 gm 生成应用，社区成员可以通过这样一款拍摄应用自拍，加上特定风格的滤镜渲染发送到社区 gm。某种程度上，这种图像自拍的方式比单一的 gm 文字表达更为丰富、更具个性，也更能实现自我表达，尤其是当你用所属社区风格的滤镜时，是一种社区成员才能懂的趣味，代表的是某种程度上的身份认同。

Tim Shel 认为，这种自拍版的 gm 方式比一个随机的 emoji 更引人共鸣，更有人的味道，也更能发挥人类的想象力，想象一下大家用自拍分享自己的感受、背后的书架和自己的脸。

![](https://img.foresightnews.pro/202303/5-1677640239619.webp?x-oss-process=style/scale70)

*Chainspace 最初启动时有白名单机制，值得注意的是，获得白名单条件之一就是使用 Chainspace 应用自拍向社区打招呼。上图为来自 Chainspace 官方 discord 中 wave-to-enter 频道的用户分享。*

 

可以说，Chainspace 是一个加密数字空间的自拍亭，和现实中的自拍亭不同，它无需付费，人人可用，并带有预设的社区风格的滤镜。

此外，保持匿名性是 Chainspace 另一个重要用例。当加密用户为自己的脸添加了某种 old-school 风格的 ASCII 滤镜，不仅实现了自我表达，而且也给自己的脸蒙上了一层抽象的覆盖物，这让加密用户可以保持自己的真实身份不被暴露。

![](https://img.foresightnews.pro/202303/5-1677640255127.webp?x-oss-process=style/scale70)

*Tim Shel 使用 Chainspace 召开「匿名」社区会议*

 

Tim Shel 称 Chainspace 是一个实验性项目，实验性意味着瞎折腾，不成熟，并不明朗的未来，而且极有可能不会成功，但同时，实验性也意味着生命力、一种可能性的探索，而从可见的人类历史经验我们会发现，实验性也是最大概率诞生 groundbreaking things 的地方。 

那么，对于 Chainspace 这样一个项目来说，它探索了哪些可能性？其独特之处是什么呢？以及在可见的未来，存在哪些潜力呢？

首个跨链的纯链上 NFT 项目
---------------

Chainspace 发明了一种全新的 NFT 发行方式，是首个跨链的纯链上 NFT 项目。它跨越的是比特币和以太坊网络，让一枚 Chainspace NFT 同时存在于两处成为可能，用来拍照的应用本体存在于比特币网络上，意味着只要比特币存在，它就永存，从这个意义上讲，它的永久存储比 Arweave 要大得多，其镜像 NFT 则存在于以太坊中，对应的是 800 个存储在比特币网络中 Chainspace 入口的所有权，并通过 BTC 网络中相应的 NFT 数据实现渲染。

存储于比特币网络中的 800 枚 Chainspace NFT，就像是一间间带有不同滤镜风格的数字自拍亭，作为一种实存于虚拟空间的数字工件，每一个 Chainspace 的地理位置由两个坐标所决定，一个是比特币的交易哈希，另一个是以太坊上的 tokenURI。

最简可行发行
------

在叙事之外，从根本性上说，Chainspace 发明的双链结构 NFT 合约模型是基于藏家体验考量的结果，通过采用一种最简可行发行（Minimum Viable Launch）模式，充分利用目前以太坊上可用的基础设施，将 BTC NFT 铸造体验上的障碍降至尽可能低，从而为项目的增长创造必要的基础条件。

比特币网络中 NFT 协议 Ordinals 的出现，让比特币网络可承载比以太坊更大的链上数据作为内容物，这意味着，NFT 这种类型的艺术载体可通过铭文的形式刻录在单个比特币网络中的 Satoshi 上，不仅保真程度远大于以太坊，而且还可以被交易和拥有。

更大的数据容量意味着 BTC 网络可被用于进行安全存储，这带来了巨大的想象空间。不过，目前比特币 NFT 基础设施并不完善，缺乏相应的用户应用和二级市场，考虑到这一点，Chainspace 选择在以太坊上发行了 800 枚镜像 NFT，对应的是 800 个存储在比特币网络中 Chainspace 入口的所有权，之后，持有人可通过其他操作将 BTC 网络中对应的 Chainspace NFT 申领至自己的比特币钱包。

Tim Shel 发布推文帖出早期尝鲜版 step by step [操作指南](https://twitter.com/TimshelXYZ/status/1627357246341656576)，引导持有人将 ETH 版本的 Chainspace NFT 进行置换，托管至比特币钱包中，它需要用户提供自己的比特币钱包地址。值得注意的是，一旦进行操作置换，原来的 ETH 版本将被销毁，因此，你将不能用它在以太坊上进行交易，不过，它仍然可在 OpenSea 市场中显示。当然你可以用 emblem.finance 将其换回以太坊版本。

这大大简化了用户拥有 BTC NFT 的门槛。对于玩转以太坊的 NFT 藏家们，可在无需拥有比特币钱包的情况下铸造自己的比特币 NFT，在铸造后，以太坊生态中丰富的二手市场和各类 NFT 相关应用和工具，也为该 NFT 的二手市场流转创造了条件。

目前，Tim Shel 正在开发一项实现从比特币到以太坊自动化置换 NFT 的解决方案，目前该方案还未完全确定，可以确定的是，该方案将是「无需信任、链上、无需许可、永久和异步的。」

毫无疑问，Chainspace 的实验性探索是成功的，数据显示，自 2 月 17 日发行以来，Chainspace 总交易量为 601ETH。

为比特币生态引入更多新用户
-------------

Chainspace 本身的实践探索证明了这种最简可行发行模式是可行的，而接下来，作为开发者的 Tim Shel 和 Iain 计划开源这种双链结构模型的合约代码，以让更多创作者和开发者使用。

Chainspace 的合约本身就是一个超结构，其中包含了以太坊和比特币两条链，这意味着可充分利用两条区块链各自优势，其中 BTC 用于可靠的永恒存储，而 ETH 则可用于互操作性和兼容 EVM 链应用。

显然，Tim Shel 不是唯一想要在 BTC 上发行 NFT 的创作者，对于更多想要在 BTC 网络中尝试发行 NFT 的创作者，则可直接使用 Chainspace 即将开源的双链结构合约代码，以最小可行发行模式现在以太坊上启动自己的项目，而在现有的可用基础设施情况下，这是最大限度地调用可用资源的有效方式。

「这种模式的好处是，核心要件独立存在于 BTC 上，即使没有 ETH 也可以访问，并将永远运行，而以太坊上的镜像 NFT 兼容 EVM，因此有利于 NFT 被更方便地持有、流转和交易。」Tim Shel 如是说道。

「Chainspace 想要探索的是如何将更多新用户导入到一个全新的 BTC 世界。」这才是 Tim Shel 所说的打造这个项目背后真正的目的。

![](https://img.foresightnews.pro/202303/5-1677640343098.webp?x-oss-process=style/scale70)

*Chainspace 社区用户生成的 gm 表情包*

 

随着 BTC 基础设施的成熟，我们期待看到更多应用的崛起。比如，对于 Chainspace 这样的应用来说，一个极具想象力的基础设施可能是一个桥接比特币和以太坊 NFT 的中间件应用，一款可实现从以太坊向比特币转移 NFT 的自动化解决方案。

数字世界的公共物品
---------

围绕 NFT 的一个争论是，NFT 只有拥有才可访问，通常附带着某种特有的权益，仅供持有人享有。而 Chainspace 的存在则破除了我们对 NFT 这种单一的想象。

Chainspace 是存储在比特币网络中 800 个视频渲染应用，可被任何人用于拍摄照片，无需付费，人人可用，属于是一种公共设施。任何人都可以用其中任何一个入口拍摄图片，然后使用这些生成的作品做任何事情，无论在社区分享、上链铸造成 NFT 还是生成衍生艺术等。虽然使用权是面向所有人，但这 800 款自拍亭应用却以 NFT 这种载体的形式拥有所有者，这个层面上，它又是一种数字资产。

既然每一个入口都可免费访问，对于拥有者来说并没有什么特殊的权益，那么，为什么人们想要出钱购买这样一个东西呢？

Timshel 称，可以将 Chainspace 类比为现实中的艺术装置，虽然是公共物品，对所有人开放，但是仍然被特定的资助人所有。作为一个加密原生的数字公共空间，Chainspace 将会随着 Ordinals 和早期链上 BTC 铭刻的趋势增长，成为一股不可忽视的作用力，而越来越多的链上媒介和 Web3 实验的出现，会让人想要拥有一个 Chainspace 入口。
