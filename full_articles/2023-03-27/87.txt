我们很高兴地宣布，Steadefi 正式加入 [Chainlink BUILD 计划](http://mp.weixin.qq.com/s?__biz=MzU0MTgyMDQwNQ==&mid=2247495052&idx=2&sn=6fcba7f6803a213a9ec18030f7bb52bd&chksm=fb26b916cc513000c1b43efe0d4d1ea253830f9c40efdae342484f1b2b7386ab1432febe0e47&scene=21#wechat_redirect)。作为 BUILD 的一部分，我们将获得行业领先的 Chainlink Web3 服务、技术专业知识和来自 Chainlink 生态系统的社区支持。这将有助于我们更好的采用自动化智能对冲策略。作为回报，Steadefi 将在 Chainlink Economics 2.0 中向 Chainlink 服务提供商贡献我们总通增供应量的 3%。

  


**针对 DeFi 的优化风险调整策略**

  


Steadefi 是一种次世代 DeFi 协议，旨在开发对所有人适用的高资本效率、可调整风险的策略。与多样化的最受信任的协议集成之后，我们为用户提供了各种精心设计的策略库，所有这些库都具有自动化头寸管理和严格控制杠杆。

  


**为什么我们加入了 Chainlink BUILD**

  


为了能让 Chainlink 网络使得 Web3 协议提供的安全性、可靠性和去中心化优势最大化，Steadefi 加入 Chainlink BUILD。作为 BUILD 的一部分，我们将获得新的 Chainlink 产品 alpha 和 beta 版本发布的访问权限，同时还有来自 Chainlink 生态系统的专业技术支持等服务。

  


“ 项目的负责人 Jeff Lam 表示：“我们很高兴加入 Chainlink BUILD，我们相信这将极大地帮助推动 Steadefi 的下一代 DeFi 自动化策略库的采用。Chainlink 的 Web3 服务对于任何重视安全性和可靠性的协议都是必不可少的，我们计划利用 Chainlink 可以提供的完整服务套件，例如 Chainlink Data Feeds 和 Chainlink Automation 来支持我们的策略库。最终，加入 BUILD 将帮助我们为更多人提供访问 Steadefi 创新、高效、安全策略的机会。”

”  


作为我们对 BUILD 计划的承诺的一部分，Steadefi 已经承诺将我们总通证供应量的 3％用作 Chainlink 服务提供商（包括质押者）的经济激励。通过调整经济激励，Chainlink 和 Steadefi 社区有望相互支持并从合作中受益。

  


**关于 Chainlink**

  


Chainlink 是业界标准的 Web3 服务平台，已经在 DeFi、保险、游戏、NFT 和其他主要行业中实现了数万亿美元的交易量。作为领先的去中心化预言机网络，Chainlink 使开发人员能够构建功能丰富的 Web3 应用程序，并无缝访问任何区块链上的真实世界数据和离线计算，并为全球企业提供通往所有区块链的通用网关。

  


通过访问chain.link或阅读docs.chain.link上的开发者文档来了解更多有关 Chainlink 信息。如需讨论集成，请联系专家。

  


**关于 Steadefi**

  


Steadefi 是一款先进的杠杆收益聚合器，专注于最大化各种来源的收益，并自动管理市场风险。通过为自动化收益策略提供非足额抵押的杠杆，Steadefi 既满足了追求高收益高风险的投资者，也满足了风险厌恶型出借人。他们目前推荐使用的策略库是建立在 GMX 永续交易平台上的 3 倍多头 GLP 库。
