让我们回顾近期重要进展。**MantaPay 上线：全行业最快的、基于 ZK 的隐私转账协议**MantaPay 是基于 ZK 技术的、支持多资产的去中心化隐私转账协议，用户可将多种类型的资产，桥接至 MantaPay，实现隐私转账。也可支持 ERC721、ERC1155 等 NFT 资产转账，及 SBT 创建。MantaPay 使用 ZK 技术，既保证转账信息 100% 无法追踪，也提供 Viewing Key ，支持用户在需要时自主披露信息。目前，MantaPay ZK 交互速度全网最快，可短至 2-4 秒。MantaPay 已正式 Calamari 上线，待 Manta Network 主网上线后，将正式于主网部署。入口：*https://app.manta.network/calamari/transact*可查看关于 MantaPay 的更多信息（[Manta Network 正式在先行网 Calamari 上线 ZK 隐私转账协议 MantaPay](http://mp.weixin.qq.com/s?__biz=MzkxMDAwMjM2Nw==&mid=2247484914&idx=1&sn=2738db9c36f3f364c875aefdbe8938c8&chksm=c1335422f644dd34917791d96dd1919b02bbf2c0fcaeb1bddd5071f8da1f9a2701c3c3b26900&scene=21#wechat_redirect)）![](https://img.foresightnews.pro/202303/2531c65d20487835dc5f5f7ccd044f01.png?x-oss-process=style/scale70)

**NPO（NFT Private Offering）：基于 ZK 的隐私 NFT/SBT 发行平台**NPO 是 zkNFT/zkSBT Launchpad 平台，开发者和项目方可直接调用 API，实现 zkAddress 和 MantaPay 的 ZK 电路调用，创建隐私 zkNFTs/zkSBTs。通过 ZK 生成 Proof Key，可实现链上生成 SBT 等 DID 信息，在链下验证，甚至可以与 Web2 应用程序实现即插即用的结合。Proof Key 是一种协议，允许用户在链下验证链上 DID 身份，无需钱包，甚至移动端 App 也可轻松检查 zkSBT 身份 / 所有权。移动端应用开发人员可轻松地将 zkSBT 直接集成到 Web2 应用程序中。用户仅需将 Proof Key 复制到移动端 App 中，与链上身份对应即可。可查看关于 NPO 的更多信息（[Manta Network 推出基于 ZK 的隐私 NFT/SBT 发行平台 NPO（NFT Private Offering）](http://mp.weixin.qq.com/s?__biz=MzkxMDAwMjM2Nw==&mid=2247484938&idx=1&sn=715f0c47fd9956c86faf0b15804b125a&chksm=c13357daf644deccdda46cb846a28334c1622ba2bbc397791c248947c6ef0a882f522d0f5e21&scene=21#wechat_redirect)）![](https://img.foresightnews.pro/202303/2b11c44abdad5b842066aff2b08bacad.png?x-oss-process=style/scale70)

**关于 Manta Network**

Manta Network 是基于 ZK 的 Layer1 公链，拥有最高的 Prover 效率和最去中心化的可信设置，致力于为 Web3 带来可编程隐私。核心产品和技术组件，包括 zkNFTs 和 MantaPay 等等，为用户提供友好的、可访问的、强大 ZK 应用。 

项目创始团队履历丰富，来自哈佛大学、麻省理工学院和 Algorand 等著名机构。团队自 2019 年在麻省理工学院数字货币倡议下，以及新加坡金融管理局跨境支付研究项目中，开始了对零知识证明的研究。

Manta Network 已获得多家顶级 Web3 基金的投资，包括 Binance Labs 和 Polychain Capital，且通过参与最好的 Web3 加速器，包括 Alliance DAO 和 Berkeley Blockchain Xcelerator，实现发展。Manta Network 已准备作为下一代 Web3 用户的入口，开创以隐私为重点的 Web3 应用新篇章。
