* 外媒：推特称部分源代码遭泄露，并已展开调查

‍

根据一份法律文件，推特的部分源代码在网上遭到泄露。报道称，推特于上周五向发布这些代码的软件开发人员在线协作平台 GitHub 发出版权侵权通知，要求删除代码。 GitHub 已对这些代码作删除处理，并补充说不清楚这些代码泄露了多久，可能至少有几个月。推特还要求美国加州北区地方法院命令 GitHub 确认分享代码及下载代码的人员。 该报援引两名了解内部调查情况的人士称，推特对此次泄密事件已展开调查。

  


* 知情人士：交通银行、中国银行、浦发银行香港分行或向加密公司提供银行服务

  


‍知情人士称，中国交通银行、中国银行、上海浦发银行的香港分行已经开始向当地加密货币公司提供银行服务，或已经对该领域进行调查、讨论内部信息。知情人士称，至少有一次一家中国的银行销售代表前往一家加密货币公司的办公室，推销其服务。
