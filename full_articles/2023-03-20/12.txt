Manta Network 基于 ZK（Zero-Knowledge Proof，零知识证明）推出隐私 NFT/SBT LaunchPad 平台 NPO（NFT Private Offering, NFT 隐私发行平台）。  
NPO 是 NFT/SBT Launchpad 平台，开发者和项目方可直接利用 zkAddress 和 MantaPay 的 ZK 电路，创建隐私 NFTs/SBTs。用户可直接在 Manta 链上，使用公开地址上的公开资产，支付 Mint 费用。NPO 可让 Manta Network 上的任何项目或用户，在没有任何技术或 ZK 密码学知识的情况下，发行基于隐私的 zkNFTs/zkSBTs。本文将全面介绍 NPO 和 zkNFT/zkSBT 各自的功能及优点。zkNFTs/zkSBTs 背后的 zk-SNARK 技术，实现了无摩擦的 NFT 铸造过程，且可在链上或链下进行验证，无需披露公共地址。zkNFTs/zkSBTs 与 Web3/Web2 应用程序实现即插即用，开发者可将其应用于移动端 App 中，创造基于 NPO 的全新应用。**什么是 NPO ？**NPO 是去中心化 zkNFT/zkSBT Launchpad 平台，将 ZK 与 NFT/SBT 结合，实现隐私 Mint 和验证。使用 NPO，用户实际上是在 zkAddress 上铸造隐私的 zkNFT/zkSBT。NPO 是改变 Web3 游戏规则的产品之一，实现了 zkSBT 的发行，这是一种隐私且用户友好的 SBT。利用 ZK 电路，NPO 提供了安全、隐私和去中心化的方式，让用户铸造、验证 NFT/SBT。**什么是 SBT ？**灵魂绑定代币（Soulbound tokens，简称 SBTs）是数字身份代币，代表一个人或实体的特征、特点、性格和成就，建立在区块链之上。与 NFT 不同，SBT 不能转移或交易。2022 年 1 月 26 日，Vitalik Buterin 在博客上发表了关于 Soulbound 文章。在文中，Vitalik Buterin 讨论了为 Soulbound 增加隐私的挑战，他指出，随着身份证明越来越广泛地采用，隐私变得越来越重要，因为所有 on-chain（链上）活动都可直接与个人身份联系。为解决这一挑战，Buterin 提出几种技术选择，包括将物品存储在一个索引、接收者地址和接收者私钥的哈希地址上。这是隐私的。因为只有拥有私钥的人，才能看到哪些物品属于接收者。其他选项包括发布 items 的哈希，并为每个接收者提供 Merkle 分支，使用 ZK-SNARK 证明物品的所有权，以及使用 ZK-SNARK 促进链上转账同时保持隐私。Buterin 强调，隐私是使这种生态系统正常运行的重要组成部分，特别是在用户可能不想披露自己拥有的一切的情况下。他建议将隐私纳入设计中，可以帮助避免不好的结果，并增加使该协议更成功。**什么是 zkSBT？为什么在 Manta Network 上使用 zkSBT？**与传统 SBT 相比，ZK 提供了安全和隐私。与 SBT 类似，zkSBT 同样易于 Mint 和使用，提供相同的用户体验，但增加了隐私保护特性。作为 Manta Network 成为 Web3 隐私中心使命的一部分，zkSBT 支持多链。换句话说，用户可在以太坊、BNB Chain、Polygon 等其他链上发行 SBT，通过 zkSBT 在 Manta Network 链上获得隐私凭证，并通过 Proof Keys 验证，无需披露任何钱包信息。链上节点仅需验证 ZK 证明有效性本身，除此之外，不会获得任何信息，以实现安全和隐私铸造。要了解 zkSBT，首先要了解 Manta Network 开发的两个关键技术 zkAddress 与 Proof Key。**zkAddress**zkAddress 是 Manta Network 使用 ZK 技术原创的隐私地址类型，可重复使用、独立且可审计（[Web 3.0 隐私基建：什么是 zkAssets 和 zkAddresses ？｜Manta101](http://mp.weixin.qq.com/s?__biz=MzkxMDAwMjM2Nw==&mid=2247484553&idx=1&sn=9fa62f540b42de559f87053f9e9314e0&chksm=c1335559f644dc4fef83c5a4900ebe594242421a082cf6b577799c6ccfefab3246f7fef3eec4&scene=21#wechat_redirect)）。单个 zkAddress 可拥有多个 zkSBT，而单个 zkSBT 只能对应一个 zkAddress。zkSBT 不可转移。zkSBT 元数据与 SBT 一样灵活，可支持如 PFP、普通照片、AI 照片、社交 App 图片、Web2 数据抓手（Twitter，Instagram 等）等。由于 zkSBT 是隐私的，不能通过查看区块链浏览器来与身份关联。**Proof Key**Proof Key 是一种协议，允许用户在链上验证其身份，无需钱包。Proof Key 使移动端 App 可轻松检查 zkSBT 身份 / 所有权。为了证明自己的身份，用户需要将 Proof Key 复制到移动端 App 中，与链上身份对应，证明自己确实在链上铸造了 zkSBT。**zkSBT 用例包括：**通过元数据以隐私保护的方式验证个人资料图片，包括照片或 AI 生成的图像；1）在链上交易资产或加密资产，无需披露地址信息；2）验证用户在链上购买的游戏道具；3）验证去中心化的社交网络信息，而不涉及用户身份或社交身份之间的关联。值得注意的是，通过使用 Proof Keys，zkSBT 可在移动应用程序中进行验证，无需钱包签名或验证，移动开发人员可轻松地将 zkSBT 直接集成到 Web2 应用程序中。**部署 zkSBT：无需密码学家**部署 zkSBT 无需过深地了解编程技术或密码学知识。Manta Network 已将复杂性抽象出来，为任何开发者提供易于理解和使用 zkSBT 的简单体验。开发者可简单使用 SBT SDK 调用，SDK 包含自定义逻辑（custom logic），例如预充值步骤（pre-charge steps），用于预订（reserving ）SBT 资产 ID，并允许用户通过提供元数据来铸造 SBT。元数据有 300 个字符限制，可包含图像 URL 或任何其他数据。  
**关于 Manta Network**

Manta Network 是基于 ZK 的 Layer1 公链，拥有最高的 Prover 效率和最去中心化的可信设置，致力于为 Web3 带来可编程隐私。核心产品和技术组件，包括 zkNFTs 和 MantaPay 等等，为用户提供友好的、可访问的、强大 ZK 应用。 

项目创始团队履历丰富，来自哈佛大学、麻省理工学院和 Algorand 等著名机构。团队自 2019 年在麻省理工学院数字货币倡议下，以及新加坡金融管理局跨境支付研究项目中，开始了对零知识证明的研究。

Manta Network 已获得多家顶级 Web3 基金的投资，包括 Binance Labs 和 Polychain Capital，且通过参与最好的 Web3 加速器，包括 Alliance DAO 和 Berkeley Blockchain Xcelerator，实现发展。Manta Network 已准备作为下一代 Web3 用户的入口，开创以隐私为重点的 Web3 应用新篇章。
