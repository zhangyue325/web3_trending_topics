首先，这不是一篇教你撸毛的文。

  


只是最近 Arbitrum airdrop 的 Token 再次暴富了一批人，于是乎，几乎被人遗忘的羊毛党又再度活跃了起来！

  


那么未来几个月，有哪些可能，只是说可能的大毛机会呢？**一起来捋一遍, 顺便盲猜下各个项目可能的规则框架。**

  


01 ETH 生态
---------

毫无疑问，ETH 几个 Layer2 生态项目，是最为「确定性」的大毛。

  


* **Zk-sync**
  


ZK-Rollup 的两大扛把子之一，撸毛党扎堆的拥挤之地，然而你要是现在才去，估计会有点晚，估计 V1 交互之类的肯定也会占不少权重，V2 现在的交互人数已经多到可怕。

  


* **Starkware**
  


官方跨链桥上的 ETH 也是肉眼可见的在增多，应该很多也是去撸毛交互的，Starkware 和其他几个 L2 还不太一样，毕竟人家一开始就只在折腾 L3，所以估计 Immutable X，dydx 等早期用户拿到Token的权重会不小。

  


* **Scroll 与 Fuel**
相对前三个，这俩的名气没有那么大。也没有太多可以交互或是撸的信息。但往往越是这种人少的地方，越有大毛的机会。

  


**Scroll 是华人团队的 ZK-Rollup**，但在业界的名气和关注度也是越来越高，**相对于 ZK 的两大扛把子，Scroll 主打的是更加完善的 EVM 兼容性。**

  


**Fuel 则是除了 Arb 与 OP 另一个很有实力的 OP Rollup，**且定位是一个模块化的执行层，并非专属 ETH 生态，又是非常另类的 UTXO 主打并行计算型，非常值得一看。

  


02 Cosmos 生态
------------

**随着 Cosmos 生态的崛起之势，以及 Appchain 和模块化区块链路线方向之正确性，**所以未来几个月，Cosmos 这边的大毛机会，应该不少

  


* **GNO**
  


这项目，感觉注定是大毛，如果 Aptos 和 Move 系是资本推动的新公链，GNO 到时候肯定就是社区推动的典范了，项目大概率明年初会看到。但是现在去撸有点晚，因为Token半年前就发给了 ATOM 的质押者 ( 没法交易），**当然，后续会不会有别的规则，现在也说不准。**

  


* **Celestia**
  


Celestia 这半年多名气已经大的不能再大，**提到模块化第一个想到的必然是他, 最近熊市几千万美金的融资也毫无疑问的彰显了未来独角兽项目的地位。**

  


Celestia 会分发Token么？感觉大概率会，毕竟分发Token是 Cosmos 系的一惯传统，而除了 Atom 质押者，Osmosis 质押或 LP 之外，Celestia 的测试网估计也会是Token 分发的权重之一，技术流的小伙伴不应该错过。

  


* **Sei**
  


之前的文章有写过《[热度比肩以太坊的 Cosmos 生态现在发展得怎么样了？（下）](http://mp.weixin.qq.com/s?__biz=MzUyNzE4MDM2MA==&mid=2247519582&idx=1&sn=61a730f9e38870ce4d00b6ebf4963f81&chksm=fa01ba79cd76336ff7f7942d9156d7812a1da5b81d298775fa04dd5c7ca38a5a51595ee39285&scene=21#wechat_redirect)》，是一个基于 Cosmos，专门针对 DeFi 优化的 DeFi Chain，之前发布过消息：**表示 1% 的 Sei Token 供应将分配至激励测试网。**引起不少人的不满，1%？？少点了吧兄弟……

  


但其实要知道，Aptos Token 分发的数量也就 2%，所以 1%，其实也没你想象的那么少。

  


03 Move 系
---------

* **Sui**
  


这个不用说，Move 系双雄，Aptos 匆忙发了 Token 之后，**现在整个加密圈最卷的项目一定是 Sui，没有之一。**

  


* **Linera**
  


当初 Diem（Meta）出来的人马其实是兵分三路的，Aptos，Sui，还有 Linera，**只不过 Aptos 和 Sui 因为用了 Move 语言，所以名气大很多，融资的体量也大很多**。但 Linera 其实背后也有 A16Z 这种巨头，所以说不定因为这里的人少，分到的毛要比 Sui 大。保持关注。

  


04 小结
-----

写在最后：总而言之，撸毛这件事，除了不能三分钟热度，**要持之以恒之外，还有下面几个基本原则可以遵循：**

1. 账号贵精不贵多，OP 的 Token 分发规则是你最应该研究明白的；
2. 账号越早开始参与越好；
3. 保持链上的活跃度，多参与几个 Dapp，尽量保证每周都有个几次的交互；
4. 参与社区。包括 DAO 的投票治理啊、多签钱包、GitCoin 捐助之类的；
5. 多用几次桥。新的公链一开始肯定会有一两个桥支持，没事儿过去把资金跨一跨，很有可能也是交互权重的一种。
当然了，遵循了也不一定能撸到，最多能提高下胜率。至于撸不撸，这完全是个见仁见智的事儿~
