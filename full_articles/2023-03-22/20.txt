  
  
![](https://img.foresightnews.pro/202303/3390250af6b81fba6748a320700ceec3?x-oss-process=style/scale70)  
当下，蓝筹 NFT 之间正悄然出现分化。即便曾一度引发 Gas War，但不少头部 NFT 项目却因未能形成长期品牌叙事和社区赋能的增长飞轮，而使其价值难以支撑，持续下跌的价格和交易量也体现了这点。

![](https://img.foresightnews.pro/202303/d0fedeeecf4bbf83c06d4c4e8441f94e?x-oss-process=style/scale70)01交易额和均价齐下滑，大部分 Holder 仍为“钻石手”
============================

尽管 NFT 市场在高速发展后迎来了集体降温，但仍是加密领域最为出圈且重要的构成组件之一。在 NFT 产品类型日渐丰富的过程中，也被依据市场人气和潜力划分出不同等级。其中，部分 NFT 凭借强大的社区共识和运营手段成为蓝筹项目，占据着绝大部分的市场份额。

NFTGO 数据显示，截至 3 月 21 日，近一年来 NFT 总市值（仅以太坊）从最高超 332.5 亿美元跌至目前的 221.7 亿美元，跌幅达 33.3%；月交易额则从最高约 35.3 亿美元跌至 10.1 亿美元，下降了超 71.3%。NFT 的总持有者数已攀升至超 428 万，但近期增长步伐明显放缓，虽然近一年的增速为 113.3%，但自今年以来仅增长了 1.9%。与此同时，月交易者数量也出现了不小幅度的回落，从过去一年最高的超 41.4 万下滑至 18.6 万，跌幅约 55%。同时，从买卖家的数量变化看，近一年买家数量呈小幅下降的趋势，而卖家数量则大幅上涨，这意味着市场的购买力出现显著下降，投资者们更愿意出售手中的 NFT 资产。

而整体 NFT 市场的低迷除了与加密市场走熊有一定关系外，蓝筹项目的市场活动放缓和资金出逃也是主要原因之一。

![](https://img.foresightnews.pro/202303/59b7e11e5b4fa6cf95c17f7b84d9674d.jpeg?x-oss-process=style/scale70)

  


根据 NFTGO 数据显示，近一年蓝筹项目指数已下降了超 54.7%。与此同时，根据统计的销售额排名前十的蓝筹项目数据来看，这部分 NFT 市值占整体市场的 33.8%，近一年来，但它们的均价平均却下滑了 46.9%，交易量则下跌超 13.3%。其中，Art Blocks、ClONE X、Moonbirds 和 Doodles 的平均售价跌幅均超过 70%。

从蓝筹持有人数量来看，大部分 NFT 的变化不大，仅 Bored Ape Yacht Club、Mutant Ape Yacht Club、Azuki 和 CLONE X 的跌幅较多。不过，这些蓝筹项目的大部分 Hodler 均为“钻石手”，平均持有三个月以上的用户占比高达 75.1%。

作为 NFT 市场的中流砥柱， 蓝筹 NFT 被视为可以产生稳定回报的长期投资，但当前并不乐观的市场表现下 Hodler 是否还会继续长久持有？从近期种种迹象来看，现阶段已有不少蓝筹 NFT 的商业模式和运营策略正引发社区成员的不满。

![](https://img.foresightnews.pro/202303/d0fedeeecf4bbf83c06d4c4e8441f94e?x-oss-process=style/scale70?wx_fmt=svg)02多个蓝筹项目引发社区不满
============

从最初的思潮实验到颠覆商业模式，指数级增长的市值证明了外界对 NFT 价值捕获的认可。但近期市值规模的缩水也说明了市场对 NFT 的创新玩法提出了更高要求。

尽管蓝筹 NFT 项目已占据不小的优势，但差距已然拉开，比如 Yuga Labs 打造了 IP 帝国并赋予持有者完全的 IP 商业使用权利，其多个项目系列在蓝筹 NFT 板块占据着多个席位，而 Moonbirds、Doodles 和 CLONEX 等 NFT 在短暂成功后经营和商业模式正饱受持有者质疑。

比如，Moonbirds 在打出多个系列 NFT 项目、开放版权等玩法后仍难复往日辉煌，毫无新意的玩法更是引发社区质疑；而 Doodles 并未完全把 IP 权利直接赋予个人，且通过商品销售获得的收入限制在 10 万美元等 Web2 式的行为本就引发了其不够 Web3 化的质疑，而近期宣布在 Flow 区块链上开展 Doodles 2 计划后更是引发投资者不满，并其创始人 Jordan Castro 宣布 Doodles 将从一个“NFT 项目”发展成一家“领先的媒体特许经营公司”后还被社区成员炮轰其无视持有者权益，一度引发了抛售潮；知名运动品牌 Nike 旗下的 NFT 潮流品牌 RTFKT Studios 与艺术家村上隆合作的 CloneX 则因为该品牌给 NFT 持有者空投折价卷、非美地区无法兑换实体鞋等多个骚操作而遭到砸盘抛售。

社区的文化与价值决定着项目生命力，而社区文化影响项目成败的例子比比皆是，对于 NFT 项目而言，长期的不作为终究会被市场淘汰，毕竟鲜少有人能在多变的市场中一直拿着稳赢剧本。比如 Blur 凭借流动性攻击打破交易市场格局就是个很好的例子。当前，NFT 应用场景正在不断被拓宽，无论是比特币 NFT、AI 生成式 NFT 项目等新叙事，还是 NFTFI、动态 NFT 等新玩法都在加快蚕食市场份额的步伐，NFT 市场正面临洗牌潮。

总之，NFT 之所以区别于小图片，关键在于背后的价值，而社区成员在捕获价值的同时也会为项目本身价值提供支撑。对于这些蓝筹项目而言，如果不能寻找新的商业或发展模式来完成差异化转型，将项目价值回馈给社区，那么被替代只是时间问题。
