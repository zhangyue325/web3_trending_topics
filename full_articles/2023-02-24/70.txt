2022 年，ChatGPT 横空出世，聊天机器人这个很早就存在的模式，以全新的方式，再次展现到大家面前。

如何理解「新」？
--------

ChatGPT 的出圈，吸引了更大范围的注意力，激发了人们的好奇心。

**如何理解这里的「激发」？**具体来说，因为 ChatGPT 看起来是大家熟悉的「聊天机器人」模型，但是它完成的功能和能力，却大大超出了人们的预期，在有些任务的表现上甚至比人类还要棒。

这种热度表现在很多方面：比如说，大家开始讨论，究竟 ChatGPT 是否会彻底替代人类？大厂们开启了新一轮 AI 的军备竞争，各大媒体头条都把 ChatGPT 作为头条，成千上百的用户对 ChatGPT 感到好奇，开始积极地使用它们，很多企业和开发者也把这项技术作为重要的战略部署。

而这些现象后面的本质，其中之一在于，要掌握 API 的战略地位。

什么是 API？
--------

API 代表应用程序编程接口，它是一组用于构建和集成应用程序软件的定义和协议。

你可以把 API 理解为服务员，ta 很会 ta 工作的那一套服务，有事儿说话就行，API 也 属于前后端间的联通。举个例子，假如我们去一家饭馆，桌面上摆着一份菜单，而厨房能提供菜单上的菜式。那么，这里还缺少一个能和后厨联系的人，服务员就是去告诉后厨你选的菜，再将菜带回到你的桌子上。服务员带着你的 request 去告诉 system，然后将 response 带回给你。

这里再举个例子：airline system ，订机票，你将你的航程提交，会访问网站的 database,来看你选择的日期，座位是否是可选。但如果你不是在他的网站上订机票，你是在一个汇总不同的航班信息的 online travel service 上，那么这个 service 调用了 前面 airline system 的 API 。

这个接口就像上面例子的服务员，带着你的要求去 airline system 上 收集信息 ，然后带回到 online travel service 上 。所以你就把它想象成你的服务员在前后端间交付数据，创建联系。

我们继续往下讲：要知道，ChatGPT 与 Bing 的聊天机器人并不是这类产品的终极形态。

**ChatGPT 和 Bing，其实属于一种应用层面的展示，主要是强调这种基于 GPT 模型，如果做成应用工具，到底能有多么先进。**

**而正是这种演示效应，和在公众、企业、科研机构中造成的影响，导致生成式 AI 领域的军备竞争帷幕，徐徐展开。**

Octane AI 的创始人 Ben Parr 认为：「对于 OpenAI 来说，所赚的钱绝大部分将来自开发者，而 ChatGPT 只是通往所有业务的入口。」

**在人工智能聊天机器人进入公众视野之前，OpenAI 早就为开发者群体准备好了 API：**

**2023 年 1 月，ChatGPT 一跃爆红，而 OpenAI 负责人 Greg Brockman 就开始预告：ChatGPT 的 API 马上就会推出。就在几乎同一时间，微软提供了基于 Azure 的 OpenAI 业务。**

而其他大厂也没有停下脚步。谷歌是这样操作的：谷歌 CEO Sundar Pichai 推出 Bard 聊天机器人的当天，也承诺会在 2023 年 3 月之前，推出底层技术（类别的产品和支持），而就在本周，亚马逊宣布将与 Hugging Face 合作，通过 AWS（亚马逊云科技）提供生成语言工具。

英伟达（*NVIDIA* ）首席执行官黄仁勋，在本周三（2023 年 2 月 22 日）的电话会议上提出：「**所有开发软件的工程师，都会逐渐觉醒，或者是被现实教育，这种教育指的是，大家会逐步将类似于 ChatGPT 这类工具整合到自己开发的应用程序中，或整合到其他服务中。」** 由于英伟达公司（*NVIDIA*）提供了技术和算力运行的芯片，因此它也从这个领域中受益，英伟达股票周四跳涨了 14%。

寻找生成式 AI 的广泛而有用的应用，将会是一种挑战，但是，我们逐步会发现：很多早期应用会很突出，比如说：

* 客户服务部门可以使用能够对话式聊天机器人
* 游戏公司可以用它来建立智能角色，使 NPC 这种固定而古老的模式，成为历史。
* 而营销人可以试着使用生成式模型与客户建立更深的联系。
**整个行业正在迅速发展。**

2022 年 2 月 21 日（周二）OpenAI 宣布它与贝恩咨询合作，帮助客户构建自己的 API

而 OpenAI 的 Zack Kass 视频中提到，OpenAI 无法跟上人们对它技术的兴趣，现在 OpenAI 被企业的需求搞得应接不暇，尽管这样的热度和关注，大家等了很久，但是，现在重点还是在搞清楚如何解决这些问题。

可口可乐公司的高层也表示，**在可口可乐的营销工作中，也会考虑应用这项技术：目标是为了高效提供创造性的内容，**可口可乐公司的 CEO 昆西也提到，他相信这项技术将会改变知识工作。

如果我们看看可口可乐的发展，也许会发现一些新的思路。在投资人 Chamath Palihapitiya 最近的演讲中，他提到，**可口可乐获得商业上的成功，得益于制冷技术的发明。**

值得注意的是，可口可乐比发明冰箱的人赚了更多的钱，同样的事情也很可能在 AI 领域发生。也就是说：「如果我们把 AI/LLMs 比作是制冷设备，那么，谁会成为下一个可口可乐？」

如果我们用 Palihapitiya 的思路来判断这件事：**促成工智能应用的公司成功的公司，可以理解为上面比喻过的制冷技术，仍将捕获巨大的利益。**

所有开发底层技术的公司正在努力，争取在自己的平台上推出能够促成 AI 应用公司成功的设施，以期待能够从中分一杯羹。**例如，OpenAI 推出了 1 亿美元的创业投资基金，希望能够与医疗保健、气候、教育和其他领域的人工智能公司合作。**

**我们发现 OpenAI 投资的一些公司，确实存在有真正（且垂直）的应用案例。**

**在这场属于 AI 的热度进程中，API 是最为重要要素之一。**

这也就是为什么微软愿意在 Bing 中那么快就发布 ChatGPT，即使这个速度看起来那么不可思议，这也是为什么当机器人的缺陷暴露在公众面前时，也要继续往前走。这从来不是关于 Bing 或 ChatGPT 的问题，而是关于这些应用背后所预示的潜在未来。

**而现在，ChatGPT 和 Bing 成功地推向大众，揭开了 AI 行业的竞争序幕。**
