> “NFT 首次为作者和其他创作者提供了进入资本市场的机会。他们可以出售其作品的副本，而不是出售其创意份额，这将改变创意经济。”

从 17 世纪左右开始，版权法开始盛行，为文学、艺术作品的市场流通提供依据和凭证，成为保护创作者的关键。

美国批准《宪法》赋予国会创造版权的权力，此后版权的范围、期限逐渐扩大，但基本思想没有改变。版权赋予作者某些制作和传播其作品副本的专有权，以鼓励这些作品的创作和传播。如果没有版权，**作者的作品将成为“公共物品”，任何人都可以随意使用**。

将眼光放到艺术市场上来看，版权的存在让艺术作品更昂贵了。虽然这一方面激励着创作者、出版商源源不断推出消费者喜爱的产品，但也会出现版权价值远超作品价值的情况。面对这种情形，消费者不仅束手无策，而且感觉“**深受欺骗**”。也是在版权的笼罩下，一些毫无经济价值的作品依旧在市场立于不败之地，久而久之，艺术市场的活力尽失。

![](https://img.foresightnews.pro/202302/5947e5e23d45299056271d2261c969dd.png?x-oss-process=style/scale70)而在 Web3 热潮来临时，**NFT 的出现解救了艺术创意经济的交易困局**，转换了版权重要性的地位，将艺术品副本作为交易主体，持续为创意效益增长加码，**今天就让我们来看看 NFT 究竟有多大的能量。**  


**代替版权证明的 NFT**

NFT 的主要价值是：为每个创意作品创建由区块链支持的独特记录，编码其真实性和稀缺性的意义，让艺术家围绕版权转让、使用和制定执行规则。虽然艺术交易市场是实体的，而 NFT 市场是虚拟运作的，但 NFT 也让我们转换了对艺术市场交易的认知。**我们购买一件艺术品或 NFT 时，消费的是与艺术家相关的未来商誉、社会影响力。**

**![](https://img.foresightnews.pro/202302/b9b06f947a1f85d6783e43cdc64295c9.png?x-oss-process=style/scale70)**

### NFT 收藏爱好者购买艺术作品，更多是凭借作品未来的“受欢迎程度”去做一笔投资。因此，在 NFT 市场最主要的不是通过版权来赚钱。在这里，每个人都可以入局投资流通，依靠产品附加值提升而赚取额外收益，更透明、流动性更强。

### 关注 NFT 的伙伴们或许都知道，在去年 11 月 X2Y2 打响了**“ NFT 零版税之战”**，带头允许交易者不支付版税，以更低价购买 NFT 。这一事件让交易用户欢呼雀跃，同时也昭示着 NFT 市场逐渐**打破“躺平族”靠着版权获益的美梦**，转而激发创意生产、市场流通的活力。

![](https://img.foresightnews.pro/202302/97d0ab030c7185587c6437bea90b3327.png?x-oss-process=style/scale70)

正如麦克卢汉曾言，“媒介即讯息”。NFT 在区块链技术的加持下成为种一个崭新的投资交易方式，也转变了过往将版权放在核心位置的观念，**用户进一步从中介、创作者等“权力对象”中获得了更多收益的可能，促进艺术交易、创意经济的繁荣生长。**  


**NFT 如何改变创意经济**


> 在凯文·凯利的文章——《1000 个真正的粉丝》中提到：“互联网将改变创意活动的经济模式。成为一个成功的创造者，不需要数百万美元或数百万客户，而只需要成千上万个会购买你的任何产品的粉丝。”而 NFT 的出现能够加速创造者与粉丝交易的盈利趋势、增强粉丝对创作者的黏性，并持续在 NFT 市场参与交易，消解了以往版权横亘其中的困扰。

作为艺术家出圈、获得更多利益的新手段，NFT 对创意经济的推动作用毋庸置疑。实际上，NFT 不仅颠覆了艺术品交易的固有流程，还从作品本身改变了艺术创作的价值，为创意经济赋能。

![](https://img.foresightnews.pro/202302/200d81d859b6e3aaf2948ac64b127594.png?x-oss-process=style/scale70)

### 

**不伪造与可溯源**
===========

艺术作品的价值在于独一性、不可复制性，这一点在 NFT 艺术市场得以体现。加密艺术不能被伪造，**其所“铸造”的每件艺术品都被永远记录在区块链上，消解了真实性和合法性的所有怀疑（版权的实际作用再次被弱化）**。同时，艺术作品的出处有了确切的记录，为创作者设计各类新颖多样的方式来呈现创意作品提供背景，更好地服务于创意经济流通。

**稀缺性和流动性**
===========

对于资深收藏者和发烧友来说，艺术作品的稀缺是值得购买的关键。虽然老派美术世界充斥着无法审核的限量版和艺术家证明，但加密艺术已然解决了这个问题，**不言自明的版本唯一性为每个珍惜艺术品打上标签、摆上货架。**同时，传统美术界通常依赖奢侈品托运和价格不菲的艺术顾问协来转运艺术作品，但加密艺术可以在几分钟内实现线上转移，实现在 NFT 市场的自如流动。

**艺术价值无限增长**
============

在艺术 NFT 市场中，来自特定收藏家的“认可”会推动创意作品不断增加其附属价值。一旦创作者的作品被购买，交易价值转换为具体的销售数量和热度排行，在同一个平台上可以被世界范围内的爱好者查阅。**真正创作好作品的草根艺术家有了更多“出人头地”的机会，而不是被传统的权威标准所埋没。**

同时，基于 NFT 资产交易的平台合约也为创意经济带来了全新模式。艺术家能够自动从二级市场上的作品交易中获得收益，艺术作品的商业价值会跟由市场热度的动态变化，并随着作品在价值和交易中不断增长。  


**提供更好的创意效益：NFT 在路上**

![](https://img.foresightnews.pro/202302/cebd02c9fc9dd978a768c4121ebf6057.jpeg?x-oss-process=style/scale70)购买加密艺术作品，不仅能以虚拟形式留存纪念，还能获得真实世界的资产体验。在这个依靠 NFT 全新打造的世界里，从 Gucci 到新雪佛兰、从 Snap 过滤器到 Fortnite 皮肤……每一次真实购买都伴随着 NFT，用户成为所有者并解锁一系列权利。**为了拉近创作者与用户之间的距离，为创意经济提供更好的商业效益，NFT 一直在路上。**我们仍需不断探寻 NFT 为创意经济赋能的新路径：### 

**削弱中介作用**
==========

在区块链的世界里，一旦购买了 NFT，它就由购买者控制、处理。因为所有权转移到创作者和用户手中，用户可以“货比三家”，倒逼市场赚取费用。那么，**降低第三方中介从中收取的费用，可能会对创作者可支配的收入产生乘数效应**，让更多利益流向创作者，给他们更多投身创意经济建设的动力。如何研究更低流通费用的交易机制，成为开发者优化平台配置的新关键。

### 

**支持价格分层**
==========

基于广告的传统商业模式中，收入高低都是统一制定生成的。N**FT 允许创作者提供更高价格的特殊项目来“筛选”积极性最高的用户**，以此设定不同加密艺术作品的价格档次，让粉丝根据实际情况选择性购买，在需求曲线下捕获更大的区域。未来加密艺术平台可以优化程序设定：**由创作者选择相应的初始价格层次，评估市场情况科学调整，让更多购买者“以好价换好物”**。

### 

**让用户成为主人**
===========

加密技术的总市值已增长超过 1 万亿美元，虽然没有各类组织支持，但被数千万人使用和喜爱。到目前为止收入最高的 NFT 项目——NBATopShot，一个月内便创造了 2 亿美元的总销售额，但其对营销的开支预算却很少。事实上，加密艺术市场的营销已经告别了传统大规模、批量式的资金投入，而是**让“用户成为主人”自发参与作品宣传、购买，从而带动作品热度攀升**，吸引更多投资者和广告商加入。因此，**搭建更多用户主动创作和互动的渠道将是未来 NFT 市场的关键。**

NFT 为创作者、用户首次提供了进入资本市场的机会。在创意经济流通的市场中，**每件 NFT 作品不是一个小小的“分母”，而是汇聚成助推艺术交易变革的重要力量。**尽管当下 NFT 运作仍有不成熟之处，但以“用户至上”的观念进行艺术作品创作，参与创意经济流通，相信会给 Web3 时代带来无尽可能！
