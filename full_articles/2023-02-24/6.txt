今天晚间，美国加密交易平台 Coinbase 宣布推出基于 OP Stack 的以太坊 layer 2 网络 Base 测试网。

Base 是一个安全、低成本、对开发者友好的以太坊 L2，旨在为 web3 带来下一个十亿用户。Coinbase 并没有计划为 Base 发行一个新的代币，仍将使用 ETH 作为原生 GAS 代币。为了纪念这一事件，Coinbase 推出了纪念 NFT——「Base, Introduced」，所有地址均可免费铸造，该 NFT 总量限时不限量（[点击跳转至 MInt 网页](https://zora.co/collections/0xd4307e0acd12cf46fd6cf93bc264f5d5d1598792)）。此外，Coinbase 还宣布将推出 Base 生态系统基金，该基金投资并支持基于 Base 且符合其投资标准的早期项目。

Optimism 官方随后发布[公告](https://optimism.mirror.xyz/2jk3D1Y8-hid8YOCUUa6yXmsyzNCYYyFJP0Nhaey9x0)称，将对 Optimism 进行定位升级，OP 将升级为基于 OP Stack 构建的多链 L2 平台。在此次与 Coinbase 的合作中，Base 是基于 OP Stack 构建的一个全新 L2 区块链。Base 将把部分交易费收入返还给 Optimism Collective 国库。近期内，这项合作将致力于将 Optimism 主网、Base 和其他 L2 升级为初始的超级链结构，并共享桥接和排序。

价格方面，Optimism 代币 OP 从晚间的 2.8 美元一度飙升突破 3 美元，过去 24 小时涨幅超过 21% ，现回落至 2.9 美元附近 ；Coinbase 股价则无明显波动，下跌 0.1% ，暂报 61.11 美元。

以下是 Coinbase 博客中关于 Base 的项目介绍，由 Odaily 星球日报编译：

今天，我们很高兴地宣布推出 Base 测试网，这是一个以太坊 L2 网络，可以为所有人提供了一种安全、低成本、对开发者友好的方式，来构建去中心化应用程序或链上「dapps」。Base 的目标是将数以十亿计的用户带入加密加密。为了实现这一目标，Base 将作为 Coinbase 链上产品的大本营和任何人都可以构建的开放生态系统。

我们正在 Coinbase 内部孵化 Base，利用我们过去十年构建加密产品的经验，并计划随着时间的推移逐步更加去中心化。我们没有计划发行一个新的网络代币。开发人员现在可以登陆 base.org 开始在 Base 上构建项目。

Base 是什么？我们将 Base 打造为开发人员在链上开发的最佳场所之一。

* 由以太坊保护：Base 具有支持去中心应用程序所需的安全性和可扩展性。它利用以太坊的底层安全性以及 Coinbase 的最佳实践，使您能够安全从容地从 Coinbase、以太坊 L1 和其他可互操作的链进入 Base。
* 由 Coinbase 授权：Base 通过访问 Coinbase 的产品、用户和工具，可以轻松构建去中心化应用程序。无缝的 Coinbase 产品集成、简单的法定入口和强大的获取工，使开发人员能够为超过 1.1 亿的经过验证的用户提供服务，并访问 Coinbase 生态系统的平台上价值 800 亿美元的资产。
* 功能强大，费用低廉： Base 以极低的成本提供完整的 EVM 等效性，并致力于推动开发人员平台的发展。使用简单的开发人员 API 为您的 Dapps 设置 gas 交易以进行帐户抽象，并使用易于使用的桥接器安全地构建多链应用程序。
* 开源： Base 的目标是去中心化、无需许可，并向任何人开放，以创建一个由 Optimism 驱动的标准、模块化、rollup agnostic 的超级链。我们将作为开源 OP Stack 的核心开发人员加入 Optimism，并致力于创建一个由其他开发人员组成的繁荣社区。
首先，Base 开发人员可以使用我们的 RPC 测试网端点在 Base 上进行构建，也可以从节点提节提供商中选择：QuickNode、Infura 和 Blockdaemon。

我们为什么要建造 Base？为了将数十亿用户引入加密经济，dapp 需要更容易、更便宜、更安全地进行交互。为了实现这一点，我们需要让开发人员更容易构建这些 dapp。在我们建设 Base 的过程中，我们明确遵循以下几个核心原则:

我们正在设计 Base，以提供对以太坊 L1、其他 L2 以及其他 L1 生态系统（如 Solana）的简单、安全的访问。我们鼓励他们从 Base 开始，去到任何地方：我们将 Base 视为用户进入加密经济的「桥梁」。这是一种易于使用的默认链上体验，可以访问其他链上的产品。随着 Base 建立与其他链的互操作性，我们还将继续在 Coinbase 产品中支持尽可能多的区块链。

全球加密经济的基础软件应该完全开源并免费提供。Base 与 Optimism 合作，建立在麻省理工学院授权的 OP Stack 之上。我们作为第二个核心开发团队加入，致力于 OP 堆栈，以确保它是对所有人都可用的公共产品。我们将此工具包视为一个开放平台，任何人都可以贡献、分叉和扩展以帮助扩大加密经济规模。

去中心化对于加密经济保持开放、全球化和人人可及是至关重要的。虽然我们已经开始在 Coinbase 内部孵化 Base，但我们坚定地致力于在未来几年朝着完全去中心化的方向发展。随着 Coinbase 为 OP Stack 带来的资源增加，我们有信心 Base 将在 2023 年从第 0 阶段升级到第 1 阶段 rollup ，并在 2024 年完成第 2 阶段 rollup 。

Base 的愿景是成为一个开放的生态系统，以 Coinbase 产品、用户和资产为种子。我们将与更广泛的社区合作，使 Base 活起来。我们很高兴拥有一个生机勃勃的社区，他们将为使 Base 成为一个强大的生态系统做出贡献。

Base 生态基金。我们还宣布了 Base 生态系统基金，该基金投资并支持基于 Base 的符合我们投资标准的早期项目。如果您有兴趣，请在此处填写表格。

下一步是什么？我们的重点是确保测试网的成功。在未来几周，我们将分享我们的主网路线图，并为在 Base 上构建提供更多开发人员资源。
