去年下半年，即便整个加密市场处在深熊困境当中，但 Arbitrum 生态代表项目 GMX 和 Treasure DAO 依然逆势增长，无论是用户数据还是 Token 市场表现都可圈可点，引人注目。

  


**2023 年以来，随着市场整体回暖，Arbitrum 生态全面开花，高热度只增不减，Arbitrum 生态有哪些值得关注的项目呢？**

  


在此之前，我们需要了解 Arbitrum 在整个 Layer2（二层网络）中的地位。

  


Layer2 是以太坊为解决高昂 gas 费和缓慢的处理速度而提出的扩容方案，将部分数据处理放在 layer2 上进行，减少主网的工作量，主网负责结算，从而提升整体的工作效率。

  


**Layer2****从提出****以来，生态持续稳步发展。**下图显示，以 Eth 本位计算，2019.11-2023.2，Layer2 总锁仓量（TVL）的趋势。

**在 Layer2 中，Arbitrum 迄今最成功的。**Arbitrum 近乎 100% EVM 兼容，使得一层网络上的应用可以无缝迁移到二层上。同时，Arbitrum 是二层中主网上线最早的，占据一定的先发优势。主网上线时，就有以 Uniswap V3 为首的 74 个项目部署在上面。

  


如图所示，迄今为止，A**rbitrum 的 TVL 占二层总量的一半以上，可以说是目前 Layer2 中当之无愧的龙头。**

01 DeFi
-------

### 衍生品

#### **1.Gmx**

  


GMX 于 2021 年 9 月推出，是一个永续合约和现货 DEX，支持用户以 0 滑点和高达 50 倍的杠杆率，没有 KYC 或地域限制。（KYC（Know Your Customer）即充分了解你的客户，对账户持有人的强化审查，了解资金来源合法性，是反洗钱用于预防腐败的制度基础。）

  


使用 Chainlink 提供的动态聚合预言机喂价（来自头部 Cex）来定价。

  


**在经济模型和运行机制上，项目有独到之处：**GMX 体系内有两种 Token，分别为协议的治理 Token GMX 和 LP（流动性提供者）Token GLP。

  


![](https://img.foresightnews.pro/202302/30283b1c20bd80ce92601c849f468d11.png?x-oss-process=style/scale70)

*官网界面*

  


GMX 和 GLP 持有者分别获得协议中现货和杠杆产生费用的 30% 及 70%，Token 有较好的价值捕获。

  


**同时，GMX** **Token****采用双重质押的机制，降低了 GMX 通货膨胀的风险，减少****Token****流通。**而 GLP 流动性提供者作为对手盘，可以获得流动池中亏损者的盈利。因为 GLP 是零和博弈，用户亏损则 GLP 获利，反之亦然。

  


Gmx 是目前 Arbitrum 上锁仓量最高的项目，TVL 占比 28.5%（根据 Defillama 数据），市值 6.8 亿美元。

  


#### **2.Gains Network**

  


**Gains Network 创始团队匿名，其产品 gTrade 是一个针对加密资产、外汇和股票的去中心化合成杠杆协议，**种类丰富，包含 Token、外汇和 Token 化股票等 100 多种，杠杆率在所有 DEX 中最高，加密资产品类最高支持 150 倍杠杆。

  


**使用自定义实时 Chainlink 节点运营商网络（DON）进行报价，为每笔价格获取中位数值，很好的保障了用户权益。**

  


Gains 采⽤了与 GMX 有点相似的模型，操作者同样与流动性提供者进行博弈，但流动性提供者只能使用 DAI 存入金库，产品相当于在 CEX 中常用的 U 本位合约。

  


**其原生 Token GNS 存在销毁机制，**当系统抵押的 DAI 达到 130% 后，会从 GNS/DAI 池中购买和销毁 GNS，达到 Token 通缩的目的。

  


![](https://img.foresightnews.pro/202302/0cf8a58b34f4602a1b62c969d20926aa.png?x-oss-process=style/scale70)

  


**目前 Gains 是 Arbitrum 上交换量仅次于 GMX 的第二大衍生品 DEX，**市值 2.7 亿美元。

  


### 期权 Arbitrum

#### **3.Dopex**

  


**Dopex 是一种去中心化的期权协议，其目标是流动性最大化，同时最小化期权卖方的损失并最大化期权买方的收益。**

  


Dopex 提供了一个简单易用的平台，这意味着任何人都可以参与期权池，用户可以为各自的资金池存入基础 / 报价，通过流动性资金池卖出和购买折扣期权来赚取被动收入。

  


**项目使用双 Token 模式，DPX 是有限供应的治理****Token****，rDPX 是给补偿期权卖方损失的回扣****Token****，可用于铸造合成资产。**

  


Dopex 是 Arbitrum 上锁仓量最高的期权类项目，目前 Token 市值 9000 万美金，目前主要流动性在 sushiswap

  


### Dex

#### **4.Camelot**

  


Camelot 的主要功能是 Uniswap V2+Curve 类型的 DEX，在手续费、激励措施、Token模型等方面做了改进。具备一个 Launchpad，项目方可以借此进行融资并引导流动性。

  


### 借代

#### **5.RNDT Radiant**

  


Radiant 是 Arbitrum 上的全链借代协议，其目标是成为一个全链资本市场，用户可以在任何支持的链上存入资产，并跨多条链借入各种资产。**跨链互操作性将建立在 LayerZero 之上，利用 Stargate 的路由接口，让抵押品的代方能够在各条链上提取资金。**头部借代项目 Aave 在 V3 版本中也会支持跨链借代，但目前还未上线。

  


**Radiant 官方近期表示，预计将于未来几周在 Arbitrum 上正式启动 V2 版本并迁移，未来还将部署在 BNB Chain 上。**目前官方采用流动性挖矿的方式，刺激用户使用借代服务，TVL 一路飙升。如图所示，Radiant 目前是 Arbitrum 上 TVL 最高的借代协议。

  


![](https://img.foresightnews.pro/202302/a9edbcad26529bc81a1b94edd360dabe.png?x-oss-process=style/scale70)

*图源：defillama*

  


### 稳定币

#### **6.Vesta Finance**

  


Vesta Finance 是一个 Arbitrum 生态内的稳定币协议，该协议允许用户存入抵押品以铸造美元稳定币 VST ,VST 与美元挂钩。

  


![](https://img.foresightnews.pro/202302/adcbd7d9d8152e0fd2b0b23e8066892e.png?x-oss-process=style/scale70)

*产品界面*

  


**Vesta 的其他几个特点包括：**

  


* **低抵押率：**要求用户的抵押金库以远低于竞争对手的最低抵押率进行抵押（例如，ETH 为 110%，renBTC 为 110%，gOHM 为 175%）。
* **立即赎回：**VST 持有人可以随时赎回他们的 VST 稳定币作为基础抵押品。赎回机制以及算法调整的费用保证了最低 1 美元的稳定币价值。
  


该项目的 Mcap/TVL 仅有 0.22，是 Arbitrum 生态内的稳定币协议中最低的，一般情况下，该数值越小，代表项目上升潜力更大。

  


02 GameFi
---------

#### **1. Treasure DAO**

  


**Treasure DAO 是一个去中心化的游戏发布平台，通过自下而上驱动的 IP 和基础设施将游戏和社区连接在一起，并通过一组通用的可组合资源统一起来。**

  


**起初其 IP 来自于一个知名 NFT 项目 Loot，由此发展出衍生的 NFT 项目和 NFT 生态，**其于 2021 年 9 月份由社区推出，方式为社区驱动、公平启动。

**MAGIC 是 Treasure 的原生 Token** ，是 Treasure 市场的唯一资产。Treasure 生态的每个社区都围绕 MAGIC Token 构造自己的故事，通过 MAGIC 将叙事和经济生态联系在一起。

  


![](https://img.foresightnews.pro/202302/181b2b3a3f7783fc4eb9564c836cb006.png?x-oss-process=style/scale70)

*（TreasureDAO 生态游戏）*

  


**TreasureDAO 由质押 MAGIC 的持有者管理，负责监督该项目。**DAO 投票决定加入哪些项目、MAGIC Token 分配，以及市场和协议的收益分享。

  


**绝大多数团队成员直接来自社区，之后在数轮战略投资和社区发展下，诞生了十余款游戏和基础设施类型的项目，其中 beacon 游戏在去年 11 月爆火**。项目的特色在于使用 magic 原生Token把整个生态都整合起来，相互联系，构建更加完善的经济结构，使 gameFi 生态更加可持续化。

  


#### **2. Trident**

  


**Trident 是一款 2D MMO、程序生成的无限宇宙游戏，由 Geyser Forge Studios 开发，**它提出了一种新的可持续 GameFi 模型「**Risk To Earn」**来增强玩家的游戏经济。

  


R2E 即玩家和持币人可以投注赚取收益，协议抽成收入反哺 Token。举例：当一个人获胜时，他们会从失败的人那里获得「Risk」的赏金。下注的Token要么在玩家之间交换，一小部分进入国库，要么可以在公开市场上进行互动。**此种方式增加了游戏****Token****的消耗方式，一定程度上延长****Token****通胀的周期，增长了游戏经济的寿命。**

  


![](https://img.foresightnews.pro/202302/4519d30d5740ab0b7cdb479e7785a3bd.png?x-oss-process=style/scale70)

  


**Trident 目前有两款游戏正在开发中：Trident MMO 和 Trident SpriteDuels（精灵决斗）**，以及 Trident 的练习模式 Legacy Trident，并将 Risk To Earn 模式融入其游戏玩法中。

#### **1.Synapse**

  


**Synapse 是一种跨链消息传递协议，该协议包括：广义跨链通信、安全模型、syn 桥。**

  


**Synapse Bridge 使用户能够使用 AMM 在一系列 L1 和 L2 资产之间进行交换。**Synapse 扩展到以太坊及其包含的链，并支持在 Canto、Avalanche 和 Harmony 等 Layer 1 以及 Arbitrum 和 Optimism 等著名的以太坊 L2 之间进行交换。

  


项目原生Token SYN，市值 2 亿美元，近期 2 月 22 日上线各大头部平台。
