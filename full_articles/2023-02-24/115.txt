![](https://img.foresightnews.pro/202302/817cb3814eda606cce3ca327bec83bf4.png?x-oss-process=style/scale70)

晚间，Coinbase 揭晓了答案。这个蓝色圆圈代表的正是 Coinbase 最新推出的以太坊 L2 网络 **Base。**

据相关介绍，Base 基于 OP Stack 构建并与 Optimism 合作，是一个安全、低成本、开发人员友好的以太坊 L2，旨在为 Web3 引入下一个十亿用户。Base 将定位为 Coinbase 的链上产品家园，也是任何人都可以构建的开放生态系统。此外，Base 在一开始就明确表示，不会创建新的代币，仍将使用 ETH 作为原生 Gas 代币。目前，为了纪念该网络的启动，Coinbase 创建了一个**可免费铸造的 NFT。**NFT 可以在公告发布后的四天内铸造，并可在 OpenSea 等 NFT 市场上进行买卖。

随后，**Optimism** 也发布公告表示，由于 L2 目前分散的用户与生态，将对 Optimism 进行定位升级，OP 将升级为基于 OP Stack 构建的多链 L2 平台。在与 Coinbase 针对 Base 合作中，Coinbase 既是 OP Stack 代码库的核心开发人员，同时 Base 也是基于 OP Stack 构建的一个全新 L2 区块链。此外，作为一条 OP 平台的区块链，Base 将把交易费收入的一部分返还给 Optimism Collective 国库。近期内，这项合作将致力于将 Optimism 主网、Base 和其他 L2 升级为初始的超级链结构，并共享桥接和排序。

除此之外，Base 已经公布了近 60 个生态合作伙伴，包括 Chainlink、Blockdaemon、Infura、QuikNode、Blockscout、Etherscan、Dune Analytics、Hop Protocol、Nansen、0x、Ondo、Panoptic、Aave、Gelato Network、Pyth Network、Rainbow Wallet、Ribbon Finance、Balancer、PoolTogether、Euler Labs、The Graph、Wormhole、SushiSwap、Covalent 等。

![](https://img.foresightnews.pro/202302/f8992669812555ffb0c0b09767d67377.png?x-oss-process=style/scale70)图片来源：Coinbase

今年年初至今，以太坊升级都在如火如荼地进行着。具体请下载链得得最新 app：**“得得专题 | 以太坊新纪元如何一路走来？”**随着以太坊质押提款日期的临近，Coinbase 选择在此时重磅推出 Base，也算得上是蹭了一波热度。作为大型中心化交易所之一的 Coinbase，以及作为 layer 2 强有力代表的公链 Optimism，这两个的合作无疑算得上是“强强联合”，也难怪 Base 一经发布就有了这么多的合作伙伴。目前，**Uniswap** 也在着手准备搭上这艘大船，相信这并不会是最后一个。

昨日，除了发布重量级的 L2 Base 外，Coinbase 还发布了 2022 年第四季度的财报。财报显示，Coinbase 第四季度净亏损为 5.57 亿美元，全年净亏损为 26 亿美元，具体请看**【链得得独家】深度解读加密第一股 Coinbase 财报：最终赤字 5.57 亿美元，持币总价 4.26 亿美元。**

被誉为“加密第一股”的 Coinbase 似乎永远都在紧跟行业热点，去年 NFT 热潮兴起时，Coinbase 重磅推出了 NFT 平台，但财报显示，Coinbase NFT 市场份额并不理想，Dune Analytics 数据显示，过去一周 Coinbase NFT 市场只有 41 笔交易。尽管该公司表示，将会继续把 Coinbase NFT 视为一个有价值的项目并探索中长期机会，但显然，如今 Coinbase 将重心转移到了新推出的 Base 上。而同样作为大型中心化交易所推出的公链，BNB Chain 似乎遇到了有史以来的劲敌。

此外，作为美股上市公司，监管层面对于 Coinbase 又一大胆突破的态度还未可知。

**—1****—**

### Base 横空出世

Coinbase 表示，曾两次考虑推出币安智能链式的 L1，但最终还是决定放弃，因为他们担心这样做会让链与蓬勃发展的加密经济隔离开来。

最终，OP Stack（Optimism 的模块化、开源蓝图，用于创建高度可伸缩、高度可互操作的各种区块链）成为了 Coinbase 游戏规则的改变者，并为他们提供了构建自己的区块链的机会，也就是 Base。

目前，Base 推出的仅仅是测试网，其将在大约两个月后发布到主网，并将作为一个去权限的 L2 运行。Base 2023 的去中心化路线图如下：

*为 OP Stack 启动至少一个故障证明器，利用一组验证器来识别和解决故障，继而保护 Base 和 Optimism 主网；*

*实现 Base 和 Optimism 主网智能合约的去中心化升级，计划推出一个安全委员会，其中去中心化法定人数不受任何单一实体控制；*

*启动 Superchain 初始版本，可以对 Base、Optimism 主网和任何 OP Stack Rollup 的交易进行排序，确保任何人参与 Base 生态系统的权利；*

*将在 2024 年把 Base 和 Optimism 主网推进到 Stage 2 rollups，使 Base 的去中心化程度和安全性能与以太坊媲美。*

在创世阶段，Coinbase 将是该链的唯一定序者，Base 被归类为完整的训练轮 “Stage 0” rollup。这与 Vitalik 的 **rollup 分类法**基本一致。

此外，Coinbase 对加密价值的总体忠诚体现在他们对去中心化和安全的承诺上： 

*Base 将过渡到完全无权限的验证器集；*

*Base 将在 2023 年 EOY 之前作为有限的训练轮“Stage 1”rollup 运行；*

*Base 将在 2024 年底之前升级到无训练轮的“Stage 2”。*

有限训练轮阶段的广泛实现和 EIP-4844 的推出将使以太坊达到 Vitalik 以太坊路线图上的“基本 rollup 扩展里程碑”，完成 Surge🌊 中两个步骤中的第一步。

![](https://img.foresightnews.pro/202302/173a963136a91910a8c104148d630b36.png?x-oss-process=style/scale70)来源：Vitalik

向去中心化的快速发展、实现链安全性的快速飞跃，以及 Base 在两个月内从发布测试网到进入主网的能力，都是借由 OP Stack 实现的。Coinbase 能够为其 rollup 提供经过实战考验的 Optimism 代码，这是 Base 开发速度的巨大推动力。

那么，这算得上是占 Optimism 团队的便宜吗？不完全是。

事实上，Coinbase 正在与 Optimism 合作，成为 OP Stack 的第二个核心贡献者。两个 L2 团队构建在同一个堆栈上，开发也会提速。Base 的开发受益于一个开源和免费可用的扩展解决方案代码库，rollup 本身是建立在以太坊之上的。

同时，Base 的部分交易费用（尚在商议中）将会捐给 Optimism Collective，为 Optimism 的公共产品提供资金。Coinbase 还承诺推出一个 **Base 原生生态系统基金**，投资并支持在 Base 建设符合其投资标准的早期项目，该基金现已对项目方开放申请。

除了提高启动速度和增强团队间的合作之外，OP Stack 的未来为共享序列器集的链提供了原子组合性，这要归功于 OP Stack 链之间的标准化消息传递格式。

**—2****—**

### Coinbase 宏大愿景

Coinbase 不仅想做一个中心化交易所。其产品涵盖了数字资产质押解决方案、流动性质押衍生品、机构托管解决方案、非托管钱包产品、应用内集成 DeFi 可访问性等。Base 是这条产品线的自然延伸，可以为 Coinbase 用户提供直接融入 Web3 世界的机会。

![](https://img.foresightnews.pro/202302/c1e23f19b299a55ae90ed6f681ce6344.png?x-oss-process=style/scale70)来源：Coinbase

尽管 Coinbase 已经建立起了一个声誉良好的品牌，并成为了加密货币交易和投机的可信用户界面，但 DeFi 和其他链上原语的日益普及使得人们对简单、可靠接口的需求日益强烈。因此，Coinbase 的目标是成为他们的服务提供商和可信前端。

Coinbase 将自己定位为链上活动的主要门户，Base 和在其之上构建的协议，包括 Coinbase 开发的 dApps 等，不仅对公司的盈亏有积极影响，而且可能会为更广泛的加密经济提供积极的推动力。

对于 Coinbase 的用户来说，一个简单、相对无风险的 Web3 交互解决方案是可取的。Coinbase 旨在提供一种默认的、开箱即用的体验，消除用户对与区块链交互的恐惧，这也是进入 Web3 世界的一个主要磨合点。

此外，Coinbase 的目标是带领其 1.1 亿经过验证的用户，以促进低成本的 dApp 用户获取。

![](https://img.foresightnews.pro/202302/6b170ab069c93f0fb53cf01bf6d55ca5.png?x-oss-process=style/scale70)

在 Coinbase 看来，当前各种的空投计划很容易受到游戏的影响，通常会带来短暂的链上活动和 TVL 的增加。而 Base 将作为基础设施层，用于寻求获得新用户的协议，同时为用户提供选项，并将他们与所需的服务连接起来。

正如信用卡提供商和银行使用财务激励来吸引新用户并改变其消费行为一样，Coinbase 希望通过类似的模式彻底改变 Web3 的消费者获取方式。

**—3****—**

### BNB Chain 强大劲敌？

目前看来不是。

鉴于没有永续合约和其他高杠杆交易解决方案，Coinbase 的用户基础远没有那么退化。由于 Coinbase 的客户是 Base 最有可能的用户，我们不太可能看到类似的杠杆热潮。借贷杠杆有助于产生收益，这也吸引了 TVL。

一个主要的 DeFi 用例可能会在 Base 上获得较少的吸引力，而且由此导致的对杠杆需求的缺乏也可能会转化为更低的收益率生成、更低的 TVL 和更少的交易量，从而削减网络活动和费用的产生。

此外，Coinbase 的许多用户已经在成熟的 TradFi 市场获得了替代金融解决方案。

BNB 已经找到了一个利基市场，为发展中市场上缺乏稳定货币或复杂金融系统的用户提供了低费用的选择。尽管加密资产价格存在波动，但它们似乎正在成为受高通胀影响的当地货币的有利替代品。区块链上的点对点交易如今很常见，而无需许可的借贷市场也逐步取代了当地传统的有息储蓄账户。

2023 年 1 月，币安的月度现货、比特币期货和 ETH 期货交易量合计为 1.108 万亿美元，而 Coinbase 的月度现货交易量占前者总量的 5.0%，约为 553.2 亿美元。

![](https://img.foresightnews.pro/202302/43799286ef51f63daf9e9c2b2a0f29e3.png?x-oss-process=style/scale70)

不幸的是，西方金融体系还停留在过去的日子里。在 Venmo 和 CashApp 上进行的点对点交易通常需要三天时间结算。我们已经接受了部分准备金银行体系，生活在一个依靠塑料卡和 ApplePay 运行的社会中。

对于 Coinbase 渗透最深的市场消费者来说，向 Web3 过渡更加困难，因为他们可以获得传统的金融解决方案，并且比起转向未知的 Web3 世界，他们更愿意维持旧的习惯。

**—4****—**

### OP 借势起飞？

在 Coinbase 发布声明的同时，另一位毫无争议的赢家是 OP。Coinbase 选择基于 OP Stack 来构建 Base，这对 Optimism 生态系统来说是一次巨大胜利。

这份公告基本上等同于告诉人们，Optimism 核心团队将被 Coinbase 大规模地收购。Optimism 不仅免费获得了开发人员的支持，而且 Base 也将交易收入的一部分给了它的姐妹链。

Coinbase 的这一举动真正验证了 OP Stack 的飞轮理论：一个大型核心贡献者的加入为 rollup 构建或迁移到 OP Stack 提供了额外的激励。

Optimism、Metis 和 Boba 都是建立在 OP Stack 上的，这不是什么秘密；增加 Coinbase 作为第一个核心贡献者将使合并后的团队能够摧毁拦截器，开发出最好的 Optimism rollup 解决方案。随着越来越多的 rollup 构建在 OP Stack 上，由模块化和开源代码所提供的设计灵活性将会逐步增加。

随着时间的推移，OP Stack 可以为 DEX、GameFi、NFT 交易或任何 Optimists 渴望的东西打开一个高度专业化的 rollup 解决方案的世界。

OP Stack 的推出降低了用户构建自己的 rollup 的门槛，就像流动性带动流动性产生一样，核心贡献者也会拉动核心贡献者的加入。

**—5****—**

### 法律监管

需要指出的是，尽管 Coinbase 推出原生 L2 听起来很乐观，但要让该平台的用户和资产进入 Base 和 Web3 世界仍有很长的路要走。

另外，虽然 Coinbase 声称，与空投相比，Base 将降低用户获取费用，同时将用户与他们需要的协议连接起来，并在此过程中提供指导，但这些说法似乎毫无根据。

Coinbase 的 Learning Rewards 标签已经提供了一个与 1.1 亿用户连接平台的协议。此外，“Web3”标签也已经为用户提供了推荐的 dApp，甚至以社区星级的形式提供了社交信号。

可能 Base 获得成功最大障碍是：**作为一家美股上市公司，Coinbase 所处的监管环境充满挑战。**

与 Optimism（使用 OP 来资助公共产品，并通过重复投放来激励行为）或其竞争对手 Arbitrum（尚未推出代币，计划进行空投，旨在吸引流动性和可持续采用）不同，Coinbase 在短时间内推出代币的可能性非常小。

让法律批准一个无许可的 rollup 可能是一项繁重的任务，美国证券交易委员会（SEC）、纽约证券交易所（NYDFS）、美国中央银行（OCC）或其他对美国金融系统有管辖权的机构仍有可能就 Base 问题敲响 Coinbase 的大门。

因此，发行代币几乎是不可能的，除非美国能够为空投及相关美股发行制定出一个明确的监管框架。而在此之前，在没有潜在代币的情况下，Base 相对于其他竞争对手或许处于了明显的下风。

**—6****—**

### 前景不明

Coinbase 确实有一个有利因素：该公司仍然拥有稳固的品牌资产。

虽然消费者可能不太愿意使用 Coinbase Web3 门户网站上列出的 Aave 和 Uniswap，但随着 Coinbase 品牌 dApp 的推出，这种情况或许会改变。

Base 很可能是通往许可的、KYC DeFi 入口，但肯定不是一个只有 KYC 的区域。

正如 Coinbase web3 主管 Jesse Pollak 在采访中提到的，Coinbase 对扩展解决方案的尝试包括一套带有去中心化身份原语的基础设施产品。

由于监管机构不清楚如何接近 DeFi 且监管透明度有限，但爱打官司的监管机构又紧随其后，因此，带有 Coinbase 品牌的、半许可的 dapp 可能会成为 DeFi 世界的“浅水区”。Coinbase 已经有了这些关系，为机构和高净值个人提供托管和大宗经纪服务。

Base 是否有可能成为 Coinbase 品牌 dApp 的启动平台，从监管风险和资金安全的角度为这些实体提供安慰？如果是的话，这可能是一个巨大的解锁，向一群不情愿的投资者打开了 DeFi 的世界，他们以前面临着进入这个领域的结构性障碍。

毫无疑问，Base 的推出对整个 Coinbase 团队来说是一项巨大的成就，代表着 OP Stack 的一次难以想象的飞跃。然而现实情况是，消费者仍然对上链犹豫不决，更愿意生活在 TradFi 的围墙花园中。虽然 Coinbase 拥有 1.1 亿的认证用户，并在平台上持有 800 亿美元的资产，但其中只有一小部分可能会迁移到 Base 上。

但对于 Coinbase 来说，800 亿美元是一个非常庞大的数字，2.5% 的迁移量也足以取代 Arbitrum，确保 Base 成为 TVL 第四大的区块链。

![](https://img.foresightnews.pro/202302/2a70b9b9f23eabc1ee097fd35b4b1353.png?x-oss-process=style/scale70)来源：DeFiLlama

Coinbase 决定推出并创建自己的 L2，这是对以太坊和更广泛的加密金融系统的巨大信心下注，可能会在未来某个未知的时间点获得丰厚的回报。但值得注意的是，Optimism 和 Arbitrum 在过去一年的收入仅略高于 900 万美元，而 Coinbase 的营收超过 30 亿美元。

![](https://img.foresightnews.pro/202302/0f36edc2bb30817d594c92ebad4f1770.png?x-oss-process=style/scale70)来源：TokenTerminal

在可预见的未来，Base 除了作为一个钱坑（money pit）之外，几乎无法再充当其他任何角色。

虽然 Coinbase 已经提供了种种迹象，表明其打算成为以太坊生态系统的忠实管家，通过贡献 OP Stack 和成为 **RocketPool oDAO** 成员等来促进去中心化，但能否做到才是最重要的。

因此，请记住：DYOR。
