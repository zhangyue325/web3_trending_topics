去中心化的 XXX，通常就叫做 Dxxx，DID，其实就是去中心化 ID，就是去中心化身份认证。

在过去很长一段时间内，尤其是在 DYDX 项目带动了巨大空投潮以后，DID 赛道引起了不少业内、资本的关注。同时，很多做 DID 的项目都也发了自己的代币。

但是关于这个赛道的投资哲学意义，很多人都没办法理解。甚至于说，关于 DID 赛道真正的机会，也并非是那些一个个的「去中心化身份验证器」的代币。

当然，在继续分享关于 DID 赛道的投资研判分享之前，我需要先聊一聊，为什么我要在行情不好的时候，来讲这个。

在去年大半年的市场里，不管是散户，还是机构、项目方，都是处于一个非常 FOMO 的情绪当中。似乎投什么都能赚，买什么都能涨。

但牛市是典型的情绪主导市场。一旦出现大范围回调，那些因为情绪起来的项目很难再有反弹的机会。

我们需要注意的是，每一次引导市场爆发的关键项目，其实都是在这个时间段里做起来、只能在这个时候才能做起来。

因为这个时候大家才会从狂热的情绪中冷静下来。机构开始谨慎投资，思考下一步的趋势是如何；同时能熬过这个阶段的项目，才能在下个周期中复苏。

所以在这个时间段，除了做好 BTC 和 ETH 的囤币计划，更多的是需要沉下心来，去看看接下来周期的启动到底会在哪个赛道，同时在普遍低估时候能够去布局一些机会。

基于此，所以我在 8 月就已经开始了对于关键赛道的研判分享课。今天是关于 DID 赛道的研判。

在现在币圈这个市场，投机和投资并存。不管是投资，还是投机，本质上都是对「利润最大化的追求」。

只不过，前者是基于对交换价值而追求利润，即短期内影响价格的指标，比如舆论，做市值管理等等；后者是因为资本共识觉得某个项目能打开更大的市场，获得更多的市场份额，通过投资这样的「预期」去获得投资回报。

DID 赛道，就是一个可以打开市场的关键赛道。

过去 DeFi 之夏，是 DeFi 第一次进入到资本的视线当中。同时，Comp 的流动性挖矿，让很多普通人也能够获得 AMM 的收益。所以 Defi 赛道进入了第一次估值泡沫周期。

严格意义上来说，DeFi 的大部分项目到最近的暴跌之前，都只是是一次估值回归的过程。因为针对他们的锁仓量，用户量，项目方获取的利润等各方面数据来看，DeFi 的项目普遍都是都是处于比较高估的状态。

那么就 DeFi 赛道而言，就从概念热点，转入了落地性应用，真正意义上靠利润去获取估值的阶段。

对于 DeFi 赛道，对于诸如 SNX、AAVE、Comp 或者 MakerDAO 等顶级 DeFi 项目而言而言，DID 是否能够完善，何时完善，都会对这些项目大规模爆发产生非常重大的影响。

为什么会有这样的判断？我们以「AAVE」为例。

AAVE 是做什么的？本质上就是一个借贷产品，是去中心化的 P2P 借贷项目。

他的问题我在过去的文章里也讲过。去 AAVE 借款，或者去其他 DeFi 项目借款，都是超额抵押。意思就是你的抵押品必须超过你的借贷资金。

恰恰就是这种「超额抵押」，破坏了 P2P 借贷的逻辑。

借贷的本质需求就是缺钱。所以超额抵押让 AAVE 的用户群体、或者让整个 DeFi 赛道的用户群体就变得非常小。

只有那些因为不想损失头寸的交易者、不想卖币又需要支付挖矿成本的矿工以及质押挖矿的参与者才会参与其中。

到现在为止，除了 AAVE 的闪电贷款和极少数项目开放对传统大型机构的信用贷之外。并没有类似现在信用卡或者信用贷款的方式。

那 DeFi 项目为什么不能超额抵押呢？原因就是链上的 DID（去中心化认证）配合的信用体系和司法体系没有完善。

因此，对于 DeFi 而言，要想进一步扩大市场规模，就必须要求跟现实社会类似的信用体系，包括对你的资质审核，还款能力审核，负债审核，乃至于配套的仲裁体系，如何清算，如何执法等等等等。

这个前提就是 DID 制度的完善。

注意，这里的 DID 并非是简单的「身份认证」。现在很多资深从业者或者投资机构都是简简单单的把 DID 看成一个链上身份认证器，但事实上他包涵的意义更为重要。

讲 DID 之前，还要讲一个东西，叫做 DeSoc（去中心化社会）。

之前 vitalik 写过一篇论文，叫做《寻找 WEB3.0 的灵魂》。那篇文章大家可以看看，这里就不讨论，去中心化社会那篇论文，其实就是反映了目前整个 Crypto 最顶层的阶级对未来 Crypto 的规划。

即：「把这块地方打造成一个跟现在社会一样的、模拟真实的一个社会体系。只不是在这个社会体系中，没有国家的独裁，大家都通过 DID 来掌握自己的数据和所有权。」

可以说，当 Crypto 发展至此，才真正具有堪比互联网时代的想象力。现在，我们讨论的，就是现在有哪些关于 DID 赛道的项目，以及哪些项目会有机会。

这种想象力会让资本陷入疯狂，资本都会高喊着建设新世界，并拿着钱疯狂的冲进来。就跟孙宏斌投资贾跃亭一样。

但现在，整个市场上关于 DID 的认知是有极大问题的。在我看来 DID 只有两种，一种是基于最底层的原始 DID，另一种是针对各类项目所使用的注册类的衍生 DID。

为了方便理解，我就用现实的「身份认证」做个比较：

 

最底层我称之为「元认证」。这有点类似国籍归属，第二种则是「次认证」，即一个个项目注册的 DID。

 

先总结 DID 和灵魂代币的意义：让用户对自己数据确权，同时避免对多个账号密码记录的繁琐而能链接这个世界的归属区域。

 

我们从常识去理解，我们生活在国内，享受国家的福利和社会医疗，接受国家的庇护，但国家需要维持秩序，就会有规范行为并对破坏规则的行为做出惩罚。

 

而身份证就是你作为国家公民的证明。所以这种灵魂代币的本质就是，需要一个「国家形态」的组织，为他们所颁发的 DID 进行福利待遇上的赋能。

 

如果我们把以太坊，Solana 当成一个国家，他们来发行归于他们公链的「灵魂代币」，并基于自己的公链实力来为灵魂代币赋能，只有接受某个特性归属以后，才能得到大部分链上组织的认可，去参与他们的社会活动和福利。

 

那么，我们就从现在这几类 DID 中，尝试寻找一下，哪些项目能够成为这样的「灵魂代币 DID」。

DID 目前的几个方向分别是：

其中最先被淘汰就是第三方登录器。比如炒得很火的登录器 Unipass、DID 验证器 Galaxy、BringhtID。未来大概率是要被淘汰的。

为什么?因为这些登录器、验证器不像钱包能提供资产存储价值，也不能像社交账号提供社交通讯价值，他们的存在仅仅是为了做去中心化验证。

当然，大多数项目在深挖护城的时候，确实会把这些认证功能交给第三方去做。

但对于所有项目方而言，这道口子最终都要留给自己来做，并且当项目发展成巨头以后，他们也有能力做这种并没有多少技术门槛的工作。那时候，这些第三方验证器还有意义吗？没有的。

钱包其实也算是 DID 赛道，因为制作钱包的时候就已经经历过一些初级的认证，才获得了一个地址，去跟 DAPP 交互也是跟地址交互。

但钱包的 DID 认证没办法成为「元认证」，因为钱包很难做成一个整体的链上社会。他只能做到类似微信、QQ 的功能，集成其他协议的登陆授权。

域名大概是误解最多的一个赛道。他对于普通用户的价值并没有大家想象的那么大。去中心化域名，就是通过 NFT 实现的某种意义上的 DID。

比如 web3.0 个人网站的域名。你如果要建立一个门户网站，你需要一个域名，但大部分人用的上么？

就拿现在非常火热的 ENS 来看。我做了一个星空社区.ETH 的域名，大概是 2000R，3 年，但你真正用到 ENS 还要去搭建个人网站。并且持续维护，还要去持续付费。

现在的 ENS 功能就是在推特上当后缀，然后大家去申请 ENS 就是等着炒作。1~2 年以后，当第一部分 ENS 到期的时候，ENS 的用户数量会出现一个断崖式的下降。

所以他也不能去做成 DID（成为链上身份入口）这种大事，没有普适性。

我们会发现，目前所有的 DID 产品都没办法达成我们所限定的「元认证」概念。但这就意味着 DID 没有投资的意义了吗？并不是。

DID 项目最大的价值，作为投资人，投出去某个项目。我们最终期待的就是他的「价值变现」。而 DID 最大的价值就是数据。

但掌握数据之后的变现路径，同样的是需要有上下游。

比如：你要有一个能够集成数据的 APP，这个可以是游戏，可以是社交，可以是功能性的 O2O 平台。

然后 O2O 平台通过大量数据的连贯性采集，可以做到精准的人物画像，之后再进行针对性的需求去做变现。或者广告投放，或者价格调整。

微信的模式就是这样，并且他已经取得了成功。

那在 Crypto 呢？Crypto 有个钱包项目叫做 metamask，就是我们俗称的「小狐狸」。

小狐狸钱包现在的累计收入就几亿美元，按十倍市盈率也就是几十亿美元的估值。单纯的利润来源只是 swaps 上的收入，这根本没办法撑开估值想象力。

但小狐狸钱包确实又有很高的上限 ，为什么？因为它有扩展性。它可以成为更多 DAPP 的登陆入口。去扩展更多的手续费收入。目前来看，Metamask 是有可能成功的，风险点在于，Metamask 能不能在未来的移动端上，持续保持现在对于 PC 端的垄断力。

另外一个值得关注的就是社交类 DID。社交类 DID 完全可以参考微信，微信已经成为了国内大部分 APP 的「ID 登陆路口」了，百分之 90 的 APP 都提供了「微信登陆」的入口。

那么在链上世界，会不会有一个去中心化社交 DAPP 来做微信的事情呢？大概率会。不过前提是出现某种对过去完全颠覆的杀手级 DAPP。

可惜的是，目前我们所看到的去中心化社交项目，大部分都没有任何价值。都没有对社交的形态有任何突破性的进展。具体的分析，可以参看我过去的文章《[我看 SocialFi：社交逻辑的颠覆与下一个超级风口！](http://mp.weixin.qq.com/s?__biz=MzkxNDE0MjYyNw==&mid=2247488882&idx=1&sn=bcbd346f3f98f89124bd5df84a8867c5&chksm=c173b1bef60438a8faea703a21de911fa3a7b2c0de4595d1cbd3c37ab7a1dcce661ea1bee104&scene=21#wechat_redirect)》

可以预见的几种新方式，即先金融，后社交，亦或者先元宇宙游戏，后社交。

从这个角度来看，AAVE 做的 Lens protocol，推特的嫁接插件型项目 Mask Network（MASK）都需要重点关注。

值得一提的是，DID 有意义的前提，就是基于「元宇宙」、去中心化社会。基于此，才有信用体系，司法体系。你通过「灵魂代币」身份认证进入这个世界，你在这个世界的资产都是 token。

如果你不遵守信用，那就可以由智能合约的方式切断你进入这个世界，或者将你驱逐出这个世界。也恰恰是因为如此。整个加密行业都呈现出一个想象空间极大，落地实践时间极长的状态。

故而，在这个漫长的发展过程中，一旦碰上「普惠性」的赚钱效应热点，就会带出一次次的泡沫。

在这些泡沫诞生，破裂的过程中，行业丢掉一些失败品，留下一些待验证，能验证的赛道、项目继续发展。

但这个世界的核心就是 BTC 和 ETH。其他的项目都不行，因为能确定的只是行业，并非项目。所以针对非 BTC 和 ETH 其他的代币，要么你凭借自己的认知提前去布局，获得了这个收益。

不然就不要去追高。
