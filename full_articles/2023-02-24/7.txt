Coinbase 官宣孵化 L2 网络 Base 一度让市场沸腾，根据官方描述，Base 是基于 OP Stack 技术堆栈构建的一个以太坊二层网络，本文归纳总结了与之相关的几大要点，方便读者了解。

1、不发币，部分手续费流入 OP Collective

目前官方已经明确表示，Base 不发币，将使用 ETH 作为网络手续费 gas；另外，未来部分 Base 交易手续费（可能是 20%）将流入 OP Collective，进一步推动 OP Stack 发展。

2、Base 跟 Coinbase 的关系，生态项目，生态基金

Base 是 Coinbase 的原生区块链网络，将运行 Coinbase 的所有链上产品，由 Coinbase 构建，为 Coinbase 用户服务。Coinbase 会鼓励平台用户将资产转移到 Base 链上，当然，这也是一个开放生态，任何人都可以基于之上进行构建，而这对开发者是极具吸引力的，毕竟拥有 Coinbase 1 亿 + 的潜在用户以及上千亿美元级别的资产规模。目前 Coinbase 也公布了 Base 首批合作伙伴，不乏我们已熟知的项目，如 Chainlink、QuikNode、Axelar、LayerZero、Gelato、Ribbon Finance 等近 50 个项目。另外，官方目前已经宣布成立 Base 生态基金，将投资支持满足要求的基于 Base 构建的早期项目。

3、为什么选择 OP Stack ?

自 2022 年下半年其，Coinbase 与 OP Labs 便共同着手研究 EIP4844，即 proto-dank sharding，能够有效 L2 网络的手续费降低 10~100 倍，双方都有共同的理念以及扩展以太坊的愿景，后来 Coinbase\Base 也加入了 OP Stack 的研究队伍中，并表示未来部分 Base 交易费用（可能是 20%）将流入 OP Collective，继续推动 OP Stack 发展。

4、Base 对去中心化的态度？

目前 Coinbase 会是 Base L2 网络上的唯一排序器，不过最终还是要走向免许可的排序器集合的。去中心化属性一直是 OP Stack 所追求的，当前唯一排序器只是暂时的权衡，Base 和 OP Labs 会以去中心化协作的方式推进去中心化进程。

5、账户抽象？智能合约钱包？

在 Base 的官方文档描述中，提及其将拥有账户抽象以及智能合约钱包的功能，Gnosis Safe 官方也发推表示将与 Base 合作来实现账户抽象功能，以实现用户的大规模采纳。

在了解了以上重点后，其实我们最需要理清楚的是，多这样一个由 Coinbase 孵化的 L2 网络 Base，其特殊意义在哪：

● 结合 Coinbase Wallet 以及账户抽象智能钱包等技术，Coinbase 当前用户群体有望引流至 Base，大大降低用户进入理解门槛，能够推进 Web3 的大规模采纳；要知道，Coinbase 当前拥有 1 亿 + 的认证用户基数以及近千亿美元级别的资产规模，Base 也表示将自身定位为用户进入加密经济的“桥梁”。

● Coinbase 的用户群体已是实名认证，引流过去 Base 是会与使命账户有所关联，类似币安此前的灵魂绑定代币 BAB，是否可期待将衍生出更多链上资产的玩法，如 RAW，链上信用等。

● 作为 OP Stack 技术堆栈的先行者，有望推动 Rollup 应用链的叙事，侵蚀 Cosmos 应用链叙事。
