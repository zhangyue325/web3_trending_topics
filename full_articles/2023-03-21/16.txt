ERC-721 于 2018 年 1 月创建，当时链上已经出现了一些 NFT 实验。但没人能够预测到它将导致 NFT 生态的诞生并释放出 NFT 应用的潜力。PFP 和 GameFi NFT 在 2021 年和 2022 年一度成为 Web3 主舞台，我们也更为兴奋地期待 NFT 增长的下一个阶段。

越来越多的创新和标准化措施正进一步带动 NFT。在本文中，我们将分析最新的 EIPs（以太坊改进提案），并提炼出 NFT 的 9 个未来方向。

1. 权限启用的 NFT
------------

此前我们一直遵循 NFT 的一个模型：每个 NFT 在元数据中都有一个所有者字段，代表它的一个所有者。然而，单一所有者的配置会导致在使用 NFT 时缺乏灵活性。如果所有者想要出租 NFT 的实用性怎么办？如果发行者想要限制总 NFT 的数量但仍然让其观众广泛访问其服务怎么办？这引导我们转向了权限启用的 NFTs。

通过启用权限，我们指的是在 NFT 上通用化权限的方法。一个 NFT 可以有一个所有者，但也可以有专门针对潜在权利持有者、忠诚度赠予者等的字段。我们可以将 NFT 的实用性分解成特定的部分，让不同的玩家从中获得最大的好处。

简而言之，我们期待权限启用的 NFT 在租赁市场上的应用，期待 NFTFi 的变种，以及通过 NFT 分发粉丝的产品。

2. 关系启用的 NFT
------------

除了在 NFT 的多个所有者和用户方面进行创新之外，我们还迫切需要实现多个 NFT 捆绑在一起属于同一所有者的功能。在游戏中，一组资产的价值或产出可能远高于单个物品。在元宇宙中，人们可能希望在多元宇宙之间进行传送，并仍然能够以兼容的格式访问其资产。如果 NFT 可以在 ERC-721 合约、应用程序和链之间进行组合，那么将会有一个巨大的市场通过 NFTFi 释放价值平衡。

3. NFT 作为分期归属工具
---------------

代币一直是投资者和早期支持者获得项目股份和回报的媒介。然而，出现了更多的短期炒作行为，而非支持生态系统的健康增长。对于项目来说，使用加密本地方式来制定公平的股份分配方式很困难，或者一般人很难在 ERC-721 中轻松创建逐步所有权声明。归属 NFT 可以成为解决方案，它有巨大的潜力扩展到各种应用程序，将传统融资转移到区块链上。

我们期待着更多的金融结构能够被 NFT 所包容并丰富整个区块链市场。

4. NFT 作为实用工具
-------------

除了作为可行的授予工具之外，NFT 还被用作支持游戏应用和俱乐部会员资格的实用工具。NFT 实用价值的潜力是什么？我们期待无限量的实用价值，从 Web2 中继承用户的行为和特征。新的 EIP 被提出来支持 NFT 的实用价值。

5. NFT 作为身份 token
-----------------

灵魂绑定代币和命名服务一直是链上身份表示的流行方式。它们都依赖于 NFT 来承载身份，并代表身份的独特特征和财产。EIP 一直在增加对 SBT 及其生态系统的支持，同时我们期望 zkID 技术堆栈，即链上隐私保护身份的技术堆栈，将很快准备就绪，并在 EIP 标准化管道中。

SBT 最初在 EIP-4671  中提出，旨在创建非交易型代币作为持有和成就的证明。它被开发为由一个机构发放的证书，不能被撤销或转让。然而，EIP-4671 未能成为以太坊的标准，被搁置了。它并没有消弭 SBT 的热情，反而在 SBT 领域启动了更直接的开发和 EIP。

6. NFT 用于更丰富的内容
---------------

目前，NFT 需要依赖外部存储来保存 PFP 或其他媒体，这导致在各种元宇宙和市场中展示和使用 NFT 时可能会出现互操作性问题，因为链接的文件可能会以不同的格式存在或不可用。我们认为，未来 NFT 本身将能够容纳更丰富的内容，以便在任何中介平台和多元宇宙中使用。

更丰富的内容可以完全提升 NFT 的体验。它为 NFT 的用户体验创造了更大的表面积，并且可以带来更多的创新和用例。

7. 实物资产的 NFT
------------

实现区块链广泛应用的重要一步是连接现实世界和区块链世界。数字 - 物理（digi-physical）是一个常用术语，用于描述数字和物理体验与资产的组合。NFT 是数字 - 物理产品和服务关系的一个有前途的载体。

8. 具有高级度量标准的 NFTFi
------------------

随着 NFT 中权限、关系、锁定期、实用价值等方面的不断增加，NFT 的估值和交易需要更加精细的机制来体现市场对 NFT 各部分价值的不同评估。EIP 的不断推进和丰富显示了 NFTFi 的演进之路，未来将出现更多层次的细节和更精细的用户体验，同时也面临更艰巨的任务来确保平稳流畅的用户体验。

我们看到，未来的 NFT 需要和促进更好的市场、交易选项和收益农场算法的出现。

9. NFT 作为 EOA（Externally Owned Account）
---------------------------------------

最后，NFT 可以成为自己的实体。它可以作为持有人的钱包、身份或者通常的 EOA。这可以限制一个人在链上的追踪，建立 NFT 的历史和声誉，并使更多的 NFT 所有权结构成为可能。

我们可以想象，随着 NFT 成为一个主权链上实体，任何人都可以自由地建立 NFT 的知识产权。此外，使用关系或权限启用的 EIP 方案，用户可以创建层级签名或以全新的方式与 NFT 捆绑包交互。
