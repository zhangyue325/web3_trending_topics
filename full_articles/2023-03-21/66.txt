![](https://cdn-images-1.medium.com/max/912/0*_Ct0S90n1lWFSnUk)

随着加密市场的回暖，不少曾经的小众赛道重新引起了人们的关注，尤以 SocialFi 赛道在过去一直无龙头跑出，其价值和潜力或许被低估。BitMEX 创始人 Arthur Hayes 近期也在博文中预测道：“如果比特币和以太坊继续上涨，在接下来的几个月里肯定会有主流币之外的一个垂直市场爆发。” Bulletin Board System (BBS) Network 是一个去中心化论坛，由不同的留言板相互连接组成的公共网络，也是笔者近期关注的一个项目。

融资背靠币安，BBS Network 于 2021 年获得 Binance Labs 领投的 150 万美元种子轮融资，灰度母公司 DCG、Polychain Capital 、Bancor 等参投，DAOMaker 负责孵化。

此外，不同于传统 Web 2 团队破圈，BBS Network 团队在加密 DeFi 领域已经有着成功经验，创始人 Eyal Hertzog 之前创建的去中心化交易所 Bancor 就是 DEX 里跑出的黑马，最高到 20 亿美金，Bancor 协议也是最早的主流去中心化交易协议之一。

BBS 类似于 Reddit 论坛，但与 Reddit 不同，BBS 是去中心化的，并且基于 EOS 链。BBS Network 是所有 BBS 版块的集合，可以在网络中选择的任何运营商的任何域上运行。所有用户可以在其中创建论坛来发布、评论和分享想法，同时从他们的内容中获得收入，将数据的拥有权及控制权真正还给用户。

![](https://cdn-images-1.medium.com/max/912/0*H4VkViS4hKf_cHp1)

BBS 的自动收益共享生态系统，由 BBS 管理员（所有者）、社区版主、内容发布者（帖子）、运营商（网络主机）、开发者（新功能）组成。每个帖子实际上都是一个 NFT，用户可以在自己帖子的广告位中进行创作、购买、销售和收益。所有广告和 NFT 收益将在生态贡献者之间透明分享，生态有多种贡献和收益方式。

BBS 是抗审查的，BBS 上的数据受区块链保护，意味着所有 BBS 用户、内容和钱包始终可以恢复，并且永远不会被任何人审查、关闭、扣押或更改，即使 BBS 也不行。BBS 所有者可以在网络中的运营商之间进行选择，允许不同的内容政策。

![](https://cdn-images-1.medium.com/max/912/0*lVJVRJ2LCzFRNJN3)

在 BBS 上发的一个帖子都会存储为一个对应的 NFT，发布者可以为每个 NFT（帖子）设置一个初始价格。大多数帖子的初始价格可能相对较低（默认设置为 0.30 美元），会随着新买家的增加而增加。

BBS 采用的是 NFT 竞价模式，任何人都可以购买。购买任何已发布的帖子就是购买该 NFT，包括该帖子的广告位，新一轮报价至少比原价高出 30%，售出后，帖子的前所有者将获得购买价格的 10%（无论他们在拥有帖子时从广告商那里获得多少收入） ，剩余的差价（20%）由发布者（帖子的原作者）和 BBS 管理员（负责 BBS 的审查和维护）平分。

这种竞价的结果是随着帖子不断拍卖，帖子的所有者将承担潜在的风险和回报，而 BBS 管理员和原发布者也可以持续获得收益。这些丰富的交易数据可以作为预测帖子在社区中潜在流行度的无价信号，首次实现了基于价值的内容管理。

● 经济模型

BBS Network 采用一种主 token + 多种子 token 的机制：主 token 为 BBS Token（$BBS），子 token 为社区代币（CT）。

BBS Token 作为 BBS Network 基于 ERC-20 的原生 token，是所有 BBS 接受的标准支付方式，用于 NFT 和广告购买，以确保节点和共享激励系统之间的金融互操作性。此外，持有者能够通过质押和投票参与 BBS DAO 治理，也可以选择将其代币锁定长达数年以提高治理权并赚取质押奖励。

社区 token（CT）是论坛内部流通货币，以美元计算。每个 BBS 都有自己的社区代币，BBS 社区定期也为会员提供 CT 奖励，如用户加入、访问、购买等行为都会获得相应 CT 奖励。这意味着每个人都可以开一个 BBS 社区并出社区代币，但其社区代币的价值取决于社区的运营情况。代币间的流动性通过出纳系统兑现，CT 持有者可以将 CT 兑换为等价的 BBS Token 进行套现。

● 经济模型

BBS Token 总量 10 亿，按以下比例分配：50%，参与挖矿；20%，流动性挖矿&质押奖励；10%，资金；10%，储备；10%，团队。为激励 BBS 用户和社区，其中 50% 参与挖矿的部分预计在未来几年内以每日分配的形式，基于用户参与度和贡献，分发给 BBS 生态系统中所有以英语为基础的 BBS 参与者。

● 功能单一

BBS 核心的留言板功能有些许打怀旧的情怀牌因素，在注意力经济时代，站内内容的呈现方式较为单一，相比于传统的社交媒体平台丰富而成熟的内置功能，BBS 产品逻辑简单，目前在功能使用方面尚未能有较大突破，而且面临着不断冒头的五花八门的 SocilFi 产品的激烈竞争。此外，人们花钱购买帖子的动力尚显不足，除了作为一种投机手段外似乎不够吸引人，何况投机有风险。用户心智的转变和使用习惯的迁移于 BBS 平台而言亦是很大挑战。

● 确权问题

对于创作者经济平台，创作者及其原创生产内容才是最大的生产力。虽然 BBS Network 设计了一个让内容流通和变现的生态，对于许多具体问题尚未清晰界定，如发布帖子的内容是否可以搬运，在原作者不知情的情况下是否涉及侵权，搬运内容的话收益又将如何分配，这样是否又与 Web 3 中信仰真正属于用户的数据确权相悖；又比如虚假信息甚至谣言的传播亦是内容平台的一道红线，加之 BBS Network 作为公共数据库不会被任何人审查、关闭、扣押或更改，这在内容管理方面亦有着避不开的潜在风险和负面影响。

● 流量泡沫

一方面，BBS 项目尚处在早期阶段，用户体量相对较小，用代币激励用户活跃的机制实则有人为鼓噪泡沫之嫌；另一方面，流量当道的时代流量来去匆匆，现象级的 Web 3 社交产品尚未出现，因此有发展空间但需要可持续的运营和健康的生态。在流量泡沫散去后 BBS 是否能维持产品的内在价值还有待市场和用户检验，这期间更需要项目方实打实的持续建设和完善。

强大的 Web 2 社交媒体竞品一直是 SocialFi 赛道跨不去的高山，对于 SocialFi 项目而言，在用户侧方面，用户体量和使用习惯是两大痛点。相比于庞大的传统社媒平台用户基数， Web 3 用户体量实在是太小了，细分的 Web 3 社交产品更显得过于小众。近期在众多推特大 V 的推手下大火的 Damus 应用及去中心化社交协议 Nostr ，尽管短期内得到了巨大流量和关注，用户活跃也难逃在短期热度消散后迅速下降。SocialFi 需要非常强的出圈能力才能真正迎来高光时刻，也只有真正能打的 SocialFi 产品才能吸引用户并最终留住用户。BBS 作为 SocialFi 现实生活应用场景的代表之一，不妨期待一下其未来表现能否经受住市场和用户的检验。
