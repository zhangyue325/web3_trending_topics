Chainlink 现在可供基于 Base 的开发人员使用。Base 还将参与 Chainlink 的 SCALE 计划，为开发人员提供对关键数据和预言机服务的低成本访问。

  


Chainlink Price Feeds 在 Base 测试网上线，Base 测试网是一个由 Coinbase 孵化的新以太坊 L2。Base 旨在为开发人员提供安全、低成本的方式，便于他们实现去中心化的应用。通过此项集成，开发人员可以获取权限，访问行业标准的 Chainlink Data Feeds 和其他 web3 服务。

  


Base 也将参与[Chainlink 的 SCALE 计划](http://mp.weixin.qq.com/s?__biz=MzU0MTgyMDQwNQ==&mid=2247495079&idx=2&sn=23e4ff72325178b24f142ca89b31b954&chksm=fb26b93dcc51302b6dadd9100208a29196eca6f97c070213e6acb026dc9b84f704aa0d9f4e31&scene=21#wechat_redirect)，该计划为开发人员而设计，旨在帮助开发人员取得更低成本的 Chainlink 数据和服务。作为计划的一部分，Base 将承担部分 Chainlink 网络服务的运营成本，减少由 Chainlink 节点在 Base 提交预言机报告时产生的 gas 费用。

  


![](https://img.foresightnews.pro/202303/687855db4cd9296c87f7aaa268cfd603.svg?x-oss-process=style/scale70)“我们非常高兴能与合作商 Chainlink 一起发布 Base 并加入 Chainlink 的 SCALE 计划，为开发人员提供构建应用程序所需的关键数据和服务，”Base 负责人 Jesse Pollak 说。

  


Chainlink 的 SCALE 计划是一项以可持续加速区块链和第 2 层生态系统发展为初衷的计划。它的重点是让 L1/L2 能够在一段时间内支付 Chainlink 预言机网络的运营成本（例如交易燃料费），从而增加对高质量、低成本和按需预言机服务的访问权限。随着 Chainlink SCALE 计划中的区块链 /L2 生态系统的成熟，预言机网络的运营成本可以逐渐过渡到完全由 dApp 用户费用支付。

  


![](https://img.foresightnews.pro/202303/2ba4cb40f65348c1d120c9804dfb51dc.svg?x-oss-process=style/scale70)Chainlink 区块链合作伙伴全球负责人 Niki Ariyasinghe 表示：“我们很高兴支持 Base 的测试网发布，并与他们合作开展 SCALE 计划，以支持生态系统的发展和 Base Layer-2 络的长期采用。“通过降低预言机节点的运营成本，Base 将能够引导其生态系统的增长，并为构建安全和可扩展的 Web3 应用程序提供更具吸引力的环境。”

  


这种资源部署将帮助 Base 成为以太坊生态系统中安全且高实用性的第 2 层解决方案，同时为其生态系统开发人员提供范围广泛且高度可靠的预言机服务。这将为 Base 奠定基础，使其能够支持安全、功能齐全且高性能的应用程序。
