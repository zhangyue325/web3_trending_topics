近日，根据相关法庭文件披露，Mysten Labs 3 月 22 日与 FTX 债务人达成协议，将以 9630 万美元的现金回购 FTX 对 Mysten Labs 的股权投资和 SUI 代币权证。

长达 123 页的文件议，详细还原了 2022 年 8 月 FTX 投资 MystenLabs 的情况。结合法庭文件和 Sui 近期的一系列动作，SuiWorld 进行了一些总结及分析：

![](https://img.foresightnews.pro/202303/fd431ccd464c20395ed1274924d61377.png?x-oss-process=style/scale70)

  


### 1、 Mysten Labs 约 20 亿美元估值完成超 3 亿美元的 B 轮融资

这个融资信息是去年的公开消息了。FTX 及 B 轮投资者们通过投资 MystenLabs 的 B 轮优先股权，获得了 Sui Token 的代币权证。

### 2、 FTX 约 1.02 亿美元获得约 8.9 亿枚 Sui Token

FTX 在 MystenLabs 的 B 轮融中总共投资了 1.02 亿美元，获得了约 57 万股优先股，和获得约 8.9 亿枚 Sui Token 的代币认购权证。作为领投方的 FTX 投资 Sui Token 的平均成本是非常低的，约 0.114 美元。

![](https://img.foresightnews.pro/202303/7e83a262ec4aaab9520c413f9b64d2e4.png?x-oss-process=style/scale70)

  


FTX 这里有两笔获得 SUI 代币权证非常重要：Cottonwood Grove Limited（Alameda）和 FTXTrading 分别以 500 美元的价格获得了约 1.73 亿和 3.46 亿枚共计 5.19 亿枚 Sui Token 的代币权证。这 5.19 枚 Sui Token 在 Sui NetWork 主网上线和 Sui Token 启动后，满足一定条件，将用于在 FTX 平台上做市交易。（Sui World 注：Cottonwood Grove Limited 是 AlamedaResearch 全资控股公司）

作为曾经的头部交易平台和做市商， FTX 领投 Sui 超过 1 亿美元的的一个重要条件，就是旗下的做市商 Alameda 和 FTX Trading 几乎免费拿到了 5.19 亿枚 Sui Token 用于在 FTX.com 平台上做市。**但是随着 FTX 在 Sui 启动之前就崩溃了，这个代币权证的协议也不存在了。**

### 3、部分 B 轮投资者的 Sui Token 筹码成本约为 0.28-0.31 美元

抛开 FTX 的做市筹码部分，其他部分 B 轮投资者获得 Sui Token 代币权证的成本仍然可以参考 FTX Ventures，平均成本约在 0.28-0.31 美元之间。

![](https://img.foresightnews.pro/202303/5e7fde99f3de198ddb87c7c19209ac9b.png?x-oss-process=style/scale70)

  


### 4、Sui 将收回大量低成本的机构筹码

除了约 0.28-0.31 美元之间的 Sui Token 投资筹码，对于领投方 FTX 来说，还额外获得了 5.19 亿枚 Sui Token 的做市筹码。

如今 Mysten Labs 以 9630 万美元的现金回购 FTX 对 Mysten Labs 的股权投资和 SUI 代币权证，意味着收回了大量 FTX 低成本的投资筹码和做市筹码。**对于未来上线的 Sui Token 来说是个好事。**

值得提的是，FTX 同样也以相似的估值、投资金额领投了另一条 Move 链 Aptos，FTX 对 Aptos 的投资方案或和 Sui 非常相似，即 FTX 持有大量（超过流通量 1/3）低成本的投资筹码和做市筹码。这意味着在 FTX 崩溃之前上线的 Aptos，部分低价筹码已经进入市场，没出售完的筹码也会等待 FTX 债权人的处置方案。

### 5、FTX 债务人可以继续「向其它第三方寻求更高的报价」

文件还披露，目前 FTX 债务人可以继续「向其它第三方寻求更高的报价」，直到法院最终确定出售日期。Mysten Labs 的股权投资和 Sui 代币权证对于不少机构来说还是有吸引力的。

### 6、SUI 主网上线少了最大的外部障碍

处理完与 FTX 债务人的问题，也就处理了 SUI 主网上线的最大的一个外部障碍。SUI 主网上线与 Sui Token 的发布，不用考虑来自 FTX 此前的投资筹码的处理问题。

### 7、9630 万美元对应了 1 亿美元基金

9630 万美元的**现金回购**一旦完成，对于去年刚融资超过 3 亿美元的 Mysten Labs 来说，**短期并无财务影响。**长期的影响或主要在于 Mysten Labs 对 Web3 的一些布局，比如投资、收购 Web3 项目。

这也对应上了此前的消息，Mysten Labs 首席执行官 Evan Cheng 正寻求为一个 Web3 基金筹集超 1 亿美元资金，该基金将不限于 Sui 生态。Hashed 联合创始人 Alex Shin 和资深对冲基金投资者 Sandeep Ramesh 将作为普通合伙人加入新创立的 Web3 风险基金。
