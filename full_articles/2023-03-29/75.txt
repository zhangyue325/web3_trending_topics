2023 年 4 月 2 日，OpenJob 联合了十家顶级 Web3 企业在北京举办线下招聘会。此次线下招聘，旨在以 Face to Face 的方式帮助求职者和招聘方之间**快速对接**，促进行业的人才流动性提升。

  


此次活动将聚焦于 Web3 行业热门岗位，包括但不限于：**前后端开发、区块链工程师、产品经理、投研分析师、量化交易员、市场运营、商务**等。包含**全职与实习**两种选择。

  


我们对接了北京顶尖高校的求职学生资源，其中包括：**清华大学，北京大学，中国人民大学，北京邮电大学，中央财经大学**。此外，我们也邀请了来自**百度、腾讯、阿里、字节、小红书、网易**等大厂背景的求职者。

**公司介绍**

Founded in Singapore in 2018, Alchemy Pay is a **payment gateway that seamlessly connects crypto and global fiat currencies** for businesses, developers, and users. **Particle Network**

**公司介绍**

Particle Network 是一个**面向 Web3 开发者的中间件平台**，致力于提高开发者的开发效率，加速推动 Web3 的 Mass Adoption。核心产品是**基于 MPC-TSS 技术的登录和钱包中间件**，开发者通过使用这个中间件可以使得其终端用户能直接使用电话，邮件，以及谷歌，Facebook，Twitter 等各类型方式直接登录到 dApp 中，并且自动创建一个 dApp 内的链上非托管钱包。通过 Particle Network 的中间件，可以极大地降低用户的登录门槛，以及大幅度提升后续用户与链交互（包括转账，签名，授权等）的使用体验。Particle Network 是**全行业首个将 MPC-TSS 技术运用到面向开发者服务的这个场景的公司**。

  


Particle Network 除了提供登录与钱包中间件以外，也提供 Nodes Services, Data API, NFT Full-stack API / IPFS 服务等中间件产品。未来 Particle Network 希望能够成为**面向开发者的全栈中间件平台**，能够帮助开发者集中在业务场景的建设，释放生产力，提高创造力。

  


Particle Network**累计完成了 1000 万美金的种子轮融资**，投资者包括：ABCDE / HashKey / Animoca / LongHash Ventures / GSR Ventures / Op Crypto /  ArkStream 等。
