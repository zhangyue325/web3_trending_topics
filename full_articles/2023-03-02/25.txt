周三，Polygon 宣布正在为其新 Web3 ID 服务转入零知识证明赛道。  


Polygon 在其以太坊侧链上宣布了 Polygon ID，这是一项 Web3 识别服务。  


该服务为基于区块链的应用程序验证用户凭据，而不会泄露个人信息。  


它将与 Collab.Land、Fractal、Kaleido 和 The Sandbox 建立合作伙伴关系，并以此获得约 400 万用户。  


Polygon Labs 正在将该服务集成到 Polygon zkEVM 中，该服务将于本月发布。  


与竞争对手的身份解决方案不同，Polygon ID 使用零知识证明（ZKP）。  


使用 ZKP，敏感信息不会公开上传到区块链以进行验证。因此，Z**KP 正在成为 Web3 中越来越流行的技术**。  


Polygon 称这是“互联网信任模式的根本改变”，并且 Polygon ID 对企业和个人都有好处。  


对于企业来说，它消除了管理用户数据的挑战；对于个人而言，它使他们在与机构和私人机构分享个人资料时，更能控制自己的身份和酌情权。  
![](https://img.foresightnews.pro/202303/317e728a17633c83aab8e9ecb2bf0486.jpeg?x-oss-process=style/scale70)  
Polygon 在推特上宣布:“未来是自主的。”参考生态系统范围内走向自我主权认同（SSI）。  


SSI 允许个人创建和管理存储在分布式账本上的自己的身份，该技术使用公钥加密技术保护信息，个人控制他们公开的内容。  
SSI 增强了隐私和安全性，同时降低了身份盗窃和欺诈的风险。然而，广泛采用的挑战仍然存在，包括互操作性和确保准确性。  


尽管如此，SSI 可以改变数字身份，并影响金融、数字投票和医疗保健等领域。  


最近，**ZKP 在区块链行业中变得越来越重要**。  


使用这种技术，用户可以证明他们知道一些东西，而不必真正透露他们知道什么，这使得在不泄露任何敏感信息的情况下提供所有权或授权证据成为可能。   


ZKP 允许在链下执行计算，同时验证多个证明，并实现更高效的数据存储和检索，从而使区块链更快、更可扩展。  


一些使用 ZKP 的 Layer1 区块链包括 Zcash、Tezos、Cosmos 和 Aleph Zero，还有更多的链正在探索其实施。
