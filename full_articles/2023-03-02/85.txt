**什么是 DAO？**  
DAO 代表去中心化自治组织，简单地说，DAO 是由智能合约组成的组织，成员（通常是代币持有者）共同决定组织或产品如何发展。示例场景：  
**1**. 10,000 名代币持有者每人将加密货币存入智能合约，并投票决定用总数购买什么。  
**2**. 100 万代币持有者控制着 Twitter 的去中心化版本，代币持有者在其中提议并投票决定功能更新。  
DAO 可以执行社区驱动的决策，决定何时以及如何更新塑造组织或产品的代码。  
**什么是 SNS？**  
SNS 代表服务神经系统，SNS 是一种强大的 DAO 形式，它允许社区完全在链上管理在互联网计算机（ICP）上运行的智能合约和去中心化应用程序（dapps）。SNS 的主要注意事项：  
**1**. SNS 的设计类似于网络神经系统（NNS）的设计，它是管理整个 ICP 的链上 DAO。  
**2**. 由 SNS DAO 控制的 dapp 由 SNS 代币持有者提交链上提案并对其进行投票来管理，没有一个开发人员或一群人控制 dapp，dapp 是由代币投票控制的。  
**3**. ICP 上可以有很多 SNS，任何开发人员都可以将他们的 dapp 的控制权交给 SNS DAO，这样做可以将控制权交给 DAO 代币持有者。  
![](https://img.foresightnews.pro/202303/c6f958dcbe3d21956ac47c91c0ecc861.png?x-oss-process=style/scale70)

  
**Web3 中的 SNS DAO 有多独特？**  
DAO 是一种不断发展的组织形式，用于实现目标。例如，在 ETH 生态系统中，Uniswap 和 MakerDAO 是流行的 DAO，在各自的智能合约中锁定了数十亿美元。  
然而，ICP 的独特之处在于它可以托管完全链上的 dapp（前端、后端应用程序逻辑和数据），因此 SNS DAO 的独特之处在于它们可以完全控制（通过投票）dapp 的每个方面，因为一切都是链上。  
拥有完全链上的 DAO 很重要，因为它使所有决策都可以在区块链上执行。这与其他区块链上的现有 DAO 形成对比，在其他区块链上，投票在链上进行，但结果的执行通常由开发人员在链下执行。这有助于实现真正的去中心化。  
因此，SNS DAO 作为一种工具来提供由其社区拥有和管理的完全去中心化的在线服务。  
**SNS DAO 的优势是什么？**  
SNS DAO 促进开发人员、企业家、终端用户和 VC 之间的健康交流，将 dapp 的控制权交给 DAO 的一些优势包括：  
**1**. **社区参与**：作为产品的共同所有者，用户成为核心贡献者，开发人员提供价值主张。  
**2**. **加速用户采用**：作为共同所有者，用户成为产品的最大拥护者。  
**3**. **资金**：项目通过社区驱动的去中心化销售获得资金。  
**什么是 SNS-1？**  
SNS-1 是 DFINITY 在主网上推出的第一个 SNS DAO，用于测试 ICP 社区参与的 SNS 去中心化销售。SNS-1 测试暴露了当前正在解决的改进领域，SNS-1 dapp 现在由代币持有者社区控制，他们将共同决定其未来。  
由于 SNS-1 旨在作为测试，因此在 SNS 中不同寻常，因为它是作为空白画布启动的，没有产品或开发团队。未来的 SNS 可能会源自已建立的 ICP dapp，ICDex 上可交易 SNS-1 令牌。  
**还有其他现有的 SNS DAO 吗？**  
ICP 社区对 SNS 技术的推出表现出了极大的兴趣和热情，OpenChat 已经公开宣布他们计划在 2023 年第一季度将他们的 dapp 转变为 SNS DAO，distrikt、Catalyze、Canistore 和 Nuance 等 Dapp 正在考虑创建自己的 SNS。

**如何参与 SNS DAO**  
**成为 SNS DAO 成员后我该做什么？**  
作为 SNS DAO 成员，您可以做的最重要的事情就是参与治理。链上治理是 DAO 与传统 Web 应用程序的区别，因为成员在 DAO 的成功中发挥着关键作用。  
DAO 成员可以通过多种方式做出贡献，从推送代码到设计徽标或 UI，再到简单地对影响 dapp 发展方式的 SNS 提案进行投票。  
**您可以通过哪些方式参与 SNS DAO？**  
成员可以通过多种方式参与 SNS DAO，包括链上和链下，创建提案需要编码技能。一些例子包括：  
**1**. **对 SNS 提案投票**：您可以手动对不同类型的提案进行投票，也可以关注其他投票的神经元。  
**2**. **创建升级提案**：提交升级提案，更新 SNS DAO 控制的 dapp 的代码，这允许任何人向 SNS DAO 贡献代码。  
**3**. **升级资产容器**：目前对资产容器的更新不通过提案，但是，您仍然可以通过升级由 SNS DAO 控制的资产容器来更改前端。为此，您必须首先创建一个提案，将您的主体 ID 添加到可以进行更新的控制器列表中。  
**4**. **创建动议提案**：动议提案不会升级 dapp 的代码，而是在新功能和 dapp 开发方面引导 DAO 的方向。  
**5**. **创建 SNS 参数更新建议**：您可以为许多 DAO 参数提出升级建议，例如最长质押期、投票奖励等等。  
与治理无关的贡献示例是：  
**1**. **内容审核**：许多社交媒体 dapp 在扩展到数百万用户时可能会纳入内容审核，版主根据 DAO 的内容政策审查和批准内容。  
**2**. **UI/UX 设计工作**：用户不仅可以通过编码做出贡献，还可以设计 dapp 的不同方面。  
**3**. **链下营销 / 邀请朋友**：DAO 的成员可以成为 dapp 的倡导者并以不同的方式进行推广。  
**SNS DAO 中的投票如何运作？**  
一旦您获得了 SNS 代币，您就可以将它们锁定在 SNS 神经元中并创建一个提案或对一个提案进行投票。如果提案被采纳，SNS DAO 的代码或它控制的 dapp 就会升级。  
根据 SNS 配置，投票的用户可能会因参与治理而获得额外的代币奖励。如果你想更深入地参与，你还可以通过提案来建议功能更新，以改进由 DAO 控制的 dapp。  
**您如何获得 SNS 令牌？**  
每个 SNS DAO 都有自己独特的 SNS 令牌，例如，当 OpenChat 将他们的 dapp 变成 SNS DAO 时，他们将拥有 CHAT 代币。  
SNS 代币的获取方式主要有以下三种：  
**1**. **参与去中心化销售**：当 SNS 发布时，它必须经过去中心化销售，您可以在 Launch Pad 选项卡下参与 NNS 前端 dapp，在销售期间，您可以使用 ICP 购买 SNS 令牌。  
**2**. **在去中心化交易所（DEX）上购买代币**：去中心化销售结束后，流动性代币通常会在多个 DEX 上列出，您可以在其中使用 ICP 实用代币或其他代币购买它们，具体取决于 DEX 列出的代币对，例如，还有比特币或以太坊。需求量大的 SNS 代币在 DEX 上的价格可能高于其去中心化销售的价格，ICDex 已经支持 SNS 令牌。  
**3**. **空投**：除了购买 SNS 代币外，您还可以通过接收空投或参与 dapp 获得代币，从而获得代币作为奖励。在这种情况下，代币被用作鼓励积极参与 dapp 的激励措施，这有助于增加活跃用户群。最后，如果您已经拥有代币并参与治理，您可以像在 NNS 中一样获得代币作为投票奖励。  
**您是否因参与治理而获得奖励？**  
SNS 的工作方式与 NNS 非常相似，但是每个 SNS DAO 可以不同地配置它们的参数。一旦您将 SNS 治理代币锁定到 SNS 神经元至少所需的最短时间，您就可以参与投票，根据配置，投票可能会用 SNS 治理代币奖励参与者。  
**什么是社区基金？**  
从 ICP 生态系统的有利角度来看，社区基金帮助引导 SNS DAO 生态系统，并重新投资于生态系统的持续增长，社区基金本身由 DAO（NNS）控制。  
从用户的角度来看，社区基金是让在 NNS 中质押 ICP 的人利用其成熟度为未来的 SNS 提供资金的一种方式，从而实现更高的多元化。由于 NNS 决定社区基金的分配方式，参与其中的人可以利用更广泛的社区“群体智慧”来决定获取哪些 SNS 代币。  
您可以在此处找到有关社区基金的更多信息：  
* wiki.internetcomputer.org/wiki/Community\_Fund

**去中心化销售**  
**什么是去中心化销售？**  
去中心化销售是 SNS DAO 变得去中心化并且用户可以通过参与购买 SNS 代币的过程，销售后，每个参与者都会收到一篮子神经元。这些神经元持有 SNS 代币，因此拥有 DAO 的投票权，这是您可以为特定 SNS DAO 获取 SNS 令牌的多种方式之一。  
**去中心化销售在哪里进行？**  
去中心化销售完全在链上运行，销售由 NNS 发起，通过这种方法，销售的每个方面，从开始到执行，都是完全去中心化的，不受任何中心化实体的控制。  
甚至创建 SNS 及其令牌的决定也是由 NNS（一个 DAO）而不是开发人员做出的，以避免对想要去中心化其应用程序的开发人员产生不良的法律影响。  
去中心化销售遵循可配置参数列表：  
**1**. 要出售的 SNS 代币数量。  
**2**. 去中心化销售必须获得的 ICP 数量才能成功，如果没有达到目标，参与者将退还他们的 ICP，并且销售失败。  
**3**. 去中心化销售在结束前可以获得的最大 ICP 数量，如果达到最大 ICP，则销售结束，dapp 的控制权将移交给代币持有者，由于要出售的代币数量是固定的，因此筹集的 ICP 数量决定了 SNS 代币的初始价格。  
**4**. 每个参与者的 ICP 最小和最大数量。  
**我能从销售中得到什么？**  
如果去中心化销售成功，您将收到一篮子神经元，其中包含您的 SNS 代币。根据 SNS 配置，篮子可能包含多个神经元，每个神经元具有不同的溶解延迟。您可以使用您的代币参与治理，或者在神经元溶解后，在交易所出售流动性代币。  
您收到的 SNS 代币数量以及为每个代币支付的价格取决于销售期间筹集的 ICP 数量，ICP 筹集的越多，您获得的 SNS 代币就越少，但它们的价格会更高。  
**SNS 代币的价格是如何计算的？**  
可以在去中心化销售期间用 ICP 购买 SNS 代币，销售期间售出的代币数量是固定的，这意味着它们的价格由筹集的 ICP 数量决定。您可以通过将筹集的 ICP 数量除以售出的代币数量来计算代币价格，**价格 = ICP 募集量 / 售出代币数量**。  
请注意，SNS 代币的最终价格在去中心化销售期间购买时是未知的，因为它取决于销售中筹集的 ICP 总量。但是，您可以根据销售接受的最低和最高 ICP 计算最低和最高价格，您可以在销售提案中验证这一点。  
几个例子：  
**1**. 去中心化销售通过销售 5000 个代币筹集了 1000 个 ICP，您贡献了 10 ICP，这意味着您将拥有 50 个 SNS 代币，每个代币价值 0.2 ICP。  
**2**. 去中心化销售通过销售 100 万个代币筹集了 500,000 ICP，您贡献了 200 ICP，这意味着您将拥有 400 个 SNS 代币，每个代币价值 0.5 ICP。  
**3**. 去中心化销售通过销售 100 个代币筹集了 10,000 ICP，您贡献了 5 个 ICP，这意味着您将拥有 0.05 个 SNS 代币，一个完整的代币价值 100 ICP。  
**我如何参与销售？**  
您可以在 NNS 前端 dapp 中参与去中心化销售，这需要 ICP 实用程序令牌。ICP 代币可以在去中心化销售期间兑换 SNS 代币，如果您没有 ICP，您可以从任何一家交易所购买一些。  
请按照以下步骤参与去中心化销售：  
**1**. 进入 NNS 前端 dapp（nns.ic0.app），点击侧边栏的 “Launch Pad”，查看所有正在进行的 SNS 去中心化销售。  
**2**. 在 “Current Launches” 下选择您感兴趣的 SNS Sale，然后点击复选框。  
![](https://img.foresightnews.pro/202303/fd84d535b94a99931c4410858c0cbab0.png?x-oss-process=style/scale70)

  
**3**. 点击“参与”。  
![](https://img.foresightnews.pro/202303/05c8e45cd59f8f3fef4c1676cc70d46b.png?x-oss-process=style/scale70)

  
**4**. 输入您要参与的 ICP 数量或点击 “Max”，然后点击 “执行”。  
![](https://img.foresightnews.pro/202303/9ab4c5ea2fc348133731cec1369ba3a2.png?x-oss-process=style/scale70)

  
**5**. 系统将提示您查看您的参与情况，如果您看到正确数量的 ICP，请选中复选框并单击“执行”。  
![](https://img.foresightnews.pro/202303/ef59e9b6f1b01673a49c3024f902e635.png?x-oss-process=style/scale70)

  
**6**. 在最后一个屏幕上，您应该会看到您为参与输入的 ICP 数量，如果你想贡献更多，你可以重复这个过程。  
销售结束并成功后，您将收到 SNS 代币，请注意，这可能需要在销售结束后的几个小时内完成。您收到的 SNS 代币数量，以及为每个代币支付的价格，取决于销售期间筹集的 ICP 数量以及 ICP 花费的数量。ICP 筹集的越多，您收到的 SNS 代币就越少。  
如果未达到去中心化销售的既定目标，参与者将获得退款，销售的持续时间是可配置的，因此取决于每个 SNS 的配置，只有在针对启动 SNS 的原始提案进行 4 到 8 天的 NNS 投票期（除非更早达到 51% 的多数票）后才能开始销售。  
**谁发起去中心化销售？**  
开发团队首先为去中心化销售创建一个 NNS 提案，一旦 NNS 提案被采纳，销售本身就会自动开始。  
**作为 SNS 去中心化销售的参与者，我需要信任什么？**  
在参与 SNS 去中心化销售以分享 SNS DAO 的投票权之前，强烈建议您对相关 dapp 进行尽职调查，以确保 SNS DAO 值得信赖。Dapp 开发人员负责提供所有必要的信息，以便您验证 dapp 及其 SNS 配置。  
请注意，在没有尽职调查的情况下参与，你就是在暗中信任 NNS 社区和去中心化 dapp 的开发人员。无论如何，您必须相信开发人员不会在 SNS 发布期间更改 dapp。  
了解您可以验证的内容以及需要注意的事项：  
* wiki.internetcomputer.org/wiki/SNS\_decentralization\_sale\_trust

  
**如何验证 SNS 去中心化销售提案？**  
根据您的技术理解水平，您可以执行多个级别的验证，可以随时验证 SNS 容器，甚至可以在去中心化销售之前，但在将它们安装到 SNS 子网上之后。  
以下是一些验证方法：  
**1**. **销售参数**：您可以在 Launch Pad 或去中心化销售 NNS 提案中验证去中心化销售的参数是什么。  
**2**. **SNS 根容器**：一个 SNS DAO 由多个具有不同功能的 SNS 容器组成，根容器是控制给定 SNS DAO 中其他 SNS 容器的容器，您应该验证 SNS 根容器是唯一控制该 SNS DAO 中其他 SNS 容器的容器，验证对于确保没有内置后门很重要。  
**3**. **dapp 容器**：您可以验证由 SNS DAO 控制的 dapp 的容器 id 是否与您正在交互的那个相同。  
在此处了解有关验证的更多信息：

**是否可以将 SNS 代币转让给他人？**  
可以的，SNS 代币是 ICRC-1 代币，可以在任何支持 ICRC-1 的基于互联网计算机的钱包之间转移，包括您的 NNS 钱包。请注意，每个 SNS 都有自己的符合 ICRC-1 标准的分类账，用于跟踪其代币。  
当以神经元的形式接收你的 SNS 代币时，神经元可能有溶解延迟，这意味着你的代币可能不是流动的，即不能立即转移。相反，这些代币会在数周、数月或数年的时间内变成流动的（可转移的），具体取决于各个神经元的溶解延迟。  
**是否可以将 SNS 代币出售给其他人？**  
可以的，因为代币可以发送给其他人，所以它们同样可以发送到交易所，从而换取其他代币。出售 SNS 代币的最简单方法是通过列出代币的去中心化交易所（DEX），您可以将 SNS 代币换成其他代币，具体取决于任何 DEX 上可用的代币对。  
同样，您可以在 DEX 上用其他代币换取 SNS 代币。  
**SNS 代币是通货膨胀还是通货紧缩？**  
这取决于具体的 SNS 配置，SNS DAO 可能同时具有通货膨胀和通货紧缩压力。  
**通货膨胀**：  
**1**. 可以设置一个 SNS DAO 来为参与治理的人铸造投票奖励（类似于 ICP）。  
**2**. 如果 SNS 代币持有者投票，SNS DAO 可以铸造代币，例如：一款游戏每年铸造总供应量的 2%，并分配给玩家。  
**通货紧缩**：  
**1**. 每笔交易都会销毁 SNS 代币。  
**2**. 将 SNS 代币锁定到神经元中会将它们从流通中移除，虽然从技术上讲不是通缩，但它起到了通缩压力的作用。
