我们之所以会深入过去，是为了更好地了解现在，这就是为什么我冒昧地分享 NFT 是如何形成的故事。  
  
您可能想知道 NFT 是什么时候引入的，以及为什么它们突然引起了如此多的关注**。在阅读了关于 NFT 历史的不同但相似的故事后，我决定使用事实和日期来给大家说明。**

  
**01** **NFT 的历史变迁**

2012 年你在做什么？如果你不记得也没关系，**但那是彩色 Token 的想法诞生的时候，经过一系列的演变，它变成了你现在所知道的 NFT。**  
  
一个有趣的事实是，有几个 NFT 由以太坊区块链（2015 年推出）提供支持，但到 2012 年，NFT 的概念已经存在于 Meni Rosenfield 的一篇论文中，他在论文中谈到了彩色 Token。尽管由于多种谬误而无法实现彩色 Token，但它们仍然为 NFT 的未来奠定了基础。  
  
**2014 年，稀有的佩佩币问世，开启了表情包经济时代。**Pepes 是 NFT 的首批实现之一。如果您是 Whatsapp 的狂热用户，您一定看过那些带有绿色青蛙 (pepes) 的同志贴纸。

2017 年加密猫的诞生为 NFT 世界带来了前所未有的病毒式传播。此外，在 2017 年，加密朋克是由 John Watkinson 和 Matt Hall 创建的，并且完全免费分发——全部 10,000 个！CryptoPunks 现在至少价值 93,624 美元，所以你可以看到它有多棒。

**从 2017 年到现在，NFT 的名气像发射到太空的火箭一样不断飙升。**这就是 NFT 是如何形成的总结。即将到来的是你一直在等待的时刻。了解下一个 NFT 是什么。

  


**02** **关于 NFT 的有趣事实**

* 以太坊使用最广泛的用于铸造 NFT 的区块链，在以太坊区块链开发之前引入了第一组 NFT（2014 年）。
* NFT 最大采用者的前五个国家在亚洲。
* 最有价值的 NFT 价值超过 9180 万美元
* 超过 50% 的 NFT 销售额低于 200 美元。这意味着您有机会拥有 NFT
  


**03** **人们为什么要购买 NFT？**

您可能想知道购买 NFT 背后的科学 / 逻辑是什么。NFT 是数字和虚拟艺术；他们并没有像购买实体艺术品的著名原因那样，比如彰显品味，作为装饰品等。  
  
那么，人们为什么要购买它们呢？有以下原因。

* **利润**
尽管加密货币具有波动性，但许多人投资代币是因为它可能会盈利。NFT 没有什么不同，但在 NFT 的情况下，这非常棘手。为什么？艺术必须有需求，并且必须有人愿意为您出价才能获利。

* **排他性又有吹牛的权利**
因为只有一个实体（即个人、组织等）可以拥有世界上其他人无法访问的特定数字艺术。它为他们提供了吹牛的权利和独家特权，例如提前访问即将推出的 NFT 收藏或进入私人在线论坛 / 社区。

**04** **NFT 是 Web3 的一部分吗？**

最后是本文的中心部分。你可能想知道 Web3 和 NFT 之间有什么相似之处或者它们是如何联系起来的。尽管对于 Web3 时代的样子存在一些争论和争议，但可以肯定的是，Web3 会像 Web2 一样影响我们的生活，从我们连接的方式到我们的生活方式。

另一方面，**NFT 是数字资产，无需依赖中介即可在 Web3 上实现商品所有权兑换和交换。**  
  
**Web3 一直在提倡奖励创造者，而 NFT 是确保实现它的一种手段。**创作者现在可以创作他们的艺术作品并将其作为数字代币出售以赚取收益，而无需等待大公司开发加密货币。

要了解 NFT 的范围，想象一下您如何访问电子商务网站以在 Web2 上购买产品。在 Web3（由区块链驱动的互联网）中，去中心化应用程序就像电子商务商店，可以买卖产品（即服装），但在这种情况下是 NFT。通过这些技术，创作者可以完全控制他们的工作和收入。

![](https://img.foresightnews.pro/202303/6df38fe6fe4ad5a4fd4f8b7b268013d6.png?x-oss-process=style/scale70)

**05** **NFT 的特点**

以下是 NFT 的三个特征：

* **独创性**
得益于数字艺术创作者签署 NFT 的能力，您可以分辨出原创艺术与赝品。如何？在智能合约的帮助下，创建者可以在区块链中发起一个动作，这允许时间戳并使 NFT 不可更改。想想水印是如何在 Web 2 中实现的，但在这种情况下，水印并不直接在创建中。

* **不可销毁**
NFT 是区块链上的资产，这意味着与实物艺术品不同，它们既不会被烧毁、破坏，也不会接触任何自然灾害。

* **不可分割**
这里不可分割的意思是：对于像比特币这样的代币，你可以购买价值 50 美元的比特币（现价每枚 / 两万左右），但你不能以 50 美元的价格购买价值 500 美元的 CryptoPunk（第一批著名的 NFT 项目之一），您要么以 500 美元的价格购买 CryptoPunk，要么一无所有！

  


**06** **NFT 的用途有哪些？**

* 博弈
游戏玩家对 NFT 最感兴趣，因为他们了解虚拟世界和货币。因为他们可以在市场上转让或交易游戏物品，所以激起了他们的兴趣。  
与禁止销售或转让游戏内物品的传统游戏相比，玩家更喜欢拥有游戏物品，因为它可以个性化他们的体验并使其更加有形。游戏玩家现在可以通过建立或增加他们的游戏内资产来赚钱。

* **艺术**
NFT 被用作数字艺术已经不是什么新闻了。创作者现在使用 NFT 来保护他们的创作免受侵犯版权或剽窃作品的侵害。由于 COVID-19 的限制，这可能是博物馆和画廊展示其艺术的一种方式。

* **认证**
某些课程可能会提供 NFT 和课程完成证书或许可证以节省访问时间。使用 NFT 存储课程完成证明并减少验证记录的负担可能要容易得多。我以前从未见过，但这个想法似乎令人难以置信。
