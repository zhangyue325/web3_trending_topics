**波卡一周观察**，是我们针对波卡整个生态在上一周所发生的事情的一个梳理，同时也会以白话的形式分享一些我们对这些事件的观察。**本文旨在提供信息和观点，不为任何项目作背书。**任何提及的项目，均不作投资推荐，仅作信息展示。  
  
  


1

观察与观点

首先是插槽拍卖的情况，去中心化计算网络 Phala Network 的先行网 Khala Network 赢得 Kusama 第 74 次平行链插槽拍卖。而波卡的第 42 次平行链插槽拍卖则竞争激烈，InvArch、Moonsama、peaq、SORA 等 4 个项目都在参与竞拍。

  


![](https://img.foresightnews.pro/202303/559c5fd9db965c2e3bc59360392fefe3.png?x-oss-process=style/scale70)

  


此前，由于波卡大力支持各类黑客松活动以及各类基础设施，已经吸引了众多团队纷纷提案对波卡进行建设，但以往的治理方式流程处理时间长，效率也不高，也阻碍了波卡生态的进一步迭代。

  


如今，在 Kusama 的治理方式成功改进为 OpenGov 之后，更多的提案被提出、讨论和处理，加速了 Kusama 生态的发展。在顺利经营一段时间后，OpenGov 终于来到了波卡生态中。波卡的#6701 Pull Request，将 OpenGov 引入了波卡并且从 Kusama Network 上删除了 Gov1，现已合并，预示着波卡也将开启加速治理的阶段。

  


而在波卡生态方面，此前我们介绍过的企业级平行链 Aventus 正在推动波卡生态与传统企业打通。具体操作方式，是通过开发一个双向 API 网关，使 Web2、Web3 和内部应用程序能够与波卡生态（包括所有的平行链）进行互动，从而便于波卡及其所有平行链与传统企业应用和新的 Web3 解决方案集成。

  


通过这种技术，可以创建许多 SaaS 服务，为传统企业软件产品和服务提供与波卡中继链及其所有平行链的双向通信渠道。网关 API 将使企业组织能够让其用户更轻松地与任何波卡平行链互动，无需直接拥有和管理链上的本地通证或任何 Crypto，甚至无需理解复杂的区块链技术概念。

  


不过 Aventus 正在将该技术以提案的方式经过国库申请，目前正在投票讨论中。期待这样的技术早日推出，以便扩大波卡技术在传统领域的更大范围的采用。
