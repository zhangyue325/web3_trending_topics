1993 年，美国著名杂志《纽约客》的漫画家施泰纳创作了这样一幅荒诞却广为流传的漫画，真切地体现了互联网世界的虚拟性与匿名性。

![](https://img.foresightnews.pro/202303/72f809746b4c0e7b70fd5bfb1f514820.png?x-oss-process=style/scale70)

“在互联网上，没有人知道你是一条狗。”是一只坐在椅子上、爪子放在电脑键盘上的大狗，对坐在它旁边地板上的一只小狗说的话。

虽然这是一种夸张的表现手法，但互联网确实给予了我们极大程度上的自由。在互联网上，我们能够去说平时不敢说的话，做平时不敢做的事，变成想要成为的人。

随着互联网走过 Web1、web2，如何证明计算机面前的是真实的人还是狗或 AI？这是 Web3 在迎来它的发展时在持续探讨的话题。

DID 概念热度上涨
----------

3 月 22 日，BA 上线第 30 个 Launchpad 项目 Space ID（ID），再次带火 DID 概念。

DID（Decentralized IDentity，去中心化数字身份），作为去中心化的数字身份，和 NFT、链游、DeFi、DAO 一样，都是作为 Web3 的一部分。

DID 的出现首先离不开区块链的存在，由于后者的分布式存储、可追溯性和去中心化性质、数据的不易篡改性等特征，才能够形成 DID 的核心特质，即它作为一个真实人类的链上数字身份证明，必然不能被中心化的集团或组织管理、数据不能丢失或遭篡改。

除了 Space ID（ID），Ethereum Name Service（ENS）、Gitcoin（GTC）、Civic（CVC）、SelfKey（KEY）、RSS3、Litentry（LIT）和 Ontology（ONT）等都是热门的 DID 项目。Polygon Labs 也在近日宣布推进 Polygon ID 发展。

ZKP 助推 DID
----------

当 ZKP（Zero-Knowledge Proof，零知识证明）技术被引入区块链建设时，进一步推进了 DID 的发展。

零知识证明技术能够做到实现信息得到验证的同时而不透露信息的具体详情。这再进一步让 DID 在实现链上数字身份的唯一性、完整性和永久性之外，再添加了隐私保障性。

它的到来，让 Web3 更“Web3”。

例如 Polygon 项目于今年 3 月 2 日推出 Polygon ID 时，就表示它的出现让开发人员能够为广大的用户构建数字自我主权，“通过开源许可面向公众，任何人都可以成为 Web3 身份的发行者（例如 KYC 提供商、DAO 等）、验证者（例如 dApp）或用户。”DID 是互联网的下一次迭代。

Polygon ID 带来什么？
----------------

今年 3 月 2 日，Polygon 联合 The Sandbox 和 Collab.Land 推出由 ZKP 提供技术支持的 Polygon ID，官方团队表示这意味着互联网信任模式的根本变化。

对于企业而言，用户数据管理的麻烦得到彻底解决；对于用户而言，可以更好的自行管理个人数据的使用；对于开发人员而言，可以启用网络信任和验证机制，提供给用户友好的体验。

这同时也意味着 Web3 基于 ZKP 技术，迎来了一个无需信任的互联网时代，互联网上的交易、社交、金融、身份等信息都可以以无需信任的方式发生。

1.无需信任。通过基于 ZKP 的 DID，用户可以证明他们是人类而不必披露个人信息。

2.用户体验的改进。用户可以利用无密码登录各种网站、加入各种线上应用和场景当中，而无需切换、输入密码、绑定身份信息等，黑客窃取用户信息的历史也将终结。

3.作为真实世界的凭证。DID 的功能不仅在线上，在线下，用户也可以使用个人的实物身份（文凭、驾驶执照或国民身份证）的线上数据生成零知识证明，与智能合约进行交互，从而在现实生活中进行验证。

4.在不牺牲隐私的情况下实现应用产品的合规性。借助零知识技术，开发人员可以在其现有合规性流程中进行开发建设，而不用牺牲用户隐私。

5.线上的真实数字身份。如果把二维的互联网看作是一个世界，DID 可以看作是通行世界的身份证明。这种身份证明可以让用户在线上活动享受和线下活动同等的权益和便利。

Polygon ID 的进行时
---------------

3 月 14 日，Polygon Labs 发布公告称与 Unstoppable Domains 合作推出 .polygon 域名，旨在为用户提供跨超 750 个 dApp、游戏和元宇宙的“可移植身份”。

该服务将提供给 Polygon 区块链生态系统中的大约 1.8 亿用户和 40000 项服务。目前已有超过 270 万个域名在 Polygon 区块链上注册。

Polygon Labs 业务发展副总裁 Sanket Shah 在一份声明中表示 Polygon 用户数字身份的重要性：“Web3 域名将为我们的社区提供他们完全拥有的数字身份，这样他们就可以在不泄露个人信息的情况下登录去中心化应用程序，并在没有长钱包地址的情况下进行加密交易。”

从 DID 的性质出发，它的形式可以是一个域名，一个加密钱包，可以是一个数字徽章或数字身份证，也可以是一个虚拟形象等，最终的形式都需要依据 Web3 未来的发展态势。

Polygon ID 是 Polygon 项目基于 ZK 技术的研究成果，但它基于 ZK 的发展或将不止于此。距离其 zkEVM（零知识以太坊虚拟机）主网上线时间 3 月 27 日不到几天，相信以 Polygon 的强大生态属性，它在 Web3 的建设将带来令人关注的发展成果，可以共同拭目以待。
