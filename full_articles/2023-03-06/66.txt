Unity 最近将 Web3 领域的顶级公司纳入其验证解决方案合作伙伴（VSP）计划，是去中心化游戏的一个重要里程碑。  


**Unity VSP 计划**

VSP 计划是由 Unity Technologies 创建的一个计划。  


该计划是一个由可信的第三方 SDK、插件和编辑器应用程序组成的生态系统，它们与 Unity 合作，以确保技术上的一致性，并不断验证最新的 Unity 版本的稳定性、生产力、性能和可用性。

最近，Web3 领域的顶级公司被纳入 Unity 的 VSP 计划，这代表了对去中心化游戏的长期增长前景的认可，以及对 Web3 对行业的颠覆性影响的认可。  


![](https://img.foresightnews.pro/202303/93c915c134a7495e551dd51eecbb9bd8.jpeg?x-oss-process=style/scale70)

**探索去中心化的游戏技术**

3 月 1 日，Unity 资产商店中推出了一个新的“去中心化”类别页面，其中包含经过审核的解决方案，以支持对实现游戏去中心化技术感兴趣的开发者。  


在视频游戏开发的语境下，去中心化通常指的是游戏中的所有权模式。在实践中，这意味着启用游戏机制，允许玩家创造、赚取或获得可以出售或交易的游戏内资源。

在游戏中实现去中心化的技术的一些例子是智能合约、虚拟钱包、不可伪造的代币（NFTs）和加密货币。  


![](https://img.foresightnews.pro/202303/97eb8fb9d7259d076047c4fa8063b0a5.png?x-oss-process=style/scale70)

**经过验证的 Web3 游戏解决方案**

目前，共有 13 个经过 Unity 验证的 Web3 游戏解决方案：  


* AIKON ORE ID - AIKON 的 ORE ID 是一个单点登录平台，允许开发者在首次登录时创建一个区块链账户。
* Algorand - Algorand SDK 为开发者提供工具，使其能够利用 Algorand 区块链轻松地将数字资产和原素整合到他们的游戏中。
* Altura - Altura 的 API 和 SDK 允许开发者在游戏中创建、更新和转移 NFTs。
* Aptos Labs - Aptos Labs 创建了一个软件开发工具包，使游戏开发者能够在 Aptos 上开始构建。
* Dapper Labs - Flow SDK for Unity 允许 Unity 开发者将他们的游戏和应用程序与 Flow 区块链集成。
* Immutable X - 一个领先的开发者平台，用于在以太坊上构建和扩展主流的 Web3 游戏（Web3 地图注：更准确的说法是“在以太坊生态的 Layer2 上”）。
* Infura (ConsenSys) - Infura 的基础设施和工具套件帮助开发者构建和连接到任何 EVM 兼容的区块链。
* Metamask (ConsenSys) - 这个 SDK 是一个库，开发者可以安装到他们的项目中，它可以帮助用户轻松连接到任何在 Unity 上开发的游戏的 MetaMask 钱包客户端。
* Nefta - 这个 SDK 让开发者获得数字资产、多链钱包、代币、自定义市场、白标服务等的无缝整合。
* Pocketful of Quarters - Quarters SDK 是一个为开发者提供的即插即用的货币化工具，使他们能够专注于创建社交和互操作的游戏。
* Solana - Unity 的 Solana SDK 让游戏开发者可以轻松地在他们的 Unity 游戏中整合基于 Solana 的 NFTs、代币和链上程序。
* Tezos - Tezos SDK 具有广泛的功能，包括跨游戏链接玩家账户、游戏内物品和货币的铸造和交易等。
* Truffle (ConsenSys) - Truffle 是 Ethereum 的智能合约开发框架。

IMX 是专门为游戏和 NFT 市场设计的，使其成为希望在以太坊上构建的开发者的热门选择。

AIKON 的 ORE ID 是一个单点登录平台，供开发者在首次登录时创建区块链账户。游戏者的私钥是加密的，可以通过他们的社交登录访问。
